video

Compatible with Flash 6 Player and above. This object allows you to display video on the stage without embedding it in your SWF movie.

Description

The Video object will allow you to display video content in your movie without actually embedding it in your SWF file. For example, you can use the Video object to stream a live broadcast video, TV content, and FLV files.

Examples and practical uses

Without the Flashcom server, this object is slightly limited. In fact, it was undocumented in the release of Flash MX. However, it is an extremely useful tool if you do have the Flashcom server and can even help you with embedded video files to a limited extent!

 

The Video object is typically used for streaming live or recorded FLV videos without storing them in your Library, or project a locally caught video stream from your Webcam, These features require the Flashcom server for functionality, and ActionScript specific to that application. However, you can do a few things with this object without needing the server. We will look at these in the next couple of entries, video.attachVideo and video.clear.

 

To create a new Video object instance, open the Library and select the Options menu. This is the pop up menu from the upper right hand corner button. Select New Video, and this will create a new Video object instance.

Tips and Precautions

Just as with any object name, make sure you do not name your video instances "video".
