The TextField.html property is a Boolean that indicates whether the text field contains an HTML representation.

Description

The TextField.html property indicates whether the context of a text field is a HTML representation. If true , a HTML text field is used; if false , the text field is non-HTML.

Examples and practical uses

The TextField.html property indicates whether the field is an HTML representation or not. If the field has the property set true , HTML is assumed; if false , then non-HTML is assumed. This property is used with the TextField.htmlText property . The TextField.html property is set with an assignment and a Boolean value :

 

myTextField .html = true;

 

See the example textfield.html.fla and textfield.html.swf .

Tips and precautions

The TextField.html property allows you to use HTML representation inside a text field. This does allow some flexibility that users will appreciate, including the use of HTML tags to indicate style formats. See the TextField.htmlText property.

 

You should consider using html text instead of TexfFormat objects when you want to make many style changes within a single text field (such as when you want color and style changes within a single sentence).

 

Generally, you will usually not want to change the html setting dynamically, and in most cases, you can simply configure this property via the Render text as HTML button on the Property inspector.
