Math.pow

a = Math.pow(b, c);

 

Where a is the variable that you want to equate to b to the power of c.

 

This method is compatible with Flash 5 or later.

Description

This method takes b, and raises it to the power of c, and returns the result. In a mathematical formula, it looks like this:

 

a = bc

 

 

Code

 

 

Notes

 

a = Math.pow(2, 2);

 

b = Math.pow(2, 8);

 

c = Math.pow(10, 5);

 

d = Math.pow(25, 0.5);

 

e = Math.pow(27, 1/3);

 

 

 

 

In the first line, a will be set to 4, because 2 to the power of 2 is 4.

 

In the second line, b will be set to 256, because 2 to the power of 8 is 256.

 

In the third example, c will be set to 100000, because 10 to the power of 5, is 100,000.

 

In the fourth example, d will be set to 5, because 25 to the power of 0.5 means square root of 25, which is 5.

 

Anything to the power of 0.5 is the square root.

 

In the fifth example, e will be 3, because anything to the power of 0.333333 (or 1/3) is the cubed root.

 

 

f = Math.pow ("r", 3);

 

If anything other than a number is passed into the function, then Flash will return NaN; which stands for "Not a Number".

 

Examples and practical uses

To see Math.pow plotted in a graph, see the example math.powplot.fla and math.powplot.swf .

Tips and precautions

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
