FStyleFormat.arrow

a.arrow = b;

 

Where a is the name of your style format, and b is a RGB color value being assigned to the arrow.

 

Compatible with Flash MX. This property will affect the color of arrows used in ScrollBar, ScrollPane, ListBox, and ComboBox Flash UI Components. It will not change the color of the arrow in disabled components. The RGB color value is written using the 0xRRGGBB format. Use FStyleFormat.foregroundDisabled to change the value of disabled arrow.

Description

This property assigns a new color value to the scrollbar arrow used by several Flash UI components.

myStyle = new FStyleFormat();

myStyle.arrow = 0x99CCFF;

myStyle.applyChanges();

myStyle = new FStyleFormat

({arrow:0x99CCFF});

This assigns a light blue color to the arrow instances in the custom myStyle style format.

 

 

myStyle.arrow = #99CCFF;

 

 

n/a

 

The color is not written in the correct RGB format.

 

Examples and practical uses

FStyleFormat.arrow is used to customize the color of active arrows on the Scrollbar, ComboBox, ListBox, and ScrollPane components. In the example file, fstyleformat.arrow.fla and fstyleformat.arrow.swf there is an example of when arrow colors are customized using this property. The file uses the following simple code, which assigns a light blue color to the myListBox and myComboBox component instances.

 

myArrowStyle = new FStyleFormat({arrow:0x99ccff});

myArrowStyle.applyChanges();

myArrowStyle.addListener(myListBox, myComboBox);

Tips and precautions

This property will only change the color of arrows, which are in active (enabled) scroll bars. If you want to alter the color of disabled scroll bar arrows, use the FStyleFormat.foregroundDisabled property. You should use FStyleFormat.applyChanges to update this property if you change the color value.
