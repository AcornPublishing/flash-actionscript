, Comma

ActionScript line, ActionScript line,... ActionScript line;

 

Where ActionScript line is any normal line of ActionScript.

 

Compatible withall versions of the Flash player. This operator is really just a quick way of writing code, or grouping related lines of code on a single line.

Description

This operator allows you to add a number of actions on a single line.

 

 

Code

 

 

Notes

 

a = 2, b = 3, c = 4;

 

 

Is the same as writing:

 

a = 2;

b = 3;

c = 4;

 

 

myClip._x = 100, myClip._alpha = 50;

 

 

 

Is the same as writing:

 

myClip._x = 100;

myClip._alpha = 50;

 

Examples and practical uses

The ", " is not a well known operator, although it is very useful in formatting scripts for legibility, because it allows you to place related actions on the same line. For example, if you wanted to initialize some variables, this code is quick to read and write for the advanced user:

 

// Initialize

x=y=20, h=w=50;

with (myClip) {

   _x=x, _y=y, _height=h, _width=w;

}

Tips and precautions

Combining lots of actions on a single line can make errors harder to find because there is more code per line. Be careful you don’t go overboard with its use if you are beginning ActionScript.

 

the ", " doesn’t seem to work well with comments:

 

/* Initialize */, x=y=20, h=w=50;

 

will not work. Don’t mix comments into a line that contains actions separated by commas.
