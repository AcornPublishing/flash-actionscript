TextField._url

myTextField ._url

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField._url property returns the URL of the SWF file that created the text field.

Description

The TextField._url property returns the full URL of the location of the SWF file that was downloaded for the current text field instance.

 

 

Code

 

 

Additional Explanation

 

 

myFile = myTextField ._url;

 

Stores the URL of myTextField in myFile .

 

 

myFile = myTextField ._url();

 

 

There is no method with this name.

 

Examples and practical uses

The TextField._url property returns the full URL of the current movie's SWF file that created the text field instance. This can be used in a variable:

 

myFile = myTextField ._url;

 

See the example textfield._url.fla and textfield._url.swf .

Tips and precautions

The TextField._url property can be used either by assigning it to a variable or by reference.
