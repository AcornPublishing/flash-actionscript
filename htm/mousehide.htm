Mouse.hide

Mouse.hide()

 

Compatible with Flash Player 5 and later.

 

The Mouse.hide() method hides the mouse cursor.

Description

By default the mouse cursor is visible in a movie. To hide the cursor the Mouse.hide() method can be used. When hidden, the cursor still exists but is not visible to the user. The mouse can be made visible again using the Mouse.show() method.

Mouse.hide()

 

 

Hides the mouse cursor .

 

Nothing is returned from the Mouse.hide() method.

 

Examples and practical uses

Mouse.hide can be used for two reasons:

 

·       You want to hide the cursor when it is in certain areas of the screen. For example, if you were designing a video player, you might decide to hide the mouse when it is over the video.

·       You want to replace the cursor with a custom cursor. To do this you first hide the original mouse pointer and then create a replacement movie clip that follows the mouse position.

 

An example of the latter is shown in file mousehide.fla / mousehide.swf. Notice the use of MovieClip.onMouseMove and updateAfterEvent(). Used together, these can produce a very smooth and responsive replacement mouse animation.

Tips and precautions

If you choose to use a replacement mouse, bear in mind that the original mouse costs nothing in performance to the Flash player, whereas the replacement does! You should therefore think twice about using a replacement mouse in performance critical applications and/orkeep your replacement mouse graphic simple.
