Mouse.show()

 

Compatible with Flash Player 6 and later.

 

The Mouse.show() method makes the mouse cursor visible in a movie.

Description

By default the mouse cursor is visible in a movie. It can be hidden using the Mouse.hide() method. To display the mouse cursor again, use the Mouse.show() method.

Mouse.show()

 

 

Displays the mouse cursor.

 

Nothing is returned from the Mouse.show() method.

 

Examples and practical uses

Mouse.show() is used to display the mouse cursor when a movie is playing. Usually, the method is attached to a movie clip on the timeline and triggered by the onClipEvent event after a Mouse.hide() method has been used to hide the mouse cursor for some reason:

 

onClipEvent(enterFrame) {

 Mouse.hide();

}

// more code

Mouse.show()

// displays the mouse cursor again

 

In this code, when the frame this code is attached to on the timeline is entered, the onClipEvent event handler is invoked and the mouse cursor is hidden from view. The mouse cursor remains hidden until displayed again later in the code using a Mouse.show() method.

 

The Mouse.show()method does not require a mouse object to be registered to receive events (using Mouse.addListener()) to function.

 

See also mouseshow.fla / mouseshow.swf.

Tips and precautions

When a movie clip is playing you may want to hide the mouse cursor to prevent distractions. The Mouse.hide() method provides an easy way to hide the cursor, until displayed again with a Mouse.show() method or the movie clip termination. You can turn the mouse cursor on and off as many times as you want in a movie clip.
