Sound

This object is addressed in Chapter 5.

 

This object is used when you want to dynamically add sound to your Flash movie at runtime. With this object you can add sound, load sounds from files on a server, and among others alter the volume of the sound. You should use this method to retain ultimate control over your sound, rather than applying a sound to a frame, which gives you limited options at runtime for dynamically controlling the sound. See also Sound Constructor and Chapter 5.

 

The object is compatible with Flash 5 and above. There are no known issues with any version of Flash that supports this object.

Description

The Sound object is used to add and control sound elements in a Flash movie via ActionScript, at runtime.

Examples and practical uses

See Sound Constructor, Sound.loadSound and Sound.attachSound.

Tips and precautions

Flash can play both stereo and mono sounds. Since stereo sounds contain twice as much data they are therefore twice the size, in bytes. Be aware of your target audience and if stereo sound is really necessary.
