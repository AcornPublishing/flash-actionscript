XML.removeNode

myXMLNode.removeNode()

 

Where myXMLNode is an instance of the XML object.

 

This method is compatible with Flash Player 5 and later. It is used to remove a node from part of the XML instances data structure. See related entries on XML.childNodes and XML.appendChild.

Description

The XML.removeNode() method removes the specified XML node object and all of the children and descendents from the specified node. Following this deletion, all of the properties of remaining parent nodes will be updated.

myXML_xml.firstChild.childNodes[0].removeNode();

Does not indicate a node to remove, so none are removed and the entire string is returned.

 

Examples and practical uses

The XML.removeNode() method removes a node (and any descendants of that node) from its parent. You need to navigate to the specific node in order to remove the array of child nodes. To illustrate the deletion of a node from a string, we can show a structure before and after the removal of the node. This is the example xml.removenode.fla and xml.removenode.swf file .

 

myXML_xml = new XML("<root><child><sock color=\"red\" /></child></root>");

trace ("myXML_xml (pre):\n"+ myXML_xml.toString() +"\n\n");

myXML_xml.firstChild.firstChild.childNodes[0].removeNode();

trace ("myXML_xml (post):\n"+ myXML_xml.toString() +"\n\n");

 

The first data traced appears as the following:

 

myXML_xml (pre):

<root><child><sock color="red" /></child></root>

 

After the node is released, the following is traced:

 

myXML_xml (post):

<root><child /></root>

Tips and precautions

The XML.removeNode() method will remove any descendants of the removed node automatically. Take care to make sure you want to remove all child nodes from the node that you specify using this method. This method can be applied to either the top level XML class, or the XMLNode class.
