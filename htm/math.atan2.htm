Math.atan2

a = Math2.atan(y, x);

 

Where a is the variable that you want to equate to the arc tangent of the ratio between y and x. The ratio is computed as y/x by the function, and then a simple atan is applied to that ratio.

 

This method is compatible with Flash 5 or later.

Description

This method will return the arc tangent value of the two numbers passed to it. Usually, these two numbers will be distances along the x-axis and the y-axis. It is these two distances, which are required in order to compute the desired angle.

 

 

Code

 

 

Notes

 

a = Math.atan2(0, 0);

 

b = Math.atan2(-1, -1);

 

c = Math.atan2(1, 1);

 

d = Math.atan2(1, 0);

 

e = Math.atan2(0, 1);

 

 

 

In the first line, a will be set to 0.

 

In the second line, b will be set to

-2.35619449019234 radians, which is -135 degrees (a 45 degree angle down and to the left).

 

In the third example, c will be set to 0.785398163397448 radians, which is 45 degrees.

 

In the fourth example, d will be set to 1.5707963267949, which is one half of the mathematical constant, PI. This is 90 degrees straight down.

 

In the fifth example, e will be set to 0, which 0 degrees because it’s a horizontal line.

 

 

f = Math.atan2 ("text", "text");

 

If anything other than two numbers are passed into the function, then Flash will return NaN; which stands for "Not a Number".

 

Examples and practical uses

Math.atan2 is generally used to find an angle between two points. This means that we can easily move something along a trajectory between two points. Here’s an example:

 

// two points: p1.x, p1.y and p2.x, p2.y

xdist = p2.x - p1.x;

ydist = p2.y - p1.y;

 

angleFrom_p1_To_p2 = Math.atan2(ydist, xdist);

 

The angle between two points is taken relative to an imaginary axis.

 

 

In the previous example, the angle returned will be the angle from p1 to p2. If we reverse the distance equations, like so:

 

xdist = p1.x ­ p2.x;

ydist = p1.y ­ p2.y;

 

Then the angle returned will be from p2 to p1.

 

 

The angle returned by Math.atan2 in the range of ­PI/2 to PI/2, or ­180 to 180 degrees.

 

To see this well demonstrated in action, look at the example math.atan2.fla and math.atan2.swf .

Tips and precautions

Remember that in the Math.atan2 function, the y parameter is passed in first, followed by the x:  Math.atan2(y, x). Reversing these is easy, and will cause the results returned to be incorrect, and somewhat confusing.

 

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
