MovieClip.getDepth

myMovieClip.getDepth()

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip.getDepth()method returns the level of the movie clip instance.

Description

The MovieClip.getDepth()method returns an integer indicating the level (depth) of the movie clip. The return is an integer.

 

 

Code

 

 

Notes

 

 

myDepth = myMovieClip.getDepth();

 

 

 

Returns the depth of myMovieClip.

 

The return from the method is an integer.

 

Examples and practical uses

You can read all the occupied depths in a timeline by treating the timeline as an object and the movie clips on it as properties. By cycling through all the properties of _root and using the MovieClip.getDepth method on those that turn out to be movie clips, you can read all the occupied depths:

 

_root.createEmptyMovieClip("one", 23);

_root.createEmptyMovieClip("two", 24);

//

//

//

for (i in _root) {

    if (typeof (_root[i]) == "movieclip") {

        trace(_root[i].getDepth()+" is an occupied depth");

    }

}

Tips and precautions

The maximum depth value supported is approximately 16000, although the Flash player will usually slow down considerably if you actually try to add this many clips onto the stage (don’t try it manually!).

 

Try using the above script on a timeline that has graphics that were placed manually onto the stage to see how depth works for those movie clips that you can’t seem to be able to delete dynamically at first sight.

 

As you will probably realize, movie clips placed on stage at author time have negative depths.
