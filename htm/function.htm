function/Function

function [a]([b,c,n]){

   statements

}

 

Where a is the optional name of the function, and b through n are optional parameters.

 

The Functionobject is compatible with Flash MX. The functionaction is compatible with Flash 5 and above. Functions should be used when you are repeatedly performing the same task. Also refer to the return action.

Description

The Function object contains the methodsFunction.apply, Function.call and property Function.prototype. The Function object is exclusive to Flash MX, and its methods and properties are used to call or invoke functions. See the following entries Function.apply and Function.call for more information on this object. Thefunction action is used in programming when the same block of code is needed many times during the application. A function is used for centralizing and reusing code, and functions can be found in other languages such as JavaScript and other Object-oriented programming languages. These kind of functions contain a series of statements, which define a task to perform.

Examples and practical uses

function actions are typically declared and placed in one central area. Different parameters can be set in this location, which can help when you have different situations you want to use the block of code in. During your movie, you can then call upon the function(s) as many times as needed. Therefore, functions are typically used when you are continually performing the same task.

Programming

In our example function.fla and function.swf , a contact list can be displayed and updated within a movie. The following code illustrates how functions can simplify complicated code by putting repetitive actions into a function.

 

function clearMessage() {

   _root.message.text = "";

   clearInterval(messageTimer);

}

 

Also, inline functions are created to change the properties in the function object.

 

this.setName = function(newName) {

   this.Name = newName;

};

 

This is expanded upon and explained in the example, function.fla, where several different types of functions are created.

Motion Graphics

Motion graphics can use functions to create objects or graphics on the fly. This code placed within a function can be reused during your movie as many times as necessary to achieve the desired effect.

Tips and precautions

When using functions to return values, you need to use the return action within the function. Anything after the return action will not be processed by the function.

 

It is good practice to use few lines of code on buttons, movie clips and components. Whenever possible, create a function and call it when the button is pressed.
