FComponent.setSmallScroll

(This method is applicable to FScrollbar.setSmallScroll. )

 

myScrollBar.setSmallScroll(integer);

 

Where integer is an integer value that specifies how many units the scrollbar will move by when the user clicks on the scrollbar’s up and down arrows.

 

This method is compatible with Flash 6.

Description

The small scroll value defines how far the scrollbar will move when you click on the up or down arrows of the scrollbar. When attached to a text field, the default value is one line.

 

 

Code

 

 

Notes

 

myScrollBar.setSmallScroll(3);

 

Sets the small scroll value of the scrollbar instance myScrollBar to 3.

 

Examples and practical uses

The default value set when you drop a scrollbar onto a text field is usually sufficient (and is actually the same sort of value that most operating systems use for their scrollbars, so most users will expect this value).

 

The only time you will ever need to change the properties of the scrollbar is when you are using the scrollbar as a spinner or slider (as you might want to for a volume control for example). The following script will set up a scrollbar (instance name spinner ) as a slider, and this will control the value displayed in a text field spinner_txt.

 

function change(component) {

    spinner_txt.text = component.getScrollPosition();

}

spinner_txt.text = 0;

spinner.setEnabled(true);

spinner.setScrollProperties(40, 0, 100);

spinner.setLargeScroll(10);

spinner.setSmallScroll(2);

spinner.setChangeHandler("change");

 

The scrollbar is set up to give values from 0 to 100. Clicking on the scroll track will advance the scrollbar value by 10, and clicking on the up or down arrows will advance it by 2.

 

See also the example file fcomponentsetsmallscroll.fla / fcomponentsetsmallscroll.swf.
