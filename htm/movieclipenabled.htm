MovieClip.enabled

myMovieClip.enabled

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip.enabled property indicates whether a button movie clip is enabled.

Description

The MovieClip.enabled property returns a Boolean that indicates whether a button movie clip is enabled or not. If they are not enabled, the button’s methods are unavailable and the event handlers do not receive Mouse events.

 

The MovieClip.enabled property has been inherited from the Button object. See also Button.enabled.

 

 

Code

 

 

Notes

 

 

myEnabled = myMovieClip.enabled;

 

 

 

myMovieClip.enabled = false;

 

 

Stores the current setting of the button movie clip to the variable myEnabled. myEnabled must be a Boolean.

 

Disables the button movie clip.

 

Examples and practical uses

The MovieClip.enabled property is a Boolean value that indicates whether a button movie clip is enabled to handle button events or not. By default, a button movie clip is enabled. If enabled (the property is true), the button’s methods receive button events. You can see the list of button events that are enabled/disabled by this method by looking at the events of the Button object (by ‘button events’ we mean all events that the Button and MovieClip objects have in common).

 

If the button movie clip MovieClip.enabled property is set to false, no button events are sent to the movie clips methods and the Over, Down and Up frames of the button are disabled. The value of the MovieClip.enabled property can be assigned to a variable:

 

myEnabled = myMovieClip.enabled;

 

Or you can use it in a conditional:

 

if (myMovieClip.enabled) {

 // code when button movie clip is enabled

}

 

The MovieClip.enabled property can be changed at any time:

 

myMovieClip.enabled = true;

 

… and the change takes effect immediately. Changing the MovieClip.enabled property has no effect on a movie: if the movie is playing it continues to play uninterrupted.

Tips and precautions

When the MovieClip.enabled property is false, the button movie clip is not included in any automatic tab ordering.
