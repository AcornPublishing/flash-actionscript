FStyleFormat.removeListener

a.removeListener(b);

 

Where a is the name of your style format, and b is an instance of a component to remove from the style format.

 

Compatible with Flash MX. This method is used to a remove a listener from a component instance. This is typically used when you need a component instance to listen to a different style format, or return to the globalStyleFormat. Related entries include FStyleFormat, FStyleFormat.addListener, and FStyleFormat.applyChanges.

Description

This method removes an instance of a component from listening to a custom or global style format.

Examples and practical uses

This method is used to remove a style format listener from a component instance. The component can be listening to the global style format, or a custom style format.

Programming

In the example fstyleformat.removelistener.fla and fstyleformat.removelistener.swf , we create a style format and apply it to several components on the stage. The instances include a radioGroup, which is a series of three radio buttons.

 

myStyle = new FStyleFormat({face:0x339966, shadow:0x33CC66, radioDot:0xCC0000});

myStyle.applyChanges();

myStyle.addListener(button1, button2, combo1, radioGroup);

 

A radio group can have listeners added as a group, but not removed in the same manner. Therefore, each must be removed as separate instances. In the following code, we remove listeners from each instance at the same time using a for...in loop and a new array :

 

//function to remove the listeners using the remove style button

function pressButton() {

   var removeStyle = [button1, button2, combo1, radio1, radio2, radio3];

   for (i in removeStyle) {

      myStyle.removeListener(removeStyle[i]);

      newStyle.removeListener(removeStyle[i]);

   }

}

 

This example is expanded on in the example FLA to include a second button, which will remove the listeners from all component instances, and simultaneously apply a new custom style format.

Tips and precautions

Removing listeners works slightly different from the addListener method. You can add listeners to radio button groups, but you must remove each instance separately. You can remove listeners separately in the following manner:

 

myStyle.removeListener(button1);

myStyle.removeListener(button2);

myStyle.removeListener(button3);

 

Or, you can build it into a function, or use the method in the previous example FLA file. It is important to remember to applyChanges to the style format when adding or removing listeners.
