TextField.variable

myTextField .variable

 

Where myTextField is a TextField instance

 

Compatible with Flash Player 6 and later .

 

The TextField.variable property returns the name of the variable associated with the text field.

Description

The TextField.variable property returns the name of the variable that is associated with the current text field instance, returning the result as a string.

 

 

Code

 

 

Additional Explanation

 

 

myFile = myTextField .variable;

 

 

Stores the name of myTextField in myFile .

 

 

myFile = myTextField .variablel();

 

 

There is no method with this name.

 

Examples and practical uses

The TextField.variable property returns the name of the variable associated with the text field instance, returning as a string. This can be used in a variable:

 

myFile = myTextField .variable;

 

See the example textfield.variable.fla and textfield.variable.swf .

Tips and precautions

The TextField.variable property is included for backward compatibility with Flash 5. Flash 6 ActionScript is a method-based language, and you are strongly recommended to use the method based version, TextField.text . See the entry for TextField.text , particularly the Tips and precautions section, which explains why it is better than TextField.variable .
