Selection.getBeginIndex

Selection.getBeginIndex()

 

This method is compatible with Flash 6 and later.

 

The Selection.getBeginIndex() method returns the number of the index at the start of the selection span.

Description

The Selection.getBeginIndex() method returns the index number at the start of the selection span. If no text field has focus, or there is no selection -1 is returned. Finally, if a text field does have focus but there are no characters in that text field, the current value of Selection.getCaretIndex() is returned.

 

 

Code

 

 

Additional explanation

 

Notes

 

selspan1 = Selection.getBeginIndex();

 

 

Stores the starting index of the selection span in selspan1.

 

 

If no field has focus a value of -1 is returned.

 

 

selspan1 = Selection.getBeginIndex() - 1;

 

Stores one place to the left of the current position in selspan1.

 

 

Examples and practical uses

The Selection.getBeginIndex() method is used to return the index of the first object in a selection span. Selection spans are zero-origin. If there is no object with focus, the Selection.getBeginIndex() method returns a value of –1. To use the Selection.getBeginIndex() method you can assign the result to a variable:

 

selspan1 = Selection.getBeginIndex();

 

With the Selection.getBeginIndex() method and Selection.getEndIndex() method together, you can determine the complete span of selected characters. See getbeginindex.fla / getbeginindex.swf for an example of using both methods.

Tips and precautions

The Selection.getBeginIndex() method is used to determine the starting index of a selection span. If there are no indexes or no object has focus  –1 is returned and should be tested for when using the Selection.getBeginIndex() method.
