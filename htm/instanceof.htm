All movie clip instance names in root, and returns true for all movie clip instances on the stage.

 

This will trace the names of all movie clip objects into the output window, and a value of true for the instances which are movie clips.

 

function newCat(whiskers) {

this.whiskers = whiskers;

}

var myCat = new newCat("Buddy");

trace (myCat instanceof newCat);

 

 

true

 

This code will trace to see if myCat is an instance of the new newCat class. myCat therefore is an instance of newCat and Buddy is a parameter to the function.

 

Examples and practical uses

You can use instanceof to check that particular instances are part of a particular class, such as a text field or movie clip. This is useful when you need to check what kind of class an instance belongs to before manipulating it in a certain way using ActionScript.

Motion Graphics

In the example instanceof.fla and instanceof.swf , the following code is used to check if an instance on the stage belongs to the MovieClip class. Using this functionality can be useful when working with motion graphics to determine what is on the Stage at any given time. The following code is included in the example file:

 

function createContact(name) {

   this.name = name;

}

var contact1 = new createContact("billy");

trace(contact1 instanceof createContact);

for (i in _level0) {

   trace(_level0[i]+" is "+(_root[i] instanceof MovieClip));

}

 

This code does two things. First of all, a function createContact is created. An instance billy is created, and assigned to the variable contact1. Then all instances are traced to illustrate how this works. This is elaborated upon in the example . Note that instances of a given class will only return true if it is constructed using the new operator. In this code, we create a new instance of the createContact class by calling our new createContact("billy") constructor. Or, for example, you could create a new instance myClip in the MovieClip class by using:

 

myClip=new MovieClip();

Tips and precautions

This operator is one of the few pieces of ActionScript that is case sensitive. If you try to use instanceOf, your script will throw an error. All characters in instanceof must be lower case. Also, when you are working with components, you should remember they are part of the MovieClip class. Refer to the operator typeof for similar functionality.
