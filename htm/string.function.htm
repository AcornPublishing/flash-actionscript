String (function)

var myString = String(expression);

 

Where expression will be converted into a string and will be the value of the variable myString.

 

Compatible with Flash 4 and above. There are no known issues with any version of Flash that supports this action.

 

The String function will convert the value of expression into a string. Based on the data type of expression different strings will be returned. This function is depreciated in favor of the String object.

Description

Will convert expression into a string.

 

 

Code

 

 

Notes


myString = String(10);
myString = String(“12”);
myString = String(“str”);

 

 

// returns “10”
// returns “12”

// returns “str”


myObject = new Object();

myString = String(myObject);

 

mySound = new Sound();

myString = String(mySound);

 


// returns “[object Object]”

 

 

// returns “[object Object]”

Examples and practical uses

General

This function is good for evaluating an expression, however it is depreciated.

Data type

The following is a table of returned values based on the data typed which is passed as expression.

 

 

Data type

 

 

Returns

 

Boolean

The string “true” or “false”.

Number

The same number, only as a string.

String

The same string.

Object

The string “[object Object]”.

Movie Clip reference

The target path of the movie clip in dot syntax.

Tips and precautions

Depreciated

The String function is now depreciated. You should use various other Flash MX methods to complete the same task that you would normally do with String.
