Math.max

a = Math.max(b, c);

 

Where a is the variable that you want to equate to the larger value of borc.

 

This method is compatible with Flash 5 or later.

Description

This method will compare both b and c, and will return the larger of the two numbers. The method expects two numbers.

 

 

Code

 

 

Notes

 

a = Math.max(0, 5);

 

b = Math.max(3, 3);

 

c = Math.max(-1, -2);

 

 

 

In the first line, a will be set to 5, because 5 is the larger of 5 or 0.

 

In the second line, b will be set to 3, because both numbers are equal to 3.

 

In the third example, c will be set to –1, because –1 is the larger of –1 or –2.

 

 

d = Math.max ("h", 3);

 

If anything other than a number is passed into the function, then Flash will return NaN; which stands for "Not a Number".

 

Examples and practical uses

To see Math.max in action, look at the example math.max.fla and math.max.swf . This demonstration shows how we can use Math.max to determine the placement of a movie clip relative to other movie clips.

 

If we want to determine the maximum between more than two values we can do this:

 

a = Math.max(Math.max(b, c), d);

 

these combined statements will determine the largest of b, c, or d.

Tips and precautions

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
