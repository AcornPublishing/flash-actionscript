TextField.textWidth

myTextField .textWidth

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.textWidth property sets the width of the individual character of the text field.

Description

The TextField.textWidth property is the width of any text content in a text field.

 

 

Code

 

 

Additional Explanation

 

 

myPath = myTextField .textWidth ;

 

 

myTextField .textWidth = 10

 

 

Stores the width property of myTextField in myPath .

 

Sets the text width to 10.

 

myPath = myTextField .textWidth() ;

 

 

There is no method with this name

 

Examples and practical uses

The TextField.textWidth property is the width of any text in a text field. It can be read and stored in a variable or set as a number:

 

myPath = myTextField .textWidth;

myTextField .textWidth = 6;

 

See the example textfield.textwidth.fla and textfield.textwidth.swf .

Tips and precautions

The TextField.textWidth property lets you set the width of the text in a text field. Along with the other TextField properties this lets you completely specify the appearance of text in a text field.
