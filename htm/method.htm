method

object.method = function {...};

 

Where object is an object name, method is a method name, and function is a function name

 

Compatible with Flash Player 6 and later.

 

The method action (method is not a reserved work but a placeholder for a method name) allows you to define methods for object when developing code in normal mode.

Description

The method action is used in Flash’s normal mode to allow you to define a method for an object. The keyword method is not used: instead you use the method’s name and define it with a function. Any number of methods can be defined for an object.

Examples and practical uses

Defining a method for an object, such as:

 

myObject.myMethod = function {

 // code for the function

}

 

You can define any method you want for an object. Instances of that object will inherit all the methods defined for the object.

Tips and precautions

You do not need to use the word “method” as part of a method definition.
