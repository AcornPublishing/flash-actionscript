false

 

This literal value is compatible with Flash 5 or later.

Description

The false literal value corresponds to the Boolean state ‘false’.

 

 

Code

 

 

Notes

 

a = 5==6;

 

a will be returned as false because 5==6 is a false statement: 5 is not equal to 6.

 

 

a = 6;

b = 6;

c = (a != b);

 

 

c will be returned as false because a!=b is a false statement: 6 is not unequal to 6.

 

 

a = 6;

b = false;

c = a+b;

d = a*b;

 

 

c = 6

d = 0

 

If you treat false as a number, it will act as if it were the number zero.

 

Examples and practical uses

false and the opposite Boolean value true are most often used when you use branching or decision making structures. For example, the if…else structure will execute code section A if x<2 is a true statement, and code section B if it is a false statement:

 

if (x<2) {

    // code section A

} else {

    // code section B

}
