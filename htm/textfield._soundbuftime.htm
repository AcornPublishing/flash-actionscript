TextField._soundbuftime

myTextField ._soundbuftime

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField._soundbuftime property specifies the number of seconds of sound to load before starting to play the sound.

Description

The TextField._soundbuftime property is used to set the number of seconds of time to buffer before a sound starts playing. If you define 10 seconds, the sound will load in 10 seconds worth of sound before it starts playing. It is useful over slow or unreliable connections.

Examples and practical uses

The TextField._soundbuftime property lets you set a buffer size for sound (in seconds) prior to starting to play a streamed sound. To assign a value to the property, simply use a standard assignment:

 

myTextField ._soundbuftime = 10;

 

This will buffer 10 seconds of sound prior to starting playback. To use the TextField._soundbuftime property in other commands, assign the result of the property to a variable:

 

myBuffer = myTextField ._soundbuftime;

Tips and precautions

The TextField._soundbuftime property is usually used to ensure a specific number of second's worth of sound have been buffered prior to playback to prevent interruptions or pauses in the playback of a movie. This is especially useful when streaming over low-speed connections where real-time updates are unlikely.
