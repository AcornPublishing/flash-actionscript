Math.acos

a = Math.acos(b);

 

Where a is the variable that you want to equate to the arc cosine of b.

 

This method is compatible with Flash 5 or later.

Description

This method will return the arc cosine value of the number passed to it. The number passed in must be from -1.0 to 1.0. Generally, Math.acos returns an angle that you already have the cosine value of, and the angle is returned in radians.

Notes

 

a = Math.acos(0);

 

b = Math.acos(-1);

 

c = Math.acos(1);

 

 

 

In the first line, a will be set to 1.5707963267949, which is one half of the mathematical constant, PI.

 

In the second line, b will be set to 3.14159265358979, which is the mathematical constant, PI.

 

In the third example, c will be set to 0, because the arc cosine of 1 is 0.

 

 

 

d = Math.acos(2);

 

e = Math.acos ("text");

 

In the first example, d will be set to NaN, because 2 is not within the range of -1.0 to 1.0

 

If anything other than a number is passed into the function, then Flash will return NaN; which stands for "Not a Number".

 

Examples and practical uses

Math.acos is used to find an angle from an already-calculated cosine value. To see Math.acos in action, plotted in a graph see math.acosplot.fla and math.acosplot.swf .

 

The angle returned in this function will be in radians, rather than degrees. To convert radians to degrees it is necessary to multiply the radian value by the number 57.2957795130823. Or, simply and more accurately: 180 / Math.PI

 

So in the following example:

 

a = Math.acos(0.8);

b = a * (180 / Math.PI);

 

a will be an angle in radians, b will be the equivalent angle in degrees. The angle in this case is about 0.6435 radians, or 36.8699 degrees.

Tips and precautions

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
