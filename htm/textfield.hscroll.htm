TextField.hscroll

myTextField .hscroll

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.hscroll property sets or retrieves the current horizontal scrolling position.

Description

The TextField.hscroll property is used to both set and display the current horizontal scrolling position. If the value is set to zero, there is no horizontal scrolling.

Examples and practical uses

The TextField.hscroll property is used to set and retrieve the horizontal scrolling position, counting characters. If set to zero, there is no horizontal scrolling. To use the TextField.hscroll property to retrieve the horizontal position of a scrolled text field, assign the result of the property to a variable:

 

myXPos = myTextField .hscroll;

 

To set a position, assign a value to the property:

 

myTextField .hscroll += 10;

Tips and precautions

The TextField.hscroll property is handy in setting the amount of horizontal scrolling allowed in a text field, and to see how much scrolling has occurred in a text field string.

 

This property is associated with the creation of scrollbars. Because components provide this feature directly, the need to use hscroll is somewhat reduced.
