with

with (a) {

   b;

}

 

Where ais an instance of a movie clip or an object, and b is a statement or group of statements.

 

Compatible with Flash 5 Player and above. This action is used evaluate properties, variables or functions within a given object. Related entries include tellTarget.

Description

The with statement can be used as a shorthand method, so you do not have to continually reference properties of an object using a full scope. The with action is used to set properties, variable, or actions for any specified object in ActionScript.

 

 

Code

 

 

Notes

 

with (_root.mytext) {

   html = true;

   _alpha = 45;

   _x = 100;

   _y = 400;

   htmlText = "<i>hello</i> world";

}

 

 

This block of code is shorthand for the following code:

 

_root.mytext.html = true;

_root.mytext._alpha = 45;

_root.mytext._x = 100;

_root.mytext._y = 400;

_root.mytext.htmlText = "<i>hello</i> world";

 

Examples and practical uses

In the following example, we are setting the properties of myFormat to a new style format that is then applied to the dynamically created text field:

 

myFormat = new TextFormat();

with (myFormat) {

   font = "helvetica";

   size = 36;

   color = 0xff0000;

   bullet = false;

   underline = true;

}

_root.createTextField("myText", 1, 10, 10, 300, 50);

_root.myText.selectable = false;

_root.myText.text = "{logo goes here}";

_root.myText.setTextFormat(myFormat);

 

The above can be found in the example with.fla and with.swf . The with action can also be used to access the properties of other objects such as movie clips, buttons, or any other instance which can be controlled using ActionScript. This includes objects such as Math and String as well.

Tips and Precautions

If you need to set a variable within the with action, you must declare it outside the with action beforehand. Otherwise, you must enter the entire scope path of the timeline on which you want the variable to live.
