TextField.password

myTextField .password

 

Where myTextField is a TextField instance.

 

Flash MX and later. Compatible with Flash Player 6 and later .

 

The TextField.password property indicates the text field is to be treated as a password field.

Description

The TextField.password property is a Boolean that indicates the text field is to be treated as a password field and have the characters that are entered into that field hidden. If true , the field is a password field; if false the field is not a password field.

 

When the password property is set to true , the text is displayed using asterisks only.

Examples and practical uses

The TextField.password property can be used to hide the characters a user enters into the field. Then the TextField.password property is true , the characters are hidden. When false , characters are displayed normally. The TextField.password property is set by assigning a Boolean:

 

myTextField .password = true;

 

See the example textfield.password.fla and textfield.password.swf .

Tips and precautions

The TextField.password property lets you hide sensitive information such as passwords by not displaying them on the screen.
