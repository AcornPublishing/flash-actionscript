MovieClip.onRelease

myMovieClip.onRelease

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip.onReleaseevent handler is invoked when a mouse button is released on a button movie clip.

 

This event is the same as the corresponding Button event Button.onRelease. See the entry for Button.onRelease.
