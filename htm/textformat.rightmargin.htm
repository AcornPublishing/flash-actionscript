TextFormat.rightMargin

myTextFormat.rightMargin

 

Where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.rightMargin property specifies the right margin.

Description

The TextFormat.rightMargin property is a number indicating the number of points of space applied to the right of all text in a text field. The default is null which means that the rightMargin setting in the text field is retained.

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myTextFormat.rightMargin = 5;

 

5 points is used as a right margin.

 

A null value means no value.

 

 

Examples and practical uses

The TextFormat.rightMargin property number indicates the number of points of space applied to the right margin in a text field.

 

The following code sets up a TextFormat object called myTextFormat and applies it to a text field called my_txt . In the following images you can see t he before (top) and after (bottom) effects of this code:

 

myTextFormat = new TextFormat();

myTextFormat.rightMargin  = 30;

my_txt.setTextFormat(myTextFormat);

 

 

See the example textformat.rightmargin.fla and textformat.rightmargin.swf .

Tips and precautions

The rightMargin property is the dynamic version of the Right Margin value seen in the Format Options window (which you can access by hitting the Format... button in the property inspector).

 

 

The right margin is less obvious than the left margin, especially when the text is "ragged" (as shown by the example above).
