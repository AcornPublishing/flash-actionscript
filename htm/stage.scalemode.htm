Stage.scaleMode

Stage.scaleMode = “exactFit”;

 

Where the scaleMode property of the Stage object is being set to the string exactFit, a legal setting for the scaleMode property.

 

This property is compatible with Flash 6 and above. There are no known issues with any version of Flash that supports this object.

 

The scaleMode property is to define the way your movie scales when the stage size is altered. You can adjust the scaling mode to one of the following settings: “exactFit”, “showAll”, “noBorder” and “noScale”. Each of these legal string values will cause a different effect. If you try to set the scaleMode property to an illegal string value (or un-recognized string value), the scaleMode property will return to the default value, “showAll”. See also Stage.align.

Description

This method will alter the way your Flash movie scales.

Notes


Stage.scaleMode = “noScale”;

 

 

Setting the scaleMode property to a legal string value. The “noScale” value will stop the movie scaling within the size of the stage. If the movie is larger than the size of the stage, the movie will be cropped as need.

 

 

Stage.scaleMode = “wrong”;

 

Incorrect string value.

 

This will result in the scaleMode property equaling the default setting “showAll” because the defined string value, “wrong”, is not recognized (an illegal value).

 

Examples and practical uses

This property is used to alter the scaling mode of a Flash movie. The following is a table containing a list of the legal settings and there corresponding effect on the Flash movie.

 

 

Scale Mode

 

 

Effect

 

“exactFit”

 

Will stretch the Flash movie both horizontally and vertically so it equals the size of the stage. This setting will possibly distort the aspect ratio of the movie.

 

 

“showAll”

 

Will proportionately stretch the Flash movie to the stage. This setting will make the Flash movie as large as it possibly can be without distorting aspect ratio of the movie.

 

 

“noBorder”

 

noBorder is similar to showAll in that it will preserve the aspect ratio of the Flash movie, but different in this fashion it achieves this. Cropping of the movie is possible is noBorder, whereas showAll will always show the entire stage, regardless of size. See the example file below to see the difference in practice.

 

 

“noScale”

 

 

This setting will cause the movie not to scale.

 

 

See the example stagescalemode.fla / stagescalemode.swf to see the various different settings in action.

Tips and precautions

By default, the movie uses the HTML parameters set in the Publish Settings dialog box as the default for the scaleMode property. However, if you modify this property via ActionScript it will override the value set in HTML.

 

In Test Movie mode the default setting for the scaleMode is actually “noScale”, rather than “showAll”, which it is when being viewed via the plug-in in a browser or by the standalone player.
