Key.getCode

Key.getCode();

 

Compatible with Flash Player 5 and later. There are no known issues with any versions of Flash that support this method.

 

Key.getCode returns the virtual key code of the last key pressed or released. No parameters are allowed.

Description

Key.getCode returns the virtual key code of the last key press or key release. Virtual key codes are assigned to every key on a keyboard, regardless of operating system and platform, for which there is a Flash player.

Examples and practical uses

The Key.getCode() method returns the virtual key code value of the last key press or key release. Virtual key codes allow you to design a movie that will work the same regardless of operating system or hardware platform, as the virtual key codes are mapped to the same functions on each platform. A list of the keyboard mappings to virtual key codes is available in the Flash documentation. The result of this method is usually assigned to a variable:

 

LastPress = Key.getCode();

Tips and precautions

No parameters are allowed with this method.
