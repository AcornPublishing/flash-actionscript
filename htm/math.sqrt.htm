Math.sqrt

a = Math.sqrt(b);

 

Where a is the variable that you want to equate to the square root of b.

 

This method is compatible with Flash 5 or later.

Description

Computes the square root of b. The parameter, b, must be a value greater than or equal to 0.

Notes

 

a = Math.sqrt(1)

 

b = Math.sqrt(16);

 

c = Math.sqrt(46);

6.78233.

 

Examples and practical uses

To see Math.sqrt plotted on a graph see the example math.sqrtPlot.fla and math.sqrtPlot.swf .

 

The most common application of the square root function is to compute the distance between two points. This is done like so:

 

distx = p2x ­ p1x;

disty = p2y ­ p1y;

 

dist = Math.sqrt((distx * distx) + (disty * disty));

 

As long as we know our two points, (p1x, p2y) and (p2x, p2y), then we can use Math.sqrt to compute the direct distance between them.

 

Take a look at the example math.sqrt.fla and math.sqrt.swf to see this in use in a motion graphics example.

Tips and precautions

The Math.sqrt function can be a processor intensive function, so it’s a good idea not to perform a large number of them at one time, in one loop or frame.

 

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
