This property is compatible with Flash 4 and later.

Description

The _parent property refers to the parent of the current movie clip or object. You can use the _parent property to specify relative paths to movie clips or objects that are above the current movie clip or object.

_parent.prop

 

 

Refers to the prop property of the parent of the current object of movie clip.

 

Examples and practical uses

This is a very common property that is used whenever you need to refer to an object higher up the hierarchy. It has the advantage that code with in embedded movie clips will work wherever that clip is placed within a movie, whereas path related to _root could be broken if objects are relocated. For example:

 

_parent.gotoAndPlay(5);

 

… will move the parent timeline to frame 5 and start playing. Without looking at the structure of the movie this could refer to _root.gotoAndPlay(5);, _root.name.gotoAndPlay(5); or _root.name.name.gotoAndPlay(5);, and so on. We don’t need to know which by moving up the hierarchy from the object, rather than down the hierarchy from the _root.
