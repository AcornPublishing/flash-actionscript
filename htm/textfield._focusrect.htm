TextField._focusrect

myTextField ._focusrect

 

Where myTextField is a TextField instance

 

Compatible with Flash Player 6 and later .

 

The TextField._focusrect property indicates whether a text field that has a yellow rectangle around it when it has focus.

Description

The TextField._focusrect property returns a Boolean that indicates whether a text field has a yellow rectangle around it when focus is active. If set to true , the yellow rectangle appears; if set to false , no yellow rectangle is shown.

Examples and practical uses

The TextField._focusrect property is a Boolean value that indicates whether a text field that has focus has a yellow rectangle around it or not. By default, a text field has a yellow rectangle around it when focus is on, unless the global_focusrect property is set to false . If enabled (TextField._focusrect is true ), the text field shows the yellow rectangle. If the TextField._focusrect property is set to false , no yellow rectangle appears. The value of the TextField._focusrect property can be assigned to a variable:

 

myFocusEnabled = TextField._focusrect;

 

The TextField._focusrect property can be changed at any time:

 

myTextField ._focusrect = true;

 

and the change takes effect immediately.

 

The TextField._focusrect property overrides the global _focusrect property.

Tips and precautions

The TextField._focusrect property tells you whether a text field has a yellow rectangle around it when focus is active. The yellow rectangle is a convention many users are familiar with, and can help act as an aid for users. However, you may want to disable the yellow rectangle for cosmetic reasons.
