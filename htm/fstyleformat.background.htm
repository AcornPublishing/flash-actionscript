FStyleFormat.background

a.background = b;

 

Where a is the name of your style format, and b is a RGB value being assigned to the background region.

 

Compatible with Flash MX. This property only affects the background region of components supporting this property. The RGB color value assigned to this property is written using the 0xRRGGBB format. Refer to FStyleFormat.face and FStyleFormat.backgroundDisabled entries for information on how to change these elements.

Description

This property changes the color of the background area of the RadioButton, ListBox, and ComboBox components. It will affect the color of the background of the text area, and the background of the radio button and check box components.

Notes

 

myStyle = new FStyleFormat();

myStyle.background = 0xFF99CC;

myStyle.applyChanges();

myStyle.addListener (myComponent);

 

myStyle = new FStyleFormat({background: 0xFF99CC});

myStyle.applyChanges();

myStyle.addListener (myComponent);

 

This code will change the background color of myComponent by assigning a listener to the component instance and applying the changes using applyChanges();

 

Examples and practical uses

You can use this property to change colors in the text area of components, and the background of the button area in RadioButton component instances. To change the primary regions of other parts of the component, use the FStyleFormat.face property.

Programming

In the example fstyleformat.background.fla and fstyleformat.background.swf we assign a new color value to the background of the component instances myBox and myCombo. Then the text color is changed to white so it is still visible with this changed background color property. These changed property values are applied to the style format, and then add listeners for this custom style format to myBox and myCombo.

 

myStyle = new FStyleFormat();

myStyle.background = 0x333399;

myStyle.textColor = 0xffffff;

myStyle.applyChanges();

myStyle.addListener(myBox, myCombo);

 

Color changes are also applied to radio button instances and a check box on the stage in the example file.

Tips and precautions

You will want to make sure your text is still visible after altering the background color of the component text area. If you have changed the background color of your radio button instances, make sure the radioDot color value still has enough contrast with the background color to be legible.
