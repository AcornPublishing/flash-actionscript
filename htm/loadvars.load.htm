LoadVars.load

instancename.load(url)

 

where instancename is an instance of the LoadVars object and url is the location of the variables to be transferred

 

Compatible with Flash Player 6 and later.

 

The LoadVars.load() method loads variables into a LoadVars instance

Description

The LoadVars.load() method is used to retrieve variables and their values from a URL and place them in an instance of the LoadVars object. The downloaded variables are sent in application/x-www-urlform-encoded MIME format only. If existing variables with the same name are present in the instance, they are overwritten.

Notes

 

myMovieVars.load("data.txt");

 

 

 

Opens the MIME format data in data.txt, parses it, and places the resulting variable names and values in the myMovieVars instance

 

 

Existing variables of the same name are overwritten

 

 

 

Examples and practical uses

The LoadVars.load() method is used to transfer variables in standard MIME format from a URL into a LoadVars instance (the opposite of a LoadVars.send() method). The contents of the URL are translated automatically into variable and value formats for use in the instance by the.load() method. To use this method it has to be associated with an instance of the LoadVars object, and then can be called for that instance like this:

 

myMovieVars = new LoadVars();

myMovieVars.load("myData.dat");

 

If the file myData.dat is not in application/x-www-urlform-encoded MIME format, the method will fail.

 

This method is similar to the XML.load() method used by the XML object.

Tips and precautions

The load() method can only be referenced from an instance of the LoadVars object.
