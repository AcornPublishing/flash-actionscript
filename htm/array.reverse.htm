Array.reverse

myArray.reverse();

 

Reverses the element order of the array myArray.

 

This method is compatible with Flash 5 or later. It reverses the indexes of all elements in an array, so that the first element becomes the last one, and so on throughout the array.

Description

The reverse method allows you to reverse the order of an array, as shown by the example below.

 

 

Code

 

 

Notes

 

fruit = ["apples", "pears", "oranges", "limes"];

fruit.reverse();

trace(fruit);

 

 

When reversed, the fruit array becomes [limes,oranges,pears,apples].

 

Examples and practical uses

Most coders may prefer to leave the array in its original order and instead restructure the code to read the array in the desired direction. This is recommended ahead of the more error prone complete reversal of an array order using Array.reverse, particularly because Array.reverse will not reverse embedded arrays (see Tips and Precautions below).

Tips and precautions

Note that for multidimensional arrays, the reverse will only apply to the referenced array only, and any other nested arrays will maintain their order. For example, the code shown below:

 

complexArray = [[2, 3, 4], [4, 6, 8], [8, 12, 16]];

complexArray.reverse();

complexArray[2].reverse();
