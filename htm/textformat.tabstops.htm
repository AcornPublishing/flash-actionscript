TextFormat.tabStops

myTextFormat.tabStops

 

Where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.tabStops property sets custom tab stops.

Description

The TextFormat.tabStops property is an array of non-negative integers indicating tab stops, specified in points. The default is null meaning, which means that the tabStops setting in the text field is retained.

 

 

Code

 

 

Additional explanation

 

 

Notes

 

myTextFormat.tabStops = [5, 10, 15];

 

Tabs are placed at 5, 10, and 15 points.

 

Non-negative integers must be specified.

 

Examples and practical uses

The TextFormat.tabStops property number overrides the default tab stop value of 4 points and sets custom tab stops at any non-negative values supplied in an array. Only number values are allowed:

 

myTextFormat.tabStops = [10, 20, 25, 34, 46];

 

See the example textformat.tabstops.fla and textformat.tabstops.swf .
