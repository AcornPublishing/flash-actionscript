TextField.backgroundColor

myTextField .backgroundColor

 

Where myTextField is a text field instance.

 

Compatible with Flash MX and later .

 

The TextField.backgroundColor property defines the color of the background fill of a text field. In order for this property to take place, the TextField.background property must be set to true , if it isn't a background on the text field is not used. A hexadecimal value should be used, i.e. 0x000000 (black) or 0xFF0099 (hot pink).

Description

The TextField.backgroundColor property is used set the color of a text field's background.

Notes

 

myColorFill = myTextField .backgroundColor;

 

myTextField .backgroundColor = 0xFF0000;

 

 

Assigns the color of the background from the text field, myTextField , to the myColorFill variable.

Sets the background fill color to red.

 

 

The default color of all text fields backgrounds is white ( 0xFFFFFF ).

 

 

myTextField .backgroundColor ( 0xFF0000 );

 

 

The TextField object doesn't contain a backgroundColor method.

 

 

 

 

Examples and practical uses

You can change the background color dynamically to indicate a number of conditions. For example, you could:

 

·       Set the background to white if a text field has input focus, and gray if it doesn't. This concept is developed in the entry for TextField._alpha , and the code to do the same thing with the background color would be very similar.

·       Set the background color to signify the validity of the current input.

 

See the example textfield.backgroundcolor.fla and textfield.backgroundcolor.swf , this file explores the second concept.

Tips and precautions

In order for the TextField.backgroundColor property to visually take effect, the TextField.background property must be set to true . However, you can set the TextField.backgroundColor property beforehand or even if the TextField.background property is set to false , you just wont be able to see the color.
