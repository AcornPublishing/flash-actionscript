TextField.htmlText

myTextField .htmlText

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.htmlText property contains the HTML representation of a text field's contents.

Description

The TextField.htmlText property gives the HTML representation of a text field's contents. The TextField.html property must be true to allow HTML representations to be used . If the TextField.html property is false , any text in the field is interpreted literally (which means the HTML tags which are treated as straight strings).

Examples and practical uses

The TextField.htmlText property contains the HTML representation of a text field's contents. If the field has the TextField.html property set true , HTML conversion of tags is assumed; if false, then non-HTML is assumed and characters are treated literally. For example, this code sets the TextField.html property to true to allow HTML tags, and then specifies text that contains embedded bold characters:

 

myTextField .html = true;

myTextField .htmlText ="This is <b>bold</b> and so is <b>this</b>."

 

See the example textfield.htmltext.fla and textfield.htmltext.swf .

Tips and precautions

The TextField.html property is used with the TextField.htmlText property to allow you to use HTML representation inside a text field.

 

You should note that the HTML renderer in Flash is a very cut down version of the standard browser renderer. For example, Flash will not render links as a separate color for you, so you will have to add the additional HTML tags to do it. See also the entry for TextFormat.target , which shows how to dynamically format text (using the TextFormat object) and assign a url /target to each link so that it looks and acts like a HTML hyperlink.
