MovieClip.lineTo

myMovieClip.lineTo ( x, y )

 

Where myMovieClip is a movie clip instance and x and y are the coordinates of the end point.

 

Compatible with Flash 6 and later.

 

The MovieClip.lineTo()method draws a line from the current drawing position to the specified end point.

Description

The MovieClip.lineTo()method is used to draw a straight line between two points within the specified movie clip. The starting point is the current drawing position, and the end point is provided as parameters to the MovieClip.lineTo()method. All coordinates are relative to the registration point of the movie. The line is drawn with the current line style.

Notes

 

myMovieClip.lineTo(100,300);

 

Draws a line from the current drawing location to 100, 300.

 

 

The current line style is used.

 

 

Examples and practical uses

The MovieClip.lineTo()method draws a line from the current drawing point to the specified end point coordinates. If there is no current drawing point 0,0 is used. All coordinates are from the registration point of the movie. The line is drawn using the current line style (see the MovieClip.lineStyle()method to set the line style). After drawing the line, the current drawing point will be the end of the drawn line (the parameters specified as arguments to the MovieClip.lineTo()method).

 

This code sets a current line style, moves the drawing position to a specific point (see the MovieClip.moveTo() method) and then draws a box:

 

myMovieClip.lineStyle(15, 0x0000FF, 100);

MovieClip.moveTo(100,100);

myMovieClip.lineTo(300,100);

myMovieClip.lineTo(300,300);

myMovieClip.lineTo(100,300);

myMovieClip.lineTo(100,100);

 

All lines drawn with the MovieClip.lineTo()method are drawn underneath existing drawing content.

 

See the example file moviecliplineto.fla / moviecliplineto.swf. This shows the minimum code required to create a fast and responsive pencil tool. The MovieClip.mouseDown event starts the drawing, and the MovieClip.onMouseUp stops it. Lines are drawn every time the mouse is moved, and screen update is forced every time this occurs via the updateAfterEvent() action, which results in the fastest screen update that the users machine can produce irrespective of frame rate settings.

 

penDown = function () {

    this.lineStyle(0, black, 100);

    this.moveTo(_root._xmouse, _root._ymouse);

    this.onMouseMove = penDraw;

};

penUp = function () {

    this.onMouseMove = undefined;

};

penDraw = function () {

    this.lineTo(_root._xmouse, _root._ymouse);

    updateAfterEvent();

};

black = 0x000000;

_root.createEmptyMovieClip("paper", 0);

paper.onMouseDown = penDown;

paper.onMouseUp = penUp;
