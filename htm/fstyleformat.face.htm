FStyleFormat.face

a.face = b;

 

Where a is the name of your custom style format, and b is the RGB value assigned to the face property of this format.

 

Compatible with Flash MX. Using this property will change the color value of the face region used in scroll bar buttons, and push buttons. It will also modify any other developed components supporting this property. The RGB color value is assigned to the property using the 0xRRGGBB format. Related properties include FStyleFormat.textColor and FStyleFormat.arrow.

Description

This property of a style format sets the face region of "listening" component instances to the specified RGB value.

myStyle = new FStyleFormat();

myStyle.face = 0x99CCFF;

myStyle.applyChanges();

myStyle.addListener (myButton);

 

 

myStyle = new FStyleFormat({face:0x99CCFF});

myStyle.applyChanges();

myStyle.addListener (myButton);

 

These will change the color of the face region in the component instance myButton.

 

Examples and practical uses

You can use this property to change the color of the push buttons and scroll buttons, as we see in the example fstyleformat.face.fla and fstyleformat.face.swf . In this file, we change the color of the face regions in component instances button1, list1, and combo1 while simultaneously changing our textColor property so the colors contrast, and text is suitably visible.

 

newFaceStyle = new FStyleFormat({face:0x336633, textColor:0xCCCCFF});

newFaceStyle.applyChanges();

newFaceStyle.addListener(button1, list1, combo1);

Tips and precautions

When changing the color value of the face areas of your component(s), remember to test what the result looks like in comparison to your text color and arrow color values. Make sure your newly assigned color values contrast enough so the text or graphic is still visible against the face color.

 

This style property also changes the color value of the scroll thumb wheel (the slider that moves within the scroll track of a scroll bar). The thumb wheel is outlined with colors assigned to FStyleFormat.shadow, FStyleFormat.highlight, FStyleFormat.highlight3D, and FStyleFormat.darkshadow.
