Math.ceil

a = Math.ceil(b);

 

Where a is the variable that you want to equate to the ceiling of b.

 

This method is compatible with Flash 5 or later.

Description

This method will return the value of b rounded up to the nearest whole number that is greater than, or equal to b.

 

 

Code

 

 

Notes

 

a = Math.ceil(0);

 

b = Math.ceil(1.1);

 

c = Math.ceil(-1.1);

 

d = Math.ceil(9.0000001);

 

 

 

In the first line, a will be set to 0.

 

In the second line, b will be set to 2, because the next whole integer greater than 1.1 is 2.

 

In the third example, c will be set to ­1, because the next whole integer greater than ­1.1 is ­1.

 

In the fourth example, d is set to 10, because even though the fractional portion is very small, Math.ceil will still round up to the next greatest whole number.

 

 

e = Math.ceil ("text");

 

If anything other than a number is passed into the function, then Flash will return NaN; which stands for "Not a Number".

 

Examples and practical uses

To see Math.ceil plotted in a graph, see the example math.ceilplot.fla and math.ceilplot.swf .

Tips and precautions

Remember, this function does not round to the nearest whole number, it rounds to the next greatest whole number. The Math.round function will round to the nearest whole number.

 

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
