TextField._yscale

myTextField ._yscale

 

Where myTextField is a TextField instance.

 

Flash MX and later. Compatible with Flash Player 6 and later .

 

The TextField._yscale property sets or retrieves the vertical scale of a text field.

Description

The TextField._yscale property is used to determine the vertical scale of a text field's registration point, relative to the (0,0) default registration point. This lets you calculate positions of objects based on scaling effects.

Examples and practical uses

The TextField._yscale property is used set or retrieve the scaling of the vertical scale (which corresponds to the percentage of the axis used by the rescaled text field. For example, if a clip is reduced in size to 75% of the original, the TextField._yscale property will allow you to calculate new offsets from the registration point for buttons and other objects in the movie. To use the TextField._yscale property you assign it a value that then affects the values set by the TextField._y property.

 

To set the value of the TextField._yscale property, simply assign it a value:

 

myTextField ._yscale = 0.75;

 

To read the value, assign the property to a variable:

 

myTextField YScale = myTextField ._yscale;

 

You can use the TextField._yscale and TextField._yscale properties to allow the user to scale a text field using some method such as a button. For example, you could have code like this that works with a "zoom up" button:

 

on(release) {

   myTextField ._xscale *= 10;

   myTextField ._yscale *= 10;

}

 

which will scale the text field up in size by ten percent with every click of the button.

 

See the example textfield._yscale.fla and textfield._yscale.swf .

 

Tips and precautions

Using the TextField._xscale and TextField._yscale properties lets you adjust positions of objects using the TextField._x and TextField._y properties properly.
