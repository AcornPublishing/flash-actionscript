chr

a = chr(b);

 

Where b is a number. a will be returned as the corresponding string character.

 

This action is compatible with Flash 4 and above. It is deprecated in Flash 5 and above, and you are advised to use the equivalent Flash 5 String method shown below:

 

a = String.fromCharCode(b);

Description

The chr action is useful in accessing characters not normally found on a keyboard.

 

It is deprecated by the String.fromCharCode() method, and you are advised to use this alternative in all new Flash content. See also String.fromCharCode.

Examples and practical uses

This action is only useful when creating content for Flash 4. You should not otherwise use it.
