Selection.getCaretIndex

Selection.getCaretIndex()

 

This method is compatible with Flash 6 and later.

 

The Selection.getCaretIndex() method returns the position of the cursor in the selected text.

Description

The Selection.getCaretIndex() method returns the index number where the blinking cursor is currently positioned. If there is no blinking cursor (in other words, the text field does not have input focus)  –1 is returned. If the text field has focus but is empty, the value of zero is returned.

 

 

Code

 

 

Additional explanation

 

Notes

 

selcur = Selection.getCaretIndex();

 

 

Stores the index of the blinking cursor’s object in selcur.

 

 

If no cursor is present a value of -1 is returned.

 

Examples and practical uses

The Selection.getCaretIndex() method is used to return the index of the object that contains the blinking cursor. Selection-spans are zero-origin. If there is no object with a cursor, the Selection.getCaretIndex() method returns a value of –1. To use the Selection.getCaretIndex() method you can assign the result to a variable:

 

selcur = Selection.getCaretIndex();

 

The Selection.getCaretIndex() method is often used with the Selection.getBeginIndex() and Selection.getEndIndex(). See getcaretindex.fla / getcaretindex.swf for an example of using the Selection.getCaretIndex() method.

Tips and precautions

The Selection.getCaretIndex() method is used to determine the index of an object with a blinking cursor in it. If there are no cursor –1 is returned and should be tested for when using the Selection.getCaretIndex() method.
