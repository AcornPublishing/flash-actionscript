MovieClip.moveTo

myMovieClip.moveTo ( x, y )

 

Where myMovieClip is a movie clip instance and x and y are the coordinates of the point to move to.

 

Compatible with Flash 6 and later.

 

The MovieClip.moveTo()method moves the current drawing position to the specified point.

Description

The MovieClip.moveTo()method is used to move the drawing position to a new point. All coordinates are relative to the registration point of the movie. The line is drawn with the current line style.

Additional explanation

 

 

Notes

 

myMovieClip.moveTo(100,300);

 

Moves the current drawing location to 100, 300.

 

Coordinates are relative to the registrationpoint of the clip.

 

 

Examples and practical uses

The MovieClip.moveTo()method moves the current drawing point to the specified point coordinates. All coordinates are from the registration point of the movie. This code moves the drawing position to a specific point and then calls the MovieClip.drawTo()method to draw a box:

 

myMovieClip.lineStyle(15, 0x0000FF, 100);

MovieClip.moveTo(100,100);

myMovieClip.lineTo(300,100);

myMovieClip.lineTo(300,300);

myMovieClip.lineTo(100,300);

myMovieClip.lineTo(100,100);

 

The MovieClip.moveTo action is used to start a fill path (see MovieClip.beginFill) and also to draw lines and curves (see MovieClip.lineTo and MovieClip.curveTo).

Tips and precautions

To move to an absolute position you need to either use the coordinates of _root (_level0) or use MovieClip.globalToLocal to convert coordinates for you. When creating advanced applications using the drawing API, confusion will quickly set in unless you use a common coordinate system, and you are recommended to use the coordinates of the _root timeline. See the entry for MovieClip.globalToLocal for more details.
