TextField.onKillFocus

myTextField .onKillFocus( newobject )

 

Where myTextField is a TextField instance and newobject is the object (movie clip, button or text field) to receive input focus.

 

Compatible with Flash Player 6 and later .

 

The TextField.onKillFocus event handler is invoked when a text field loses keyboard input focus.

Description

The TextField.onKillFocus event handler is invoked when a text field loses keyboard input focus. This will most commonly occur when the user unselects the current text field by selecting another one, or does the same thing by hitting the TAB key. When hitting the TAB key, it is possible for the user to give a button or a movie clip input focus. The optional argument newFocus will give the name of the text field, movie clip or button that now has focus.

Examples and practical uses

The TextField.onKillFocus event handler is invoked when a text field loses keyboard input focus and transfers that focus to another object. The function defined as part of the event handler definition is launched on receipt of data and contains the new object to receive focus. To define the TextField.onKillFocus event handler, the function is written like this:

 

myTextField .onKillFocus = function (newobject) {

  // code for the focus change

}

 

The text field onKillFocus Event handler function has an optional argument, and this is equal to the object that has received focus.

 

onKillFocus is useful because once focus is lost on a text field, you know the user has finished their current text entry for the field, so the event can be used as a validation script (i.e. checking that what the user has just entered is a valid input, and marking the text field as incorrect if it is not valid.

 

See the example textfield.onkillfocus.fla and textfield.onkillfocus.swf .

Tips and precautions

When using text fields, you may want to alter the tab order (it is set to the order in which you placed the text fields, movie clips and buttons onto the stage by default). You can do this via the MovieClip.tabIndex , Button.tabIndex and TextField.tabIndex methods. You may also want to prevent movie clips and certain buttons and text fields from being in the tab order. To do this, see the entries for MovieClip.tabEnabled , Button.tabEnabled , and TextField.tabEnabled .
