play

play()

 

This action is compatible with Flash 2 and later.

 

The play() action starts the playback of a movie on the timeline.

Description

The play() action is used to start playback of a movie loaded in the timeline. The playhead will move across the timeline as the playback progresses. If the movie is already being played, issuing another play() action will have no effect. For a complete discussion on the use of this action, see the MovieClip.play() entry.

Notes

 

 

play();

 

 

Starts playback of the movie.

 

 

By itself the play() action starts playing the timeline that this action reside on. This action is usually encountered in an event handler for a button, in which case it will start playing the timeline the button is on.

 

 

with(root.myMovieClip(){

 play();

}

 

 

 

The play() action starts playing the timeline of the movie clip _root.myMovieClip.

 

Examples and practical uses

After issuing a play() action, when a movie clip arrives at the last frame, it will automatically repeat playback unless stopped. If there is a multi-scene timeline involved, when the playback reaches the end of a scene it will move on to frame 1 of the next scene and continue playback there. Although the play() action can be used in a very simple manner to begin playback on the timeline where the command is embedded, it can get more complex with nested timelines. For a more complete discussion of the play() action and examples, see the MovieClip.play() entry.
