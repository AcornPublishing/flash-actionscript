TextField.wordWrap

myTextField .wordWrap

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.wordWrap property determines whether the text field will wrap text or not.

Description

The TextField.wordWrap property is a Boolean that indicates whether the text field instance has word wrap capabilities or not. If set to false, word wrap is disabled; otherwise word wrap is active.

 

 

Code

 

 

Additional Explanation

 

 

myTextField .wordWrap = true;

 

The myTextField object allows text to be wrapped.

 

 

myTextField .wordWrap = "true";

 

Only Booleans can be used for this property.

 

Examples and practical uses

The TextField.wordWrap property controls whether a text field is to allow automatic word wrap. The TextField.wordWrap property can be assigned as a Boolean:

 

myTextField .wordWrap = false;

 

See the example textfield.wordwrap.fla and textfield.wordwrap.swf .

Tips and precautions

You can set this property manually at author time using the Line Type drop down menu on the property inspector.
