MovieClip.getURL

myMovieClip.getURL( URL [, target, varmethod])

 

Where myMovieClip is a movie clip instance, URL is the URL from which to get content, target is the window or frame in which the content is loaded, and varmethod is the method of transferring variables (GET or POST).

 

Compatible with Flash 5 and later.

 

The MovieClip.getURL()method loads a document from a URL into a particular location.

Description

The MovieClip.getURL()method loads a document (such as a data file) into a location that may be explicitly specified as an argument. The optional GET and POST methods indicate how the variables and their values will be sent to the movie.

Examples and practical uses

The MovieClip.getURL()method loads a document (usually containing variables and their values) from a URL into the movie clip. The MovieClip.getURL()method can also be used for passing variables to an application defined at the URL. The only required argument for the MovieClip.getURL() method is the URL of the document:

 

myMovieClip.getURL(“datafile.txt”);

 

The MovieClip.getURL()method allows you to specify a target for the incoming document. The target can be a name, a frame, an expression, or a reserved target name for the window or HTML frame the document should be loaded into. The reserved target names allows with the MovieClip.getURL() method are:

 

·       _blank – a new window

·       _parent – the parent of the current frame

·       _self – the current frame in the current window

·       _top – the top-level frame of the current window

 

When specifying a target, you also specify whether the GET or POST method is used to transfer the variables and the values in the document. This option is not required if there are no variables in the document. The GET method uses a URL descriptor while the POST method uses an HTTP header. If there are a lot of variables to be sent, POST is the better method. The code format is as follows:

 

myMovieClip.getURL(“datafile.txt”, _blank, GET);

 

… this will load the contents of datafile.txt into a new blank window using the GET method.

Tips and precautions

The MovieClip.getURL()method is a method of transferring data into a movie clip or to another application referenced at a URL (usually a data file). The MovieClip.getURL()method is similar to the getURL() method.
