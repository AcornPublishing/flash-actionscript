return

return [expression]

 

Where expression is the expression to evaluate and return as a value to the function.

 

This action is compatible with Flash 5 and later.

Description

The return action is used to return a value from a function. We can specify an expression to be returned, or the return action can be used with a parameter to signify the completion of the function called.

 

 

Code

 

 

Additional explanation

 

Notes

 

return;

 

 

return expression;

 

 

Returns execution to the calling code.

 

Returns the value of the expression to the calling code.

 

 

If the return action is used without any expression, it returns a null.

 

Examples and practical uses

The return action is usually included in a function to indicate a return to the calling code. For example, the code:

 

function myFunc(num1, num2){

 // code

 return;

 // more code

}

 

… will return from the function execution when the return action is reached. Any statements beyond the return action will be ignored. The return action can be embedded in a conditional to check for exit criteria.

 

An expression can be provided as a parameter to the return() action. If an expression (string, number, array or object) is provided, it is evaluated and the result returned to the calling code. For example:

 

function myFunc(num1, num2){

 // code

 return num1 + num2;

 // more code

}

 

will return the value of the expression to the calling code. When a value is returned from a function’s return action it should be assigned to a variable or used by some other action:

 

myVal = myFunc(5,6);

Tips and precautions

If the return action is used without any expression, it returns a null.
