Stage.removeListener

Stage.removeListener(myListenerObject);

var success = Stage.removeListener(myListenerObject);

 

Where myListenerObject will be removed as a listener from the Stage object and will no longer receive callback notification from the onResize event.

 

This method is compatible with Flash 6 and above. There are no known issues with any version of Flash that supports this object.

 

This method is used to remove a listener object from receiving callback notification from the onResize event of the Stage object. Once you have removed a listener object, you can re-add it using the addListener method. See also Stage.addListener, Stage.onResize.

 

The removeListener method returns a Boolean value true or false. true will be returned if the listener was successfully removed and false if it failed. This method will return false if the specified listener object has already been removed.

Description

This method removes a listener object from the Stage object.

This is the most common form of using the method.

 


clearListener = Stage.removeListener(myListenerObject);

 

The removeListener method will return true on success and false on failure. Assign the returned information to a variable to determine its success.

 

Examples and practical uses

The most common usage for this method is to deregister an object from receiving callback notification. See the example stageremovelistener.fla / stageremovelistener.swf.

Tips and precautions

As the _listener property of the Stage object can be accessed and modified directly. It is possible that a listener has already been removed, which will result in the method returning the Boolean false.
