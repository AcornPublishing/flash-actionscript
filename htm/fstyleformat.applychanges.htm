FStyleFormat.applyChanges

a.applyChanges(["b","c","d",n]);

 

Where a is a custom style format, and b,c,d, and n are optional text strings of FStyleFormatproperty names.

 

Compatible with Flash MX. This method will apply the changes you make to a style format. You use this after you have made changes to a style format. Refer to the related entry on FStyleFormat.addListener.

Description

This method will update and apply the changes you make to a style format. If you make changes to an existing format and do not include this method, no changes will be made to the appearance of the component instance.

This will apply changes to the specified properties. Properties are written in quotations.

 

 

myStyle.applyChanges();

 

 

nothing

 

All of the changed properties will be applied to the style format.

 

 

myStyle.applyChanges("myArrow", "myStyle");

 

 

nothing

 

Changes are made to the properties in the style format, and not the instances themselves. Listeners are used to apply style format changes to components.

 

Examples and practical uses

It is good practice to applyChanges whenever you make changes to a style format of any kind. When you make a change to a style format after it is listened to, you must apply the changes you make. In our example fstyleformat.applychanges.fla and fstyleformat.applychanges.swf we have instances on the stage, which are listening to custom style formats. Style formats are located in the actions layer. When a button is pressed on the stage, text in the myListbox instance will be resized. Therefore, we are changing the textSize set in the textStyle format the component is listening to. A function will assign and then apply these changes. The following function is in the functions layer of the FLA file, which is passed when the instance button1 is pressed:

 

function myButton() {

   textStyle.textSize = 15;

   textStyle.applyChanges ("textSize");

   myListBox.setSize (150, 100); //stops the box from shrinking

}

 

If you remove or comment out the line of code:

 

buttonstyle.applyChanges ("textSize");

 

from the previous function, button1 will no longer increase the size of the text. A different method of changing properties is also in this example FLA file.

Tips and precautions

Using applyChanges will register property changes to the specified style format. The style format is then updated with the changed values. Even when it doesn't seem necessary to use this method, it is a recommended practice to include it when constructing your style formats.

 

When you are changing a specific property but not other properties of the style format, it is recommended that you include it in your code. For example, if you are changing the color of your text, apply changes using: myStyle.applyChanges("textColor");
