Math.random

a = Math.random();

 

Where a is the variable that you want to equate to a random number between 0 and 1.

 

This method is compatible with Flash 5 or later.

Description

Computes a completely random, non-repeating number from 0, up to (but not including) 1. Between 0 and 0.99999, one might say.

 

 

Code

 

 

Notes

 

a = Math.random();

 

b = Math.random() * 10;

 

c = (Math.random() * 10) - 5;

 

 

 

 

 

 

In the first line, a will be set to a random number, such as:

 

0.956732779275626 or

0.106070402078331 or

0.761447730474174 etc.

 

In the second line, b will be set to a random number between 0 and 9.9999999.

 

In the third line, c will be set to a random number between ­5 and 4.9999999.

 

Examples and practical uses

When we want to determine a random number between a number range, say low to high, we must do this:

 

a = (Math.random() * (high ­ low)) + low;

 

This will return a random number from low, to almost high (high ­ 0.00000000001).

 

To see a motion graphics example of this function, look at the example math.random.fla and math.random.swf . To see Math.random plotted on a graph see the example math.randomplot.fla and math.randomplot.swf .

Tips and precautions

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
