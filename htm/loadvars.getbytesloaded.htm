LoadVars.getBytesLoaded

instancename.getBytesLoaded()

 

where instancename is an instance of the LoadVars object.

 

Compatible with Flash Player 6 and later.

 

The LoadVars.getBytesLoaded() method returns the number of bytes transferred.

Description

The LoadVars.getBytesLoaded()method returns an integer specifying the number of bytes transferred with a.send() or.sendAndLoad() method. If no.send() or.sendAndLoad() method has been invoked, or the operation is still underway, a value of undefined is returned. The LoadVars.getBytesTotal method can be used to determine the number of bytes that has been sent by a data transfer using one of the LoadVars methods.

 

 

Code

 

 

Additional explanation

 

Notes

 

transferred= myMovieVars.getBytesLoaded();

 

 

 

Determines the number of bytes sent with the last.send() or.sendAndLoad() method and returns that value as an integer, storing it in the variable transferred

 

 

If no operation has been conducted or the operation is underway, undefined is returned

 

 

 

Examples and practical uses

The LoadVars.getBytesLoaded method returns the number of bytes transferred using the.send() and.sendAndLoad() methods. This can be used to verify the number of bytes transferred. To use this method it has to be associated with an instance of the LoadVars object, and then can be called for that instance like this:

 

myMovieVars = new LoadVars();

// initiate the send method

transferred=myMovieVars.getBytesLoaded();

trace("Bytes transferred: " + transferred);

 

If the transfer has not started yet, or has not completed, the value of undefined will be returned by getBytesLoaded(). This can be checked using a conditional. You can determine the number of bytes to be sent prior to the operation by using the getBytesTotal() method.

Tips and precautions

To calculate the percentage of data that has been transferred, you can combine the.getBtyesLoaded() and.getBytesTotal() methods like this:

 

myMovieVars = new LoadVars();

myMovieVars.load("data.dat");

// the load could be a movieclip using loadMovie

// check to see if the load has started; if not it is undefined

if (myMovieVars.getBytesTotal() > 0){

   totalsize = myMovieVars.getBytesTotal();

   percentloaded = Math.round(MovieVars.getBytesLoaded / totalsize * 100);

   // code to show results

}

 

If you want to show a continually updated percentage on the screen, such as a thermometer bar, you would place this code in an event handler and add the code inside the condition to display the results.

 

The getBytesLoaded() method can only be referenced from an instance of the LoadVars object.
