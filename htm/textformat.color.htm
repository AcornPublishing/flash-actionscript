TextFormat.color

myTextFormat.color

 

Where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.color property indicates the color of text.

Description

The TextFormat.color property is a hexadecimal value used to indicate the color of text.

 

 

Code

 

 

Additional Explanation

 

myTextFormat.color = 0x000000;

Text black

Examples and practical uses

The TextFormat.color property indicates the color of text. Only RGB hexadecimal values are allowed, such as this command, which sets the color of text to black:

 

myTextFormat.color = 0x000000;

 

See the example textformat.color.fla and textformat.color.swf .

Tips and precautions

Both hexadecimal or normal numbers are allowed, but hexadecimal is closer to the internal numbering used by Flash to form digital color. If you want a quick way of converting a color value to a hexadecimal value, consider using the Color mixer panel. The hexadecimal value is on the bottom left of the fully expanded window (hit the little arrow at the bottom left if your mixer window is minimized). The value in the color mixer is shown in HTML format (where hexadecimal is preceded with a "# "). You should change this for "0x " in ActionScript.
