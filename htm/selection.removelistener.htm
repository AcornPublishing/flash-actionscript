Selection.removeListener

Selection.removeListener (listener_name)

 

Where listener_name is the Selection object to be removed from watching for focus changes.

 

This object is compatible with Flash 6 and later.

 

When a Selection object has been registered using the Selection.addListener() method and you want to remove it from the listener list, use Selection.removeListener(). See also the entry for Selection.addListener.

Description

Selection objects that have been registered to listen for focus using Selection.addListener() can be removed from the listener list using Selection.removeListener() method. If the listener is successfully removed, true is returned. If the listener did not exist on the list of listener objects, false is returned.

Notes

 

Selection.removeListener(myTextobject)

 

Removes myTextobject from the listener.

 

 

If myTextobject is registered and removed successfully, true is returned.

 

If myTextobject is either not removed or not registered as a listener, false is returned.

 

Examples and practical uses

Selection.addListener() is used to register a Selection object to detect focus and keyboard activities. To remove the Selection object from the listener list, use Selection.removeListener(). The method will return true if the removal was successful, false if not successful (for any reason). To remove the listener, specify its name as a parameter:

 

Selection.removeListener (myTextField);

 

You can test the result of the method using an if conditional to verify the removal.

Tips and precautions

Remove listener objects after they have served their purpose. Too many listener objects can slow a Flash movie down whenever focus is changed.
