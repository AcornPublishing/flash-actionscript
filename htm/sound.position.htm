Sound.position

var soundPosition = mySound.position;

 

Where the variable soundPosition will have a value equal to that of the number of milliseconds that the mySound Sound object has been playing for.

 

This property is compatible with Flash 6 and above. There are no known issues with any version of Flash that supports this object.

 

This read-only property will return a number, expressed in milliseconds, representing how long the sound has been playing. The position property will be reset to 0 each time the sound loops. The position property is useful for indicating to the user where the sound is currently at, compared with the length of the sound. The position property can be used with the duration property to determine the current position of the song in a percentage of the total and elapsed times.

Description

This property represents how long the sound has been playing, expressed in milliseconds.

Notes


var soundPosition = mySound.position;

 

trace(“the song has been playing for ” + soundPosition/1000 + “ seconds.”);

 

 

A number, expressed in seconds, will be sent to the Output window representing how long the sound has been playing.

 


mySound.position = 200;

 

Incorrect usage.

 

The position property is read-only and can therefore not be defined by the user.

 

Examples and practical uses

The position property is perfect for creating a completion bar. Coupled with a movie clip onEnterFrame event, the example soundposition.fla / soundposition.swf uses the position property to create a completion/progress bar.
