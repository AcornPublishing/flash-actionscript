MovieClip.setMask

MyMovieClip.setMask (maskclip)

 

Where myMovieClip is a movie clip instance and maskclip is the name of the movie clip to be used as a mask.

 

Compatible with Flash 4 and later.

 

The MovieClip.setMask()method is used to set one movie clip as a mask for the movie clip instance.

Description

The MovieClip.setMask()method allows you to designate another movie clip that will be used as a mask for the current movie clip, showing only specific areas depending on the content of the mask clip. The MovieClip.setMask()method is used for multi-layer complex movies where you want to block off (mask) parts of a clip. Changes to MovieClip.setMask()method take effect immediately.

Examples and practical uses

The MovieClip.setMask()method lets you specify a movie clip that will be used as a mask for the movie clip calling the method. Masking allows you to block off areas of the movie clip with an opaque mask, allowing only certain regions to show through. The MovieClip.setMask()method is called by specifying the movie clip that is to be used as a mask:

 

myMovieClip.setMask(myMaskMovie);

 

This command will set myMaskMovie as the mask for myMovieClip. A mask movie can be used to mask only a single movie (unlike mask layers, which can affect multiple layers at once). When set as a mask, the target mask movie remains a mask until another MovieClip.setMask()method call. To turn off a mask, use a null as the parameter (this will not affect any movies in the timeline):

 

myMovieClip.setMask(null);

 

You can switch mask layers at any time as a movie plays, and the effect is immediate. For example:

 

myMovieClip.setMask(myMaskMovie);

myMaskMovie.setMask(myMovieClip);

 

… will start with myMaskMovie masking myMovieClip, then the second command sets myMovieClip to mask myMaskMovie.

 

The MovieClip.setMask()method is often useful for restricting the shape of the movie to a degree. For example, you could have a clip containing a circle moving around the stage, and use that as a mask for another layer to reproduce a spotlight effect:

 

myMovieClip.setMask(myCircleMovie);

Tips and precautions

A movie clip cannot mask itself.
