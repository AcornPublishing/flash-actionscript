TextFormat.leading

myTextFormat.leading

 

Where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.leading property indicates the leading vertical space between lines.

Description

The TextFormat.leading property is a number indicating the amount of leading vertical space that is to be applied between lines of text in a text field. The default is null which means that the leading setting in the text field is retained.

 

 

Code

 

 

Additional Explanation

 

 

myTextFormat.leading = 5;

 

5 points is used between lines of text as a leading vertical space.

 

Examples and practical uses

The TextFormat.leading property number indicates leading vertical space between lines. Only number values are allowed:

 

The following code sets up a TextFormat object called myTextFormat and applies it to a text field called my_txt . In the following images you can see t he before (top) and after (bottom) effects of this code:

 

myTextFormat = new TextFormat();

myTextFormat.leading = 5;

my_txt.setTextFormat(myTextFormat);

 

 

See the example textformat.leading.fla and textformat.leading.swf .

Tips and precautions

A leading value of zero does not mean no leading but use the default value set by the font.

 

The leading property is the dynamic version of the Line Spacing value seen in the Format Options window (which you can access by hitting the Format... button in the property inspector).

 

 

The leading value adds white space between lines of text and can make small text font sizes more readable. In general, as you reduce the font size, you should increase the leading value to maintain readability.
