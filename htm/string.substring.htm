String.substring

newString = myString.substring(start [, to]);

 

Where newString will contain a portion of myString from start to to.

 

The object is compatible with Flash 5 and above. There are no known issues with any version of Flash that supports this object.

 

The substring method is used to extract a portion of the specified string. A new string is created and the original string is not modified. If the to parameter is omitted, the length of the string is used ( myString.length ). See also String.slice and String.substr.

 

If the value of to is greater than the value of from the parameters are swapped before the function executes. This code demonstrates this:

 

myString = "Number: 12345, Name: John Doe.";

memberNumber = myString.substring(5, 0); // returns Numbe

memberNumber = myString.substring(0, 5); // returns Numbe

 

The following code illustrates that if the value of start is negative, the value 0 is used, unlike String.slice and String.substr :

 

myString = "Number: 12345, Name: John Doe.";

memberNumber = myString.substring(-5, 10);

Examples and practical uses

Search and replace

The substring method can be used to extract a string and manipulate the original string. The example stringsubstring.fla / stringsubstring.swf is a search and replace system.

Tips and precautions

Be careful to avoid confusion between String.substring, String.subtr and String.slice. These methods are all do the same thing, extract a slice or substring from a string, but they are implemented in different ways. See String.slice for comparison information.
