Boolean (as a function)

a = Boolean(b);

 

Where b is an expression, property, variable or value that you want to convert to a Boolean. a will be returned as the Boolean value true or false.

 

This global function is compatible with Flash 5 and later.

Description

The Booleanfunction will:

 

·       Assume all strings are false, except those that can be converted to a non zero number.

·       Return false for undefined and zero length strings.

·       Return true for all non-zero numbers (or expressions that evaluate to non zero numbers).

·       Return false for zero.

 

 

Code

 

 

Notes

 

x = Boolean(0);

y = Boolean(1.2);

z = Boolean(-300);

 

x = false

y = true

z = true

 

 

a = Boolean("true");

b = Boolean(3<4);

c = Boolean("3<4");

d = Boolean("-300");

e = Boolean("300");

 

a = false

b = true

c = false

d = false

e = true

 

Examples and practical uses

The Boolean function is neither necessary (the ActionScript interpreter will convert to Boolean objects as required anyway), nor does it always give the results you would otherwise expect in some cases (Boolean("true") returns false for example). This function is not therefore widely used, and its use cannot be recommended because there are no advantages of doing so.

 

The Boolean() function only really exists to provide core functionality for other methods and properties of the Boolean object.

Tips and precautions

As mentioned above, you will never need to use Boolean as a function, and its use in such a way is not normally useful.
