TextField.setNewTextFormat

myTextField .setNewTextFormat(object)

 

Where myTextField is a TextField instance and object is the TextFormat object that contains the text properties.

 

Compatible with Flash Player 6 and later .

 

The TextField.setNewTextFormat() method sets a TextFormat object for new text.

Description

The TextField.setNewTextFormat() method sets a TextFormat object used for all new text inserted into the field. This applies to text typed by a user or inserted by a script with a TextField.replaceSel() method, for example. All assigned TextFormat properties are assigned to the new text entered into a text field.

Notes

 

myTextFormat.setNewTextFormat(myObject);

 

 

 

Sets the properties of the myTextFormat text field to those defined in myObject

 

 

The object is in an instance of the TextFormat object.

 

 

myTextFormat.setNewTextFormat();

 

 

No parameters are specified so the command fails.

 

 

 

 

Examples and practical uses

The TextField.setNewTextFormat() method sets the text field's text format using the TextFormat object. Any new text entered by the user or by a TextField method has this format applied automatically. The content of the TextField.setNewTextFormat() method can be obtained from the TextField.getNewTextFormat() method. To set the format assign the TextFormat object as a parameter

 

myTextField .setNewTextFormat(myTextObject);

Tips and precautions

The TextField.setNewTextFormat() method allows you to set the text properties for all text fields, or just a field at a time. Coupled with the TextField.getNewTextFormat() method you can copy text formats from one text field to others, or define the properties manually by instantiating a new TextFormat object.
