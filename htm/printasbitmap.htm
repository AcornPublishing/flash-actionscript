printAsBitmap

printAsBitmap ("target" [, "bounding box"])

 

Where target is the name of the movie clip to print and bounding box sets the area of the movie to print.

 

This method is compatible with Flash 4 and later.

 

The printAsBitmap() method lets you print some or all of a movie.

Description

The printAsBitmap() method is used to print some or all of a movie based on boundaries that can be set within the movie, treating the movie clip as a bitmap. The printAsBitmap() method will always print at the highest possible resolution the printer will support.

Examples and practical uses

In its simplest form, the printAsBitmap() method prints an entire movie. The command is used with the name of the movie, like this:

 

printAsBitmap("myMovie");

 

The entire movie will be printed by default. If you want to print only certain frames, use the #p frame label to indicate the frames to print.

 

The bounding box lets you specify an area of the movie frame to be printed instead of the entire frame. One of three parameters can be specified for the bounding box:

 

·       bframe – designates the bounding box of each printed frame is to be used as the print area for that frame (the printed area will change as the bounding boxes change from frame to frame).

·       bmax – designates a bounding box composed as a composite of all the bounding boxes of all printed frames. This is used when the bounding box shapes and sizes are not consistent across a movie.

·       bmovie – designates the bounding box of a specific frame as the print area for all printed frames in the movie. The specific frame should have a #b frame label to indicate it is the bounding box frame.

 

For example, to print all the frames marked with #p in a movie called myMovie, and to use the bounding box of the frame with the #b label, you would issue the command:

 

printAsBitmap("myMovie", "bmovie");

Tips and precautions

No printing can occur until the entire movie has been loaded. The Flash player supports both PostScript and non-PostScript printers, but non-PostScript printers require vectors to be converted to bitmaps before printing can occur. This can slow down the print request a little. See also the print(), printNum(), and printAsBitmapNum() methods.
