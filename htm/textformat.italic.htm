TextFormat.italic

myTextFormat.italic

 

Where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.italic property indicates whether the text is italicized.

Description

The TextFormat.italic property is a Boolean used to indicate the text is italicized. If true , the text is in italics: if false text is not italicized. The default is null , which means that the italic setting in the text field is retained.

 

 

Code

 

 

Additional Explanation

 

 

myTextFormat.italic = true;

 

Text is shown in italics.

 

Examples and practical uses

The TextFormat.italic property Boolean indicates whether italics is applied or not. Only Boolean values are allowed:

 

myTextFormat.italic = true;

 

See the example textformat.italic.fla and textformat.italic.swf .

Tips and precautions

You can only apply font face style changes (bold, italic, underline) if either:

 

·       Font outlines are not specified as embedded for the text field.

·       Font outlines are specified, and the font outlines for the text field's current font are actually embedded.

 

See also the discussion in TextFormat .
