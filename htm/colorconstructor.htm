Color constructor

myColor = new Color(a);

 

Where myColor is the name of a new color object that will be applied to the graphic object identified by the target path a.

 

This constructor is compatible with Flash 5 or later. In Flash 5, the Color object can be applied to movie clips only. In Flash 6, it can be applied to movie clips, buttons, or dynamic/input text fields. See the entry for Color to see how you can work around this via a wrapper.

Description

The Color object constructor creates a new color object, and applies it to the specified targeted graphic object. Upon construction, the color object will be initialized with the RGB color value of the target.

Notes

 

myColor = new Color(_root.my_txt);

myOtherColor = new Color(my_mc);

mySiteColor = new Color(_root);

 

The color object myColor will control the color of the text field object _root.my_txt. This will only work in Flash 6.

 

The color object myOtherColor will control the color of the movie clip object my_mc. Because this is a relative path, my_my must be on the same timeline as this code.

 

The color object mySiteColor is attached to the timeline _root. This will work because _root is really only a special case of the movie clip object. You can use mySiteColor to alter the overall color of your site (typically via tinting).

 

Examples and practical uses

You must create a color object and apply it to a target before you can use it.

Tips and precautions

When you apply a color object to a movie clip, remember that the color object will be applied to everything within that movie clip, including embedded movie clips. This can be very useful when you want to control the coloring of a number of things at the same time.

 

The constructed color object will exist on the timeline that it is defined and not the timeline of the target clip. This has a number of implications, including that if you remove the target clip, the color object will continue to exist.

 

If the target clip is removed, the color object will raise no errors if you continue to use it.
