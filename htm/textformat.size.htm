TextFormat.size

myTextFormat.size

 

Where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.size property indicates the point size of the font.

Description

The TextFormat.size property is a number indicating the point size of all text in a text field. The default is null which means that the size setting in the text field is retained.

 

 

Code

 

 

Additional Explanation

 

 

myTextFormat.size = 5;

 

5 point fonts are used.

Examples and practical uses

The TextFormat.size property number indicates the size of the font in number of points for text in a text field. Only number values are allowed:

 

myTextFormat.size = 10;

 

See the example textformat.size.fla and textformat.size.swf .

Tips and precautions

When using small text sizes consider:

 

·       Pixel snapping your text fields by making sure that the TextField._x and TextField._y properties are whole integer values. This will stop Flash blurring the text via anti-aliasing. this applies to all text where you have embedded font outlines.

·       Increasing the TextFormat.leading value as you decrease the font size. This increases readability of large blacks of text by increasing the whitespace between lines.
