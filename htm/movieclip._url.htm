MovieClip._url

myMovieClip._url

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 4 and later.

 

The MovieClip._urlproperty returns the URL of the SWF file for the current movie.

Description

The MovieClip._urlproperty returns full URL of the location of the SWF file that was downloaded for the current movie clip instance.

 

 

Code

 

 

Notes

 

 

myFile = myMovieClip._url;

 

 

Stores the URL of myMovieClip in myFile.

 

myFile = myMovieClip._url();

 

 

There is no method with this name.

Examples and practical uses

The MovieClip._urlproperty returns the full URL of the current movie’s SWF file. This can be used in a variable:

 

myFile = myMovieClip._url;

Tips and precautions

The MovieClip._urlproperty can be used either by assigning it to a variable or by reference.
