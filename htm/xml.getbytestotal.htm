XML.getBytesTotal

myXML.getBytesTotal()

 

Where myXML is an instance of the XML object.

 

Compatible with Flash Player 6 and above. The XML.getBytesTotal()method returns total number of bytes in a document, and is commonly used for preloading situations. Refer to the related entry XML.getBytesLoaded().

Description

The XML.getBytesTotal() method returns an integer indicating the total number of bytes that are in an XML document.

myXML_xml.load("mydoc.xml");

_root.onEnterFrame = function(){

trace(myXML_xml.getBytesTotal());

}

 

 

Returns the total number of bytes in the myXML document once it has loaded.

 

The return from the method is an integer. When tracing this block of code, you will receive undefined, then the total amount of bytes in the document.

undefined

 

Returning the initial value of undefined since it is not put on an enterFrame handler.

 

Examples and practical uses

The XML.getBytesTotal() method returns an integer, which is the total number of bytes in an XML document. This is usually stored in a variable.

 

stop();

myXML_xml = new XML();

myXML_xml.load("hamlet.xml");

this.createEmptyMovieClip("countme", 3);

this.onEnterFrame = function() {

   amtloaded = (myXML_xml.getBytesLoaded()/myXML_xml.getBytesTotal())*100;

   if (isNAN(amtloaded)) {

      _root.stop();

   } else {

      gotoAndStop(5);

   }

   if (amtloaded>=100) {

      delete this.onEnterFrame;

   }

};

 

getBytesLoaded() and getBytesTotal() are usually used together. See the example xml.getbytestotal.fla and xml.getbytestotal.swf for an example of how to preload XML documents.

Tips and precautions

The XML.getBytesTotal() method returns an integer showing the number of bytes in a document, so the method provides an easy way to determine when the loading of an XML document is complete.
