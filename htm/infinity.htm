Examples and practical uses

Infinity is properly part of the Math object. It cannot be expressed as a value, and because of this it cannot be traced as a value. However, it can be used symbolically and you can also cancel out Infinity and have a number returned. Unfortunately, only one undocumented method works, which is Math.infinity.

 

Inifinity can be used in Math calculations involving potentially infinite numbers. Refer to the Math object for more information on how to work with these calculations. However, it is important to use the Math.infinity method and subtract Infinity from itself if you plan to use Infinity. For example, if you subtract something from itself, you should get the number zero (or nothing). If you divide something by itself, you should get 1. You can use Infinity in this way to get zero or one. However, in Flash, this is only the case with Math.infinity when you subtract Infinity from itself. If you try to use Infinity in other ways, it will not produce an expected result. Refer to the sample file for an example of the possible combinations, and what results they return.

Tips and precautions

Infinity cannot be traced because it is a symbolic value. It can be used in this symbolic nature to return a result. Infinity can be cancelled out by using the Math.infinity method as mentioned earlier. If you subtract Infinity from itself, the result should be 0. For example, Infinity can be represented by 1/0, which is an undefined value. However, you should also be able to subtract it: Infinity-Infinity = 0. Similarly, if you divide Infinity/Infinity, the correct result value is 1.Unfortunately this is not the case with most of the methods mentioned above.
