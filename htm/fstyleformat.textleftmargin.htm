FStyleFormat.textLeftMargin

a.textLeftMargin = b;

 

Where a is the name of your custom style format, and b is a number measured in pixels to specify a margin.

 

Compatible with Flash MX. You should use this property to create a left margin width. The integer is measured in pixels. Related entries include FStyleFormat.textRightMargin and FStyleFormat.textIndent.

Description

This property specifies a left margin for text in all component instances listening to the custom style format.

 

 

Code

 

 

Shortcut

 

 

Notes

 

myStyle = new FStyleFormat();

myStyle.textLeftMargin = 7;

myStyle.applyChanges();

myStyle.addListener (myComponent);

 

 

myStyle = new FStyleFormat({textLeftMargin:7});

myStyle.applyChanges();

myStyle.addListener(myComponent);

 

 

This code will specify a left margin for the myComponent of 7 pixels width.

 

Examples and practical uses

Programming

In the example fstyleformat.textleftmargin.fla and fstyleformat.textleftmargin.swf , there are several component instances being manipulated using this property in two different style formats assigning a different value in each one.

Tips and precautions

If you set your textLeftMargin to a value of 0, it will not make the text flush with the left side of the component. This property can be particularly useful when working with other components and component sets loaded into Flash.
