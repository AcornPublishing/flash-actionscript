Array.sort

myArray.sort();

 

Where myArray is the function you want to sort. The sort will sort:

 

·       Numbers in ascending order, with myArray[0] been assigned the lowest value.

·       Strings in ascending ASCII order (or more correctly, the character order defined by ISO 8859).

 

If you want to use a sort on another order, you can define it using the following:

 

myArray.sort(myOrderFunction);

 

Where myOrderFunction is a function that defines a custom ordering scheme.

 

This method is compatible with Flash 5 or later. It allows you to reorder the position of the values that make up the array. This ordering can be user defined.

 

function myOrderFunction(a, b){

  // code

  return val

}

 

·       If value a should come before b in the list, then val should be positive.

·       If value a should come at the same position as b, then val should be 0.

·       If value a should come after b, then val should be negative.

Description

The Array.sort method allows you to sort the array. By default, it will sort numbers in numerical order, lowest first, or character order, lowest character first (this approximates alphabetical order, but is not true alphabetical order. See the < operator for more information on ASCII/ISO 8859, which is the character set used).

 

For example, the following code will sort an array myNumbers in ascending order:

 

myNumbers = [20, 40, 1, 0.2, 100, -5];

myNumbers.sort();

trace(myNumbers);

You can also use the default sort scheme to sort alphabetically, but be aware that for the sort to work, you should either make sure that all the characters are the same case, or force this to be the case by using the String.toUpperCase or String.toLowerCase methods on each element in the array before sorting. The following listing will perform an alphabetical sort, and the output is shown below:

 

myStrings = ["sham b", "glen rhodes", "john davey"];

myStrings.sort();

trace(myStrings);

 

 

If you want to define your own order, you have to use the form:

 

myArray.sort(myOrderFunction);

 

myOrderFunction is the name of a function. This should look something like this:

 

function myOrderFunction(a, b){

  // code

  return val

}

 

·       If value a should come before b in the list, then val should be positive.

·       If value a should come at the same position as b, then val should be 0.

·       If value a should come after b, then val should be returned as negative.

 

The following code will sort numbers in descending order:

 

function descending(a, b) {

  return (b-a);

}

myNumbers = [1, 2, 4, 7, 1234, 5632,7, 3211, 6];

myNumbers.sort(descending);

trace(myNumbers);

 

 

This works because if b is the greater of the two numbers, a positive number is returned, and a negative number returned if b is the lesser number. If b = a, then zero is returned.

 

You can also create sorts that are not based on size. The following sort will list even numbers, followed by all the odd numbers:

 

function evenOdd(a, b) {

 if (b/2 == Math.round(b/2)) {

return 1;

 } else {

    return -1;

 }

}

myNumbers = [1, 2, 4, 7, 1234, 5632, 7, 3211, 6];

myNumbers.sort(evenOdd);

trace(myNumbers);

 

Examples and practical uses

One of the prime uses of arrays is that it is easy to sort them into various orders. Ordering is even easier in Flash; all you need is the Array.sort method.

Tips and precautions

The sort method can be a little unpredictable when used with strings, because it is not really sorting based on alphabetical order, but is instead converting the characters in your strings to numbers, using the character number values as the conversion method.

 

To find the character code myCode of the first character in an array element myArray[n] for yourselfuse:

 

myCode = myArray[n].charCodeAt(0);

 

… or look up the value of each character using an ASCII table (search the web for ‘ASCII character table’ or ‘ISO 8859’).

 

Because uppercase characters A-Z come before lowercase characters in ASCII/ISO 8859, you may consider converting all your strings to the same case before sorting using either the String.toLowerCase or String.toUpperCase methods.

 

The Array.sort method changes the original array order. If you want to preserve the original order, copy the array and sort the copy. The easiest way to copy an array is to simply use:

 

myArrayCopy = myArray;
