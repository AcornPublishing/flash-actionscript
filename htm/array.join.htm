Array.join

myString = myArray.join();

myString = myArray.join(“seperator”);

myString = myArray.join(mySeperatorString);

 

Where “seperator” is a string literal, and mySeperatorString is a string variable.

 

This method is compatible with Flash 5 or later. myString will be returned as a single string consisting of all the elements of myArray, separated by commas (if you do not give the method any arguments). You can also define your own separator as a literal string (“seperator” ) or a string variable (mySeperatorString ).

Description

The join method converts an array to a single string.

 

 

Code

 

 

Notes

 

numbers = [2, 4.5, 6];

numString = numbers.join();

 

 

 

numString = “2,4.5,6”

 

Note that if you wanted to display numbers in a text field in the format shown for numString, you could do so much easier simply by displaying numbers in the text field.

 

 

menu = ["home", "about", "links", "contact"];

menuText = menu.join(" | ");

 

 

menuText = “home | about | links | contact”

 

The separator is defined as “ | “.

 

Examples and practical uses

The join method is useful when you want to format the contents of a string for display in a text field. The conversion of an array to a string can also be useful when you want to edit the characters, because you can apply the methods of the String object to the result. In the example below, the array myArray has text that has mixed capitalization styles. For the purpose of display, it may be neater to standardize them. Line 2 copies the array into a string months. Using the String.toLowerCase method, the style is quickly changed to lower case only.

 

myArray = ["JAN", "feb", "Mar", "APR", "may"];

months = myArray.join(" ");

months = months.toLowerCase();

trace(months);

Array.join() is usually used to format an array for output to print or a text field. See the files arrayconcat.fla / arrayconcat.swf or arraypush.fla / arraypush.swf for examples of the array.join() method being used to format an array for output to a text field.

Tips and precautions

Notice that the array you apply the join method to is preserved. The method creates a new string, and leaves the array as it is.

 

You can use control codes as your separator. The following code uses \n (new line) as its separator:

 

myArray = ["JAN", "feb", "Mar", "APR", "may"];

months = myArray.join("\n");

months = months.toLowerCase();

trace(months);

 

 

If you are using a text field set up to show HTML formatting, you can also add HTML as your separator.
