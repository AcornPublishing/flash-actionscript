XML.parseXML

myXML.parseXML([a])

 

Where myXML is an XML instance, and a is an optional XML data string to be parsed.

 

This method is compatible with the Flash 5 Player and above. XML.parseXML() is used to parse an internal (in the Flash movie, not loaded) string XML document text. See related entries on XML.load, XML.sendAndLoad, and XML.status.

Description

The XML.parseXML() method parses an XML structure and returns the result to the XML object instance, populating that object with the XML tree resulting from the parse operation. If there are any existing trees attached to the XML object, they are discarded.

Examples and practical uses

The XML.parseXML() method lets you parse an XML internal structure and attach the tree resulting from the parse operations to an object, replacing any existing tree. To parse a document called myXMLStr and attach the resulting tree to the existing myXML_xml, you would write the following ActionScript:

 

myXMLStr = "<hello world=\"true\" />";

myXML_xml = new XML();

myXML_xml.parseXML(myXMLStr);

trace(myXML_xml.toString());

 

Or, you may want to parse new data which has been entered into your Flash movie. For example, you could greet a user using this same method as above, with a simple addition:

 

myXMLStr = "<greeting><hello>" + thename.text + "</hello><age>" + userage.text +"</age></greeting>" ;

myXML_xml = new XML();

myXML_xml.parseXML(myXMLStr);

trace(myXML_xml.toString());

 

For an example of parseXML, refer to the example xml.parsexml.fla and xml.parsexml.swf .

Tips and precautions

The XML.parseXML() method is usually loaded and run automatically by the XML.onData event handler, although this can be overridden. This method is useful if you are adding information based on user input. For example, if a user enters their age into a text field, you can add this data into your structure. You should also note that this method can only be used with instances of a top level XML class node.
