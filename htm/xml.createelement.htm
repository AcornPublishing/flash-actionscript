XML.createElement

myXML.createElement("a")

 

Where myXML is an XML instance, and a is a string which represents the nodeName of the new element.

 

This method is compatible with the Flash 5 Player and above. Only nontext XML nodes are created using this method. See XML.createTextNode and XML.appendChild for related entries.

Description

The XML.createElement() method creates a new XML element, which can be added to an XML object instance using the appendChild method.

Returned values

 

 

Notes

 

myXML_xml = new XML();

myElement = myXML_xml.createElement("socks");

myXML_xml.appendChild(myElement);

trace (myXML_xml);

 

 

<socks />.

 

Creates a new element with a nodeName of socks. Socks is stored in myElement, which is then appended to the myXML_xml object instance.

 

Examples and practical uses

Using the XML.createElement() method allows you to create a new XML element. However, you must use appendChild method to add it to an XML object instance. The reference can be stored in a variable:

 

myXML_xml = new XML("<laundry><shirts>3</shirts></laundry>");

myElement = myXML_xml.createElement("socks");

myXML_xml.firstChild.appendChild(myElement);

trace(myXML_xml);

 

Which would output:

 

<laundry><shirts>3</shirts><socks /></laundry>

 

Then, if you wanted to add some text into the socks element, you would have to use the createTextNode method. You could do this in the following way:

 

myXML_xml = new XML("<laundry><shirts>3</shirts></laundry>");

myElement = myXML_xml.createElement("socks");

myXML_xml.firstChild.appendChild(myElement);

myText = myXML_xml.createTextNode("none");

myXML_xml.firstChild.firstChild.nextSibling.appendChild(myText);

trace(myXML_xml);

 

Which would output the following XML:

 

<laundry><shirts>3</shirts><socks>none</socks></laundry>

 

This concept is expanded on in the example xml.createelement.fla and xml.createelement.swf .

Tips and precautions

You should remember that in order to add a newly created element to an XML data structure, you must append it to the object using the appendChild method. The only other method used to create new elements is XML.createTextNode(), which is restricted to creating text nodes. You can also create attributes, which is done in a slightly different way. First of all, you need to create a new XML object instance. Then, you can create attributes simply by adding one manually to a specified node, as is seen in the boldfaced text.

 

myXML_xml = new XML("<skunk>");

myXML_xml.childNodes[0].attributes["spray"] = "stinky";

myText = myXML_xml.createTextNode("stripes magee");

myXML_xml.childNodes[0].appendChild(myText);

trace(myXML_xml);

 

This block of code will output the following XML data:

 

<skunk spray="stinky">stripes magee</skunk>

 

You should note that this method can only be invoked on an instance of the XML class.
