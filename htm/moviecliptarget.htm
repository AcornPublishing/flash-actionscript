MovieClip._target

myMovieClip._target

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 4 and later.

Description

The MovieClip._targetproperty returns the target path of the movie clip instance.

 

 

Code

 

 

Notes

 

 

myPath = myMovieClip._target;

 

 

Stores the path of myMovieClip in myPath.

 

myPath = myMovieClip._target();

 

 

There is no method with this name.

Examples and practical uses

The MovieClip._targetproperty returns the target path for the movie instance. This can be used in a variable:

 

myPath = myMovieClip._target;

Tips and precautions

None.
