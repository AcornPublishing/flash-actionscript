mblength

mblength(string)

 

Where string is a string.

 

Compatible with Flash 4 and later. Deprecated by String methods in Flash 5 and later.

 

The mblength() string function returns the length of a string.

Description

The mblength() function is a string function for returning the number of characters in a string.

 

 

code

 

 

Notes

 

 

 

myVar=mblength(“test”)

 

 

Returns the length of the string and stores it in myVar.

 

 

myVar=mblength()

 

 

An argument must be supplied.

 

Examples and practical uses

The mblength() function is used to determine the number of characters in a string. For example:

 

myVar = mblength(“This is a string”)

 

will return a value of 16.

Tips and precautions

The mblength() string function was included in Flash 4, but replaced by the String.length object and its methods in Flash 5. While mblength () is still supported, using the String.length methods is the preferable method based approach:

 

myVar = mblength("This is a string");

myVar2 = "This is a string".length;

 

Both myVar and myVar2 will be equal to 16.
