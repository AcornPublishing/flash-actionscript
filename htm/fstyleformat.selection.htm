FStyleFormat.selection

a.selection = b

 

Where a is the name of the custom style format, and b is the RGB value being applied to the text selection area.

 

Compatible with Flash MX. The text selection is used to represent which item in a list box or combo box is currently selected. The RGB color value is assigned to the selection property by using the 0xRRGGBB format. Related entries include FStyleFormat.selectionDisabled, FStyleFormat.selectionUnfocused, andFStyleFormat.textSelected.

Description

This property assigns a new color value for selection to a custom style format.

Shortcut

var myStyle = new FStyleFormat();

myStyle.selection = 0xFF0000;

myStyle.applyChanges();

myStyle.addListener (myListBox);

 

 

var myStyle = new FStyleFormat({selection:0xFF0000});

myStyle.applyChanges();

myStyle.addListener (myListBox);

 

 

This code applies a new RGB value for the selection property by creating a new style format. Changes are applied, and the myScrollBar listener is added.

Examples and practical uses

The selected item in a list box or combo box component is highlighted using the color value in this property.

Programming

The example fstyleformat.selection.fla and fstyleformat.selection.swf , creates a new style format called selectStyle. This style is applied to components called list1 and combo1. Color values are carefully chosen so the textSelected color contrasts with the selection color. The selectionUnfocused color is active when the list1 instance is unfocused and our text is unreadable, so this color must be changed as well.

 

selectStyle = new FStyleFormat({

   selection:0x993333, textSelected:0xCC9999,

   selectionUnfocused:0x000066});

selectStyle.applyChanges();

selectStyle.addListener(list1,combo1);

Tips and precautions

As seen in the example FLA, make sure to test your movie and determine if the text is still legible with the new color values. You may also want to make sure the colors for all the selection properties match. The selectionUnfocused color will become active if another component has focus in the movie, and the selectionDisabled color will be active if a selection is made, and then the component is disabled.
