System.capabilities.screenColor

a = System.capabilities.screenColor;

 

a will be returned as a string value.

 

This property is compatible with Flash 6 or later. This property is read only.

Description

This property is used to find out the color capabilities of the users screen. For a standard desktop, it will be "color ". For other devices, it may be "color ", "gray " or "bw ".

Examples and practical uses

This property will be returned as "color " for a standard PC or Mac, given that the lowest color resolution most graphics adapters support is 16 or 256 colors, and black and white or grayscale only is not normally an option. For other devices, you may find lower color resolutions.

 

See the example system.capabilities.fla and system.capabilities.swf .

Tips and precautions

You should note that this value refers to the platform and not the specific user. The value returned will come from a value held in the Flash player, and not a PC's system registry for example.

 

This property is geared towards versions of the Flash player on non-standard devices, where the Flash player for a particular family of devices will give a generic value for the screen used for that device type, and these will generally be mobile/pocket/wireless devices.
