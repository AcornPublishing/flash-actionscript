TextField.multiline

myTextField .multiline

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.multiline property indicates whether multiple lines can be entered into a text field.

Description

The TextField.multiline property is a Boolean used to indicate whether the text field is multiline or not. If true , the text field is multiline; if false it is a single line.

Examples and practical uses

The TextField.multiline property is a Boolean value used to indicate whether a text field is multiline (if set to true ) or a single line (if set to false ). To use the TextField.multiline property assign it a Boolean value:

 

myTextField .multiline = true;

Tips and precautions

The TextField.multiline property lets you control which text fields are single line, and which are multiline.

 

The multiline property can be set manually via the property inspector Line type drop down menu. Because there is little need to change this setting dynamically, this is not a property you will need to use often. The only practical application of it is when you create text fields dynamically (via MovieClip.createTextField ), and need to set it to a definite value. In all other cases, consider setting this property manually via the property inspector Line Type drop down;
