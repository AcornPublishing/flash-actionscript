TextField._visible

myTextField ._visible

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField._visible property determines whether the text field is visible or not.

Description

The TextField._visible property is a Boolean that indicates whether the text field instance is visible or not. If not visible, the text field is disabled and cannot be selected.

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myTextField ._visible = true;

 

The myTextField object is visible and can be clicked.

 

 

Non-visible objects cannot be clicked.

 

 

 

myTextField ._visible = "true";

 

 

Only Booleans can be used for this property.

 

 

Examples and practical uses

The TextField._visible property controls whether a text field is visible or not. A non-visible text field cannot be clicked. The TextField._visible property can be assigned as a Boolean:

 

myTextField ._visible = false;

 

See the example textfield._visible.fla and textfield._visible.swf .

Tips and precautions

This property allows you to effectively disable a text field. The text field is neither drawn, nor is it selectable. Using the visible property to make text disappear is preferable the setting the TextField._alpha property to zero. In the latter, the text field is still drawn (albeit at zero alpha), and this means that it will:

 

·       Still take up processor redraw time.

·       Still be selectable.
