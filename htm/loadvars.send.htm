LoadVars.send

instancename.send(url [,target [,method]])

 

where instancename is an instance of the LoadVars object, url is the location of the variables to be transferred, target is the frame window into which responses are to be displayed, and method is a GET or POST method

 

Compatible with Flash Player 6 and later.

 

The LoadVars.send() method sends instance variables into a specified URL

Description

The LoadVars.send() method sends variables from the instance as a string to a URL (the opposite action of the.load() method). By default, all the variables in the instance are concatenated into an application/x-www-urlform-encoded MIME format and sent using the POST method unless the GET method is specified. If LoadVars.contentType was used to specify a new MIME format prior to calling this method, that format is used instead of application/x-www-urlform-encoded.

 

If there is a response from the server where the URL is located, it is discarded unless a target frame window has been specified.

Examples and practical uses

The LoadVars.send() method is used with a URL to transfer variables from a LoadVars instance to a URL (the opposite of a LoadVars.load() method). The contents of the instance are translated into a MIME string and sent using an HTTP POST method (unless GET was specified). To use this method it has to be associated with an instance of the LoadVars object, and then can be called for that instance like this:

 

myMovieVars = new LoadVars();

myMovieVars.City = "Ottawa";

myMovieVars.Temperature = 25;

myMovieVars.Condition = "Sunny";

// send variables from myMovieVars to a file

myMovieVars.send("myData.dat");

 

This will write the variables from the myMovieVars instance into the file myData.dat. The contents of the data file will look like this (assuming standard application/x-www-urlform-encoded MIME encoding):

 

City=Ottawa&Temperature=25&Condition=Sunny

 

The contents of this file could be read by the LoadVars.load() method and stored in another LoadVars instance, like this:

 

myMovie2 = new LoadVars();

myMovie2.load("myData.dat");

 

The myMovie2 instance will now have the same three properties as myMovieVars.

 

This method is similar to the XML.send() method used by the XML object.

Tips and precautions

The send() method can only be referenced from an instance of the LoadVars object.
