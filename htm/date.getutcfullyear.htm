Date.getUTCFullYear

a = my_date.getFullYear();

 

a will be returned as an integer value corresponding to the day of the month in UTC time. This value will be a four figure integer, such as 1984.

 

This method is compatible with Flash 5 or later.

Description

This method will return a value expressed in UTC time, but is otherwise identical to the method Date.getFullYear. See the entry for that method for more details.

 

The difference between UTC time and local time is available via the Date.getTimezoneOffset method. See also the entry for Date for a fuller description of the difference between UTC and local time.
