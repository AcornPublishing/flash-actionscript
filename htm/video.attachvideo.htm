video.attachVideo

myvideo.attachVideo(a)

 

Where myvideo is an instance of the video object, a is a camera object, or video source. To drop a connection or source, pass null for the value of a.

 

Compatible with Flash 6 Player and above. This method allows you to attach a source to an instance of the video object. Related entries include Camera.get.

Description

This method can be used to specify and display a live or streamed video (such as a FLV) on the stage. To display a streamed FLV or project a live Webcam stream to the masses, you will require the Flashcom server.

 

 

Code

 

 

Notes

 

myCamera = Camera.get();
videoInstance.attachVideo(myCamera);

 

 

Attaches the camera, myCamera to the video instance aptly called videoInstance.

 

Examples and practical uses

The video source or camera capture is displayed within the bounds of the Video instance on the stage. In order to create a new instance of the Video object, you will need to open your Library, select the Options menu from the upper right corner, and then select New Video. You should then notice a new Video object in the Library. Drag this instance to the stage, and provide an instance name. When using the code we saw earlier:

 

myCamera = Camera.get();
videoInstance.attachVideo(myCamera);

 

If you have a Webcam or DV camera hooked up to your PC, then you will then be able to watch yourself in your Flash movie in the testing environment, or in a Web browser. Also, you can even watch yourself in the Flash movie uploaded to a live server. Don’t be fooled though – no one else can see you unless you get the Flashcom server, and use some ActionScript specifically for the server application.
