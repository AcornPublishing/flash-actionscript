TextField.background

myTextField .background

 

Where myTextField is a text field instance

 

Compatible with Flash MX and later .

 

The TextField.background property is used to define it a background is applied to a text field. This property has to be applied a Boolean value, either true or false . A true value indicates the text field has a background and false indicates the text field doesn't have a background. If you set the property to true , you must then define the background color or the default value of 0xFFFFFF (white) will be used. See also TextField.backgroundColor .

Description

The TextField.background property defines if a text field has a background

Examples and practical uses

The TextField.background property can be used to visually denote a text field, which the user should write something in, or read a value from; in other words, an input text field or a dynamic text field. A text field in which you want a user to write something in, like a form gather users details, should have a background (or at least a border) so the user knows where it is. A text field that is used only to display a value to the user shouldn't contain a background; this provides a visual distinction between the two uses of a text field. See the example textfield.backgroundcolor.fla and textfield.backgroundcolor.swf .

 

The following code applies settings for the user to input a value into the text field:

 

myTextField .background = true;

myTextField .background = 0xFFFFCC;

myTextField .border = true;

myTextField .type = "input";

myTextField .text = "Enter your address here…";

 

whereas the following code applies settings to a text field which will be used only to display a value, rather than receive information from the user.

 

myTextField .background = false;

myTextField .border = false;

myTextField .type = "dynamic";

Tips and precautions

The TextField.background property only defines if a background is displayed. If you set this property to true , by default, the background's color will be white (0xFFFFFF ). Use the TextField.backgroundColor property to change this setting. See the example code below:

 

myTextField .background = false;

myTextField .backgroundColor = 0xFFFFCC;
