XML.getBytesLoaded

myXML.getBytesLoaded()

 

Where myXML is an instance of the XML object.

 

This method is compatible with Flash Player 6 and above. XML.getBytesLoaded()is used toreturn the number of bytes that have been loaded. Refer to the related entry XML.getBytesTotal().

Description

The XML.getBytesLoaded() method returns an integer indicating the number of bytes that have been loaded from a streaming XML document.

Examples and practical uses

The XML.getBytesLoaded() method returns an integer showing the total number of bytes that have been loaded from a streaming XML document to that point. This is usually stored in a variable, in this case amtloaded:

 

stop();

myXML_xml = new XML();

myXML_xml.onLoad = function(success) {

   if (success) {

      trace("loaded");

   } else {

      trace("badload");

   }

};

myXML_xml.load("bigxmldoc.xml");

this.createEmptyMovieClip("holder", 1);

holder.onEnterFrame = function() {

   amtload = (myXML_xml.getBytesLoaded()/myXML_XML.getBytesTotal())*100;

   }

   if (amtload == 100) {

};
