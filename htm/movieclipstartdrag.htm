MovieClip.startDrag

MyMovieClip.startDrag ( [ lock [, left, top, right, bottom]] )

 

Where myMovieClip is a movie clip instance, lock is a Boolean indicating whether the clip is locked in the center of the mouse pointer, and left, top, right and bottom are a coordinates of a constraining rectangle for the clip.

 

Compatible with Flash 5 and later.

 

The MovieClip.startDrag()method is used to indicate whether a movie clip is draggable, and if it is, whether there are bounds on the areas that can be dragged.

Description

The MovieClip.startDrag()method allows you to designate whether a movie clip is draggable or not. Only one movie clip can be draggable at a time. If you do set a movie clip to be draggable, you can enter coordinates for a rectangle that restrain the coordinates of the drag operation.

Examples and practical uses

You can use the MovieClip.startDrag()method with the mouse methods to allow dragging. When a clip is draggable, clicking on the mouse and dragging should position it as necessary, with the mouse button release terminating the drag. This code:

 

onClipEvent(mouseDown) {

    if (this.hitTest( _root._xmouse, _root._ymouse, true )) {

 this.startDrag ( true, 100, 200, 400, 0 );

 }

}

 

… triggers on the mouseDown event then calls the MovieClip.hitTest()method to make sure the mouse is in the movie clip’s area. If it is, the MovieClip.startDrag()method is called with bounds placed on the drag area. To terminate the drag process the MovieClip.stopDrag() method can be used, again triggered by the mouseUp event:

 

onClipEvent(mouseUp) {

    if (this.hitTest( _root._xmouse, _root._ymouse, true )) {

 this.stopDrag ();

 }

}

Tips and precautions

The MovieClip.startDrag()method is useful for both locking a clip in place as well as allowing the user to reposition the clip, as needs dictate. You can issue as any MovieClip.startDrag()methods as you need. You can turn off dragging of a movie clip either by making another clip draggable or by using the MovieClip.stopDrag()method.
