FStyleFormat.scrollTrack

a.scrollTrack = b

 

Where a is the name of a custom style format, and b is a color expressed in RGB values.

 

Compatible with Flash MX. The scroll track is used in several components, including the list box and combo box components. The RGB color value is assigned to the scrollTrack property, and uses the 0xRRGGBB format. Related entries include FStyleFormat.face, and those for the FScrollBar component.

Description

This property is used to change the color of the scroll track contained within the scroll bar.

 

 

Code

 

 

Shortcut

 

 

Notes

 

var myStyle = new FStyleFormat();

myStyle.scrollTrack = 0xFF0000;

myStyle.applyChanges();

myStyle.addListener (myListbox);

 

 

var myStyle = new FStyleFormat({scrollTrack:0xFF0000});

myStyle.applyChanges();

myStyle.addListener (myListbox);

 

 

This code changes the scroll track of the scroll bar in myListbox to red.

Examples and practical uses

The scrollTrack property assigns a new RGB color value to the scroll track. It does not change the color value of the scroll bar thumb wheel (slider which moves within the scroll track).

Programming

In the example fstyleformat.scrolltrack.fla and fstyleformat.scrolltrack.swf , we change the track color of several components at once using the custom style format, scrollstyle.

 

scrollstyle = new FStyleFormat();

scrollstyle.scrollTrack = 0x99CCFF;

scrollstyle.applyChanges();

scrollstyle.addListener(pane1,combo1,list1);

 

The color value change away from the global style format is applied, and listeners are added to component instances pane1, combo1, and list1.

Tips and precautions

When you are changing style properties within your movie, you must make sure that you use the applyChanges method. Remember that the color value of the face of the scroll bar thumb wheel, which is assigned using the FStyleFormat.face property instead.
