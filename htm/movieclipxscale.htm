MovieClip._xscale

myMovieClip._xscale

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 4 and later.

 

The MovieClip._xscale property sets or retrieves the horizontal scale of a movie clip.

Description

The MovieClip._xscale property is used to determine the horizontal scale of a movie clip’s registration point, relative to the (0,0) default registration point. This lets you calculate positions of objects based on scaling effects.

Examples and practical uses

The MovieClip._xscale property is used set or retrieve the scaling of the horizontal scale (which corresponds to the percentage of the axis used by the rescaled movie clip). For example, if a clip is reduced in size to 75% of the original, the MovieClip._xscale property will allow you to calculate new offsets from the registration point for buttons and other objects in the movie. To use the MovieClip._xscale property you assign it a value, which then affects the values set by the MovieClip._x property.

 

To set the value of the MovieClip._xscale property simply assign it a value:

 

myMovieClip._xscale = 0.75;

 

To read the value, assign the property to a variable:

 

myXMouse = myMovieClip._xscale;

 

You can use the MovieClip._xscale and MovieClip._yscale properties to allow the user to scale a movie clip using some method such as a button. For example, you could have code like this that works with a “zoom up” button:

 

on(release) {

 myMovieClip._xscale *= 10;

 myMovieClip._yscale *= 10;

}

 

… which will scale the movie clip up in size by ten percent with every click of the button.

Tips and precautions

Using the MovieClip._xscale and MovieClip._yscale properties lets you adjust positions of objects using the MovieClip._x and MovieClip._x properties properly.
