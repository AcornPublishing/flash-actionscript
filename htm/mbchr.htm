mbchr

mbchr(num)

 

Where num is an ASCII value to convert to a multibyte character

 

Compatible with Flash 4 and later. Deprecated by String.fromCharCodein Flash 5 and later.

 

The mbchr() string function converts an ASCII value to a multibyte character.

Description

The mbchr() function is a string function for converting an ASCII numeric value to a multibyte character, especially useful for non-QWERTY characters (such as those with accents or non-English character sets).

Notes

 

myVar=mbchr(56)

 

 

Converts ASCII value 56 to a multibyte character, assigning it to myVar.

An argument must be supplied.

 

 

The argument must be an ASCII numeric value, not a character.

 

 

The parameter is not a valid ASCII value.

 

 

 

 

Examples and practical uses

The mbchr() function is used when you need to print a character that is not directly available on your keyboard. Particularly useful character to the designer are printed by the code below:

 

copyright = mbchr(169);

trace(copyright);

//

registered = mbchr(174);

trace(registered);

 

This will output the © and ® symbols.

Tips and precautions

The mbchr() string function was included in Flash 4, but replaced by the String.fromCharCode function (part of the String object) in Flash 5. While mbchr() is still supported, using the String object method is the preferable approach because of greater flexibility.
