TextField.maxChars

myTextField .maxChars

 

Were myTextField is a TextField instance.

 

Compatible with Flash Player later .

 

The TextField.maxChars property indicates the maximum number of characters allowed in a text field.

Description

The TextField.maxChars property sets the maximum number of characters that can be entered in a text field, including any special characters such as Enter and Tab .

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myTextField .maxChars = 10;

 

 

 

 

myStrLength = myTextField .maxChars;

 

Sets the maximum number of characters allowed in myTextField to 10.

 

Stores the maximum number of characters allowed in myTextField to a variable.

 

 

An integer must be specified.

 

 

Examples and practical uses

The TextField.maxChars property indicates the maximum length of the text in a text field, including special characters. The TextField.maxChars property can be set like this :

 

myTextField .maxChars = 20;

 

Tips and precautions

The TextField.maxChars property allows you to limit the length of a string entered into a text field. This is often necessary to prevent extraneous data, or to limit options such as state codes to two characters. You would for example, use it you expected a user input of "y " or "n ". By embedding only the outlines for "y " and "n ", and also limiting the maxChars property to 1 , you force the text field to only accept these two very specific inputs.
