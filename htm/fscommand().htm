fscommand()

fscommand ("a", "b");

 

Where a is a command which is passed to the host application or the stand-alone application. b consists of parameters made of a string or value, which is passed to the Flash player or plug-in.

 

Compatible with Flash 3 or later. fscommand does not work in several situationsnoted below in the Tips and precautions section. It will work with programs that are compatible with Active X controls. The commands passed back to the Flash player can only be predefined parameters including quit, fullscreen, allowscale, showmenu, exec, and trapallkeys. Related actions include getURL function.

Description

This function is used to communicate with external applications by sending commands to the other application, or send commands back to the Flash player.

on (release) {

   fscommand ("panIt");

}

 

if (command == "panIt"){

fscommand01_test.Pan (25,25,1);

}

 

This fscommand will pan a zoomed in movie by 25% (horizontally and vertically) each time the button is clicked. The value of 1 measure pan in percentages, and a value of 0 measures pan in pixels.

 

Examples and practical uses

fscommand is typically used to send commands to JavaScript, VBScript or the web browser, interact with Lingo in Macromedia Director, or send the built in commands back to the Flash player. You can use fscommand to manipulate the size of the standalone player window, and control the scaling of your Flash movie in either the Flash player or the Flash plug-in.

Tips and precautions

Most importantly,fscommand does not work with Internet Explorer for the Macintosh, or Netscape 6 (and greater). You must make sure that swLiveConnect is set to true in the HTML page containing your SWF file, or fscommand will not work with other versions of Netscape. When publishing your Flash file, select Flash with FSCommand from the Template pull-down under the HTML tab. This will modify your published HTML file to include the necessary attributes for fscommand.

 

When you are working with Flash documents intended for a web browser, you will have to further edit the code within your HTML page. fscommand works by calling functions using JavaScript and VBScript and is largely similar to the way getURL works with JavaScript. Sometimes it is much easier, and covers a wider user base, to use the getURL action as opposed to fscommand. However, you will avoid a browser click sound and possible error messages by using fscommandinstead ofgetURL.
