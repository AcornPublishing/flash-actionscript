FStyleFormat.check

a.check = b;

 

Where a is the name of your style format, and b is a new RGB value being assigned to the check graphic.

 

Compatible with Flash MX. This will change the color value of the check used in check box component, and any custom and/or third party components using this particular property. The RGB color value is assigned to the property using the 0xRRGGBB format. Related entries include those for the FCheckBox component.

Description

This property sets the check mark of a component instance to the specified RGB value.

This code applies a new color value to the check in the check box component.

 

Examples and practical uses

This property is used to change the color of the check mark in an active check box component instance.

Programming

In the example fstyleformat.check.fla and fstyleformat.check.swf , the check mark color value is changed by creating a custom style format, checkStyle, and setting the check color value to blue. The changes from the global style format are applied. A listener is added to five check boxes (c1-c5 ), which then have light blue check marks when selected.

 

checkStyle = new FStyleFormat();

checkStyle.check = 0x99CCFF;

checkStyle.applyChanges();

checkStyle.addListener(c1,c2,c3,c4,c5);

 

This example is expanded upon in fstyleformat.check.fla by removing all listeners from an array of component instances, and then applies a custom global style format when a button is pressed.

Tips and precautions

The color format must be written in the 0xRRGGBB format, and the color value should contrast from the background color of the check box component. You can change the background color of a check box usingFStyleFormat.background.
