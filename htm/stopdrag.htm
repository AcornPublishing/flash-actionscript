stopDrag

stopDrag();

 

Where a movie clip that is being dragged, will cease following the mouse.

 

Compatible with Flash 3 and above. There are no known issues with any version of Flash that supports this action.

 

The stopDrag function is used to stop a movie clip from being dragged which started being dragged using the startDrag function. This function is depreciated in Flash MX in preference of MovieClip.stopDrag. See also MovieClip.stopDrag.

Description

This function will cease the movie clip that is currently being dragged around from being dragged around.

When the mouse button is release, the movie clip will stop being dragged around.

 

Examples and practical uses

See the MovieCip.stopDrag function for examples as this function is depreciated.

Tips and precautions

Timeline independent

As both startDrag and MovieClip.startDrag will replace the last movie clip being dragged with the movie clip that last executed the method, the stopDrag command will stop any movie clip from being dragged, regardless of the issuing timeline (on the startDrag and stopDrag parts).
