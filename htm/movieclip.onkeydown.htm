MovieClip.onKeyDown

myMovieClip.onKeyDown

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip.onKeyDownevent handler is invoked when a key is pressed and the movie has input focus.

 

This event is the same as the corresponding Button event Button.onKeyDown. See the entry for Button.onKeyDown.
