video.clear

myvideo.clear();

 

Where myvideo is an instance of the Video object.

 

Compatible with Flash 6 Player and above. This method allows you to clear the image in the designated video instance, in this case, myvideo. Refer to the related entry on video.attachVideo.

Description

This method is used to clear the current image which is in the video object instance.

 

 

Code

 

 

Notes

 

function clik(){

   myVideo.clear();

   _root.stop();

}

 

 

Clears the video object instance.

Examples and practical uses

This method is useful if you are dealing with a live video broadcast, and your current connection breaks. Instead of having to hide the Video object instance, you can use this method to help display a custom message. This method clears the current image in the Video object.

 

Let’s take a look at our earlier example:

 

function clik(){

   myVideo.clear();

   _root.stop();

}

 

We have embedded a FLV video in our movie, and dragged it onto the stage. The video is looping via a gotoAndPlay action placed on the final frame of the frame sequence. The video instance has been given an instance name myVideo. A component with a click handler of clik is also on the stage, and on the main timeline, we have the above code listing. This will clear the video image, and stop the movie. If the stop() action is not included, the video will clear the single frame when the button is pressed, and then continue. Refer to the example video.clear.fla and video.clear.swf .
