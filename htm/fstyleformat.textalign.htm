FStyleFormat.textAlign

a.textAlign = "b";

 

Where a is the name of your style format, and b is a text string of left, center, or right.

 

Compatible with Flash MX. The value assigned to the global style format for this property is left. Related entries include those on FStyleFormat.textIndent, FStyleFormat.textLeftMargin, andFStyleFormat.textRightMargin.

Description

This property justifies the text in a component by specifying center, left or right alignment.

myStyle = new FStyleFormat();

myStyle.textAlign = "center";

myStyle.applyChanges();

myStyle.addListener (myComponent);

 

 

myStyle = new FStyleFormat

({textAlign:"center"});

myStyle.applyChanges();

myStyle.addListener(myComponent);

 

 

This code centers the text in myComponent, which is listening to the myStyle custom style format.

 

 

myStyle = new FStyleFormat();

myStyle.textAlign = center;

myStyle.applyChanges();

myStyle.addListener (myComponent);

 

 

myStyle = new FStyleFormat

({textAlign:center});

myStyle.applyChanges();

myStyle.addListener(myComponent);

 

 

This property value is expressed as a string, so you must have double quotations around right,center, or left alignment.

 

Examples and practical uses

You can use this property as to align any text within Flash UI components, including check box and radio button component instances.

Programming

In the example fstyleformat.textalign.fla and fstyleformat.textalign.swf , text alignment is changed in applicable components to illustrate how each one appears after the property has changed. This example illustrates how this property is used, and how each alignment affects the Flash UI components. For right alignment, we create a custom style format called alignRStyle and apply changes to the style format. Then listeners to the custom format are applied to the button3, combo3, list3,check3, and radio3 instances.

 

alignRStyle = new FStyleFormat({textAlign:"right"});

alignRStyle.applyChanges();

alignRStyle.addListener(button3,combo3,list3,check3,radio3);

Tips and precautions

This property will change text alignment, but you are not able to indent amounts in pixels. In order to do this, use the FStyleFormat.textIndent, FStyleFormat.textRightMargin, or FStyleFormat.textLeftMargin.
