Math.abs

a = Math.abs(b);

 

Where a is the variable that you want to equate to the absolute value of b (with absolute value being the number b without the positive or negative sign).

 

Description

This method will return any number passed to it, without the positive or negative sign. This is the absolute value of the number. In maths, this is written as:

 

a = | b |

 

 

Code

 

 

Notes

 

a = Math.abs(5);

b = Math.abs(-5);

 

c = -9;

d = Math.abs(c);

 

e = "-15";

f = Math.abs(e);

 

 

In the first line, a will be set to 5.

 

In the second line, b will be set to 5, because the negative sign was stripped off.

 

In the third example, c is being set to the number -9, and therefore d will be set to 9.

 

In the final example, e is being set to the string -15. Flash will convert this from a string to a number before it then applies the Math.abs function, and returns 15.

 

 

c = Math.abs ("text");

 

If anything other than a number is passed into the function, then Flash will return NaN; which stands for "Not a Number".

 

Examples and practical uses

See the example math.abs.fla and math.abs.swf , which uses the Math.abs method to create a reflecting motion using the mouse. It illustrates that the main use in motion graphics is to cause something to change direction at a particular axis. By default, items will change direction at 0, because that’s when Math.abs drops the sign. If we wanted Math.abs to start flipping the sign at a different number, then we must perform some modifications:

 

If we want 30 to be the point at which the direction is flipped, then we must do:

 

a = Math.abs(b - 30) + 30;

 

This means that a will never drop below 30. Once b reaches 29, a will be set to 31.

 

Also take a look at the example math.absplot.fla and math.absplot.swf to see the Math.abs function plotted on an x, y graph.

Tips and precautions

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
