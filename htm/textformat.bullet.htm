TextFormat.bullet

myTextFormat.bullet

 

where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.bullet property indicates whether the text is part of a bulleted list.

Description

The TextFormat.bullet property is a Boolean used to indicate the text is bulleted. If true , the text is bulleted; if false text is not bulleted. The default is null which means no bullets. If bulleted, the text is indented to the right and a bullet appears to the left of the text.

 

Assuming we had a text field called my_txt as shown:

 

 

The following lines would result in a change as shown:

 

myTextFormat = new TextFormat();

myTextFormat.bullet = true;

my_txt.setTextFormat(myTextFormat);

Additional Explanation

 

 

Notes

 

myTextFormat.bullet = true;

 

Text is bulleted.

 

A null value means the value of the text field is unchanged.

 

 

Examples and practical uses

The TextFormat.bullet property Boolean indicates whether bulleting is applied or not. Only Boolean values are allowed:

 

myTextFormat.bullet = true;

 

See the example textformat.bullet.fla and textformat.bullet.swf .

Tips and precautions

This option doesn't not appear on the property inspector, and can only be applied through ActionScript.
