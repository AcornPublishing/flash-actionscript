Math.LOG2E

a = Math.LOG2E;

 

Where a is the variable that you want to equate to the base-2 logarithm of the constant e, or Math.E. This is a constant.

 

This method is compatible with Flash 5 or later.

Description

This constant is approximately equal to 1.442695040888963387. A base-2 logarithm is taken as "2 to the power of". We can use this constant to convert a natural logarithm into a base 2 logarithm.

 

 

Code

 

 

Notes

 

a = Math.LOG2E;

 

b = Math.log(64) * Math.LOG2E;

 

 

 

In the first line, a will be set to approximately 1.442695.

 

In the second line, b will be set to 6, because 2 to the power of 6 is 64. We’re using the natural logarithm Math.log, and converting it into a base-2 logarithm using the Math.LOG2E constant.

None

Tips and precautions

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
