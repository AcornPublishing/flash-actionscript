Function.call

a.call(b,[c,d,n])

 

Where a is the name of the function, b is an object specifying a value of this, and c to n are optional parameters.

 

Compatible with Flash MX. This method is similar to the call action in earlier versions of Flash. Functions are called and executed when using this method. Also refer to entries on Function.applyandFunction.

Description

This method is used to invoke a function being represented by a Function object.

Examples and practical uses

Function.call is typically used when you need to call a dynamic function on the fly.

Programming

In the example function.call.fla and function.call.swf , a dynamic function is executed. This is based on what a user selects in a dropdown list. In the movie, a comboBox component has three different function names as options. An input text field is on the Stage where a user can enter a value from 0-100. The user can select functions from either instance and submit changes to image.

 

At this time, the callFunction function is executed and determines the value of the selected item in the comboBox. This function is then executed, and passes the value of a as a parameter. This procedure is seen in the following ActionScript:

 

function setAlpha(a) {

   if (a<0) {

      a = 0;

   } else if (a>100) {

      a = 100;

   }

   _root.value.text = a;

   _root.circle._alpha = a;

}

function callFunction() {

   var a = Number(_root.value.text);

   var thisFunct = function_to_call.getSelectedItem().data;

   thisFunct.call(null, a);
