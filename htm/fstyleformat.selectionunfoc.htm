FStyleFormat.selectionUnfocused

a.selectionUnfocused = b;

 

Where a is the name of your style custom format, and b is a RGB color value being assigned to the unfocused selection area.

 

Compatible with Flash MX. This property is used to set the color of the selection area when a component has a selection made, but focus is on another component. The RGB color value is written using the 0xRRGGBB format. Related entries include FStyleFormat.selection.

Description

This property assigns a new color value to an unfocused text selection.

This assigns a light blue color to a selecton area when the component on the Stage is unfocused.

 

Examples and practical uses

This property is used to customize the color of the unfocused selection area on list box components, or any others supporting this particular region.

Programming

The example fstyleformat.selectionunfocused.fla and fstyleformat.selectionunfocused.swf on the

CD, includes an example of the default and custom settings for this property. This file illustrates how the selection color changes when you change focus from one selected list box instance to another. When focus is removed from the list boxes assigned to a custom style format, the selection area turns light blue. As you can see in this FLA, the color value change from focused to unfocused is very minor in the global style format.

Tips and precautions

The color value you give to this property should be selected based on your requirements. If you intend for the focus changes to be very pronounced, you may want to choose highly contrasting values for the FStyleFormat.selection, and FStyleFormat.selectionUnfocused properties. If you do not need or want to draw attention to this change, you will probably want to choose very similar colors, as is the case in the global style format.
