TextFormat.underline

myTextFormat.underline

 

Where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.underline property indicates whether the text is underlined.

Description

The TextFormat.underline property is a Boolean used to indicate the text is underlined. If true , the text is underlined, if false text is not underlined. The default is null .

 

 

Code

 

 

Additional Explanation

 

 

myTextFormat.underline = true;

 

 

Text is shown underlined

Examples and practical uses

The TextFormat.underline property Boolean indicates whether an underline is applied to text or not. Only Boolean values are allowed:

 

myTextFormat.underline = true;

 

See the example textformat.underline.fla and textformat.underline.swf .

Tips and precautions

If TextFormat.underline property is set to null, no underlining is performed.

 

If you wish to remove any or all TextFormat properties such as underline, bold, italics, etc, you have define or re-define the values as false . This will successfully remove them from the TextFormat object.
