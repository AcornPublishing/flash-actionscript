System.capabilities.pixelAspectRatio

a = System.capabilities.pixelAspectRatio;

 

a will be returned as an integer value.

 

This property is compatible with Flash 6 or later. This property is read only.

Description

This property is used to find out the aspect ration of the screen pixels. Pixels on a standard PC are square, giving an aspect ratio of 1.

Examples and practical uses

This property is only useful for non-standard systems, such as PDAs and pocket/mobile devices. Such devices will typically support a reduced set of the normal flash player's features, and may include custom or proprietary capabilities.

 

See the example system.capabilities.fla and system.capabilities.swf .

Tips and precautions

You should note that this value refers to the platform and not the specific user. For example, a PC will always have a Flash player that assumes a pixel aspect of 1, even if that machine has the latest TFT/plasma monitor with a different aspect ratio. The property refers to the Flash player, and certainly will not look through the operating system registry to derive detailed hardware specifications.

 

This property is geared towards versions of the Flash player on non-standard devices, where the Flash player for a particular family of devices will give a generic value for the screen used for that device type, and these will generally be mobile/pocket/wireless devices.
