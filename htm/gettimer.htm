getTimer

getTimer()

 

Compatible with Flash 4 and greater. This function returns the elapsed time that your movie has been running in milliseconds. Related entries include those forthegetSeconds and getTime methods of the Date object.

Description

This functionwill return, in milliseconds, the elapsed time since your Flash movie started playing.

Examples and practical uses

This function can be useful when you want to show the elapsed time of a presentation, or video that is playing in your movie. This function is commonly used for games where an elapsed time measurement is required to time user input.

Programming

In the sample gettimer.fla and gettimer.swf , there is a dynamic text field on the stage called elapsedTime. This text field will be used to display the current duration of our movie. setInterval is used to display what the timer returns every 500 milliseconds.

 

setInterval(function () {

   _root.elapsedTime.text = getTimer();

}, 500);

 

There will be a slight variation in numbers (i.e. there will not always be a count of 500 between each return). This difference is essentially the time it takes the processor to count the numbers displayed, and this lag will ever so slightly be evident in the milliseconds (as text) returned to the Stage.

Tips and precautions

This function could be useful for expressing the duration of movies. getTimer operates differently on Macintosh systems, where the function rounds to the nearest 1000. This is due to the differences between the two systems in measuring ticks. Therefore, getTimer could be unreliable as a timing source since there are discrepancies between the two operating systems. Either use detection scripts and modify your movie accordingly, or use the Date.getTimemethod.
