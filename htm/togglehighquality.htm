toggleHighQuality

toggleHighQuality()

 

Compatible with Flash Player 2 and later.

 

The toggleHighQuality property turns anti-aliasing on or off. See also MovieClip._quality.

Description

The toggleHighQuality property toggles anti-aliasing on or off to change the quality of a movie. When toggled on, anti-aliasing is applied and edges of objects are smoother. When off, no anti-aliasing is applied. There are a number of other issues that can affect the quality, and these include:

·         Whether the graphics being anti aliased are pixel snapped or not. Pixel snapped graphics are not aliased.

·         The resolution of the users screen. Higher resolutions are less susceptible to "jaggies", and the need for aliasing is reduced.

 

A fuller description of aliasing and its effects is included in the entries Button._highQuality, Button._quality and the corresponding entries for the movie clip and text field objects.

Examples and practical uses

The toggleHighQuality setting is used to toggle anti-aliasing on or off inside a clip. There is no value associated with the setting: it simply toggles the existing value. The value of toggleHighQuality can be changed (toggled to the other state) like this:

 

toggleHighQuality();

 

This setting can be tied to an event, such as a button release (which is the normal way to use toggleHighQuality):

 

on(release){

   toggleHighQuality();

}

 

The toggleHighQuality setting is excellent for speeding up your code on slower machines.

Tips and precautions

Anti-aliasing allows edges of objects to be smoothed, resulting in a higher quality image. However, anti-aliasing imposes an overhead on processing, so slower machines may experience movie playback problems with anti-aliasing turned on. The toggleHighQuality setting simply toggles anti-aliasing on or off with every call.
