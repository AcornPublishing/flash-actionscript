TextField.maxhscroll

myTextField .maxhscroll

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.maxhscroll property sets the maximum value of the TextField.hscroll property.

Description

The TextField.maxhscroll property is used to set the maximum TextField.hscroll value .

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myTextField .maxhscroll = 50;

 

 

 

Sets the maximum horizontal scroll position of myTextField to 50 characters

 

 

Sets the maximum value that can be used by TextField.hscroll

Examples and practical uses

The TextField.maxhscroll property is used to set the maximum horizontal scrolling position, used by the TextField.hscroll property . To use the TextField.maxhscroll property, assign it a value:

 

myTextField .maxhscroll = 50;

Tips and precautions

The TextField.maxhscroll property is handy in setting the amount of horizontal scrolling allowed in a text field. It can use used with the TextField.maxscroll property to constrain scrolling in both dimensions.

 

This property is associated with the creation of scrollbars. Because components provide this feature directly, the need to use hscroll is somewhat reduced.
