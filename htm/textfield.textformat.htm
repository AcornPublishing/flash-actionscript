TextField.getNewTextFormat

myTextField .getNewTextFormat()

 

Where myTextField is a TextField instance

 

Compatible with Flash Player 6 and later .

 

The TextField.getNewTextFormat() method returns a TextFormat object containing a copy of a text field's text format. A TextFormat object defines how the text will be rendered, such as its style (bold, normal, italics, etc), color, and so on.

Description

The TextField.getNewTextFormat() method returns a copy of a text field instance's text format. The return is as a TextFormat object. This text format is used for all new text inserted into the field.

Examples and practical uses

The TextField.getNewTextFormat() method reads the text field's text format and returns a copy of the format, using the TextFormat object. Any new text entered by the user or by a TextField method has this format applied automatically. If there are no properties to be returned in the TextFormat object, the return is null .

 

The return from the TextField.getNewTextFormat() method array can be stored in a variable:

 

myTextFormat = TextField.getNewTextFormat();

Tips and precautions

There is usually little reason to want to know a text fields current text format because you usually know it already! It will be either:

 

·       The property inspector settings you applied during authoring.

·       The last textFormat settings you applied during run time.

·       The default formatting for a new text field when you create them dynamically via MovieClip.createTextField .
