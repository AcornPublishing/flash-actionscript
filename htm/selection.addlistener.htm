Selection.addListener

Selection.addListener (listener_name)

 

Where listener_name is an object with an onSetFocus method.

 

This method is compatible with Flash 6 and later.

 

The Selection.addListener() method registers an object to receive keyboard focus changes.

Description

The Selection.addListener() method registers the object to allow it to receive keyboard focus change notifications. Whenever a focus change is detected, all registered objects have their onSetFocus() method invoked.

 

 

Code

 

 

Additional explanation

 

Notes

 

Selection.addListener (myObject);

 

Registers myObject so the onSetFocus() method is invoked on focus.

 

 

If myObject is already registered nothing happens.

 

Examples and practical uses

Selection.addListener() is used to register an object to detect changes to the focus. The object’s onSetFocus() method must be defined and is invoked when the object receives focus. After the methods have been defined, the object is registered so that the onSetFocus() method is invoked properly:

 

Selection.addListener (myObject);

Tips and precautions

If the listener is already registered, then the Selection.addListener() method has no effect. Although there are no limits to the number of objects that can respond after being registered using Selection.addListener(), registering too many objects can cause slow response from Flash.
