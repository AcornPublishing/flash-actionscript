System.security.allowDomain

System.security.allowDomain(domain1, domain2,... domainN);

 

Where domain1 to domainN are domain names that you to want to permit to share assets with the current SWF.

 

This method is compatible with Flash 6 or later.

Description

This method allows you to share assets between SWFs even if the SWFs originated from different domains.

Examples and practical uses

The allowDomain would allow you for example to:

 

·         Maintain a client site that is on the client domain, www.clientDomain.com, by loading assets from www.myDesignHouse.com, where you regularly update the content via a service contract.

·         Where you have a number of different domains (such as www.myPersonalDomain.com and www.myCommercialDomain.com ), you can use load levels, load targets, and other flash loading schemes as if all your assets were loading from the same domain.

Tips and precautions

None.
