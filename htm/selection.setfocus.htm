Selection.setFocus

Selection.setFocus("variable")

 

Where variable is a string indicating the path to the variable associated with the field.

 

This method is compatible with Flash 6 and later.

 

The Selection.setFocus() method applies focus to a field.

Description

The Selection.setFocus() method is used to apply focus to an editable text field specified as a parameter. The parameter is a string indicating the path to the variable (relative and absolute paths, as well as dot and slash notation are allowed). When a text field or button instance path is specified with the Selection.setFocus() method, that button or text field has focus. If a parameter of null is provided, current focus is removed. If the focus is set correctly using the Selection.setFocus() method, a true Boolean is returned; if the focus fails, a false value is returned.

Examples and practical uses

The Selection.setFocus() method is used to apply focus to a text field or button. For example, the command:

 

Selection.setFocus("_root.myName");

 

… will apply focus to the text field or button, which is referenced by the variable myName in the _root window. If a null is provided as a parameter, current focus is removed.

 

You can see a simple use of the Selection.setFocus() method in the code below. This is used on a form that has four text fields. When the Tab key is pressed, the Selection.getFocus() method is used to check which of the four text fields has focus, and then uses Selection.setFocus() to move to the next text field. If the last of the four text fields has focus, focus is changed to the first text field, so the tab can be used to cycle through the four fields:

 

on (Key.isDown(9) {

 if ( Selection.getFocus() == “_level0.textfield1”) {

 Selection.setFocus(“_level0.textfield2”);

 } else if Selection.getFocus() == “_level0.textfield2”) {

 Selection.setFocus(“_level0.textfield3”);

 } else if Selection.getFocus() == “_level0.textfield3”) {

 Selection.setFocus(“_level0.textfield4”);

 } else if Selection.getFocus() == “_level0.textfield4”) {

 Selection.setFocus(“_level0.textfield1”);

 }

}

 

Note that when comparing the result of the Selection.getFocus() method we include the _level0 string, as that is stored as part of the focus path. See the Selection.getFocus() method entry for more information.

Tips and precautions

The Selection.setFocus() method is used to move focus specifically to a text field or button. A sneaky way to use the Selection.setFocus() method arises when you have a movie playing that has a text field in it. That text field will have input focus only if the movie itself has focus. To force the movie to have focus, add a button to the start of the movie that requires a user click to force focus on the movie (such as a Start button). Once the movie has focus, you can force input focus on a text field with the Selection.setFocus() method.
