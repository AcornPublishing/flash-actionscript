MovieClip._ymouse

myMovieClip._ymouse

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 5 and later.

 

The MovieClip._ymouse property retrieves the vertical coordinate of the mouse. This property is exactly the same as MovieClip._xmouse except that it affects the vertical instead of horizontal value. See the entry for MovieClip._xmouse.
