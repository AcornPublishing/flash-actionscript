Object.valueOf

myObject.valueOf()

 

Where myObject is a Object instance.

 

This method is compatible with Flash 5 and later.

 

The Object.valueOf() method returns the value of the object.

Description

The Object.valueOf() method returns the primitive value of the object. If the object does not have a value, the object itself is returned.

Notes

 

myType = myObject.valueOf();

 

 

Returns the value type of myObject, storing it in myType.

 

Examples and practical uses

The use of this method is self-explanatory. A value type is returned in response to code of the format illustrated in the Description section above.
