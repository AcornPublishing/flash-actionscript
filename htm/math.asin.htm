Math.asin

a = Math.asin(b);

 

Where a is the variable that you want to equate to the arc sine of b.

 

This method is compatible with Flash 5 or later.

Description

This method will return the arc sine value of the number passed to it. The number passed in must be from -1.0 to 1.0. Generally, Math.asin returns an angle that you already have the sine value of, and the angle is returned in radians.

Notes

 

a = Math.asin(0);

 

b = Math.asin(-1);

 

c = Math.asin(1);

 

 

 

In the first line, a will be set to 0, because the arc sine of 0, is 0.

 

In the second line, b will be set to -1.5707963267949, which is negative one half of the mathematical constant, PI.

 

In the third example, c will be set to 1.5707963267949, which is one half of the mathematical constant, PI.

 

 

 

d = Math.asin(2);

 

e = Math.asin ("text");

 

In the first example, d will be set to NaN, because 2 is not within the range of -1.0 to 1.0.

 

If anything other than a number is passed into the function, then Flash will return NaN; which stands for "Not a Number".

 

Examples and practical uses

Math.asin is generally used to find an angle from an already-calculated sine value. To see Math.asin plotted on a graph, see the example math.asinplot.fla and math.asinplot.swf .

 

The angle returned in this function will be in radians, rather than degrees. To convert radians to degrees it is necessary to multiply the radian value by the number 57.2957795130823. Or, simply and more accurately: 180 / Math.PI

 

In the following example:

 

a = Math.asin(0.8);

b = a * (180 / Math.PI);

 

a will be an angle in radians, b will be the equivalent angle in degrees. The angle in this case is about 0.9273 radians, or 53.13 degrees.

Tips and precautions

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
