Number.valueOf

Number.valueOf()

 

Where Number is the Number class.

 

This property is compatible with Flash 5 and later.

 

The Number.valueOf() method returns the value type of the Number object.

Description

The Number.valueOf() method returns the primitive value type of the number object.

 

 

Code

 

 

Notes

 

 

myType = myNumber.valueOf();

 

 

Returns the value type of myNumber, storing it in myType.

 

 

Examples and practical uses

This method allows you to strictly enforce data types by converting unexpected results. See the Tips and precautions section for more details.

Tips and precautions

The Number.valueOf() method can be used to determine values and data types, but the returned values cannot always be useful. Having said that, there are times when the Number.valueOf() method can be a great aid to debugging. This is because Flash is a loosely-type language (meaning the data types are not strictly enforced) and you may be working with a string instead of a number. For example, if you prompt the user for a numeric value in a form, this code:

 

myNum = 5;

myNum = myNum+myText.text;

 

… may give you results that do not immediately make sense. This is because Flash assumes all input fields are strings (unless you force them to be numbers only). If the user typed “10” into the input field, the value of myNum will be “510” instead of “15”. You can prove the use of a string for the input field using:

 

trace(typeof(myText.text));

 

Using the Number.valueOf() method in your code does allow you to more strictly enforce data types by converting unexpected results (such as strings to numbers).
