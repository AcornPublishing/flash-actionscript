Math.atan

a = Math.atan(b);

 

Where a is the variable that you want to equate to the arc tangent of b.

 

This method is compatible with Flash 5 or later.

Description

This method will return the arc tangent value of the number passed to it. Generally, Math.atan returns an angle that you already have the tangent value of.

 

 

Code

 

 

Notes

 

a = Math.atan(0);

 

b = Math.atan(-1);

 

c = Math.atan(1);

 

d = Math.atan(-Infinity);

 

e = Math.atan(Infinity);

 

 

 

In the first line, a will be set to 0, because the arc tangent of 0, is 0.

 

In the second line, b will be set to

-0.785398163397448.

 

In the third example, c will be set to 0.785398163397448.

 

In the fourth example, d will be set to -1.5707963267949, which is negative one half of the mathematical constant, PI.

 

In the fifth example, e will be set to 1.5707963267949, which is one half of the mathematical constant, PI.

 

 

f = Math.atan ("text");

 

If anything other than a number is passed into the function, then Flash will return NaN; which stands for "Not a Number".

 

Examples and practical uses

Math.atan is generally used to find an angle from an already-calculated tangent value. The angle returned will be in radians, rather than degrees. To convert radians to degrees it is necessary to multiply the radian value by the number 57.2957795130823. Or, simply and more accurately: 180 / Math.PI

 

In the following example:

 

a = Math.atan(0.8);

b = a * (180 / Math.PI);

 

a will be an angle in Radians, b will be the equivalent angle in degrees. The angle in this case is about 0.67474 radians, or 38.66 degrees.

 

To see Math.atan plotted on a graph, see the example math.atanplot.fla and math.atanplot.swf . The arc tangent of a number is a good way to determine the angle between two points. However, the procedure is somewhat convoluted, as we must first compute the ratio of two sides of the triangle (y distance over x distance). For that reason, it’s best to use the Math.atan2 method to compute the angle between two points. See Math.atan2 for more details.

 

The angle returned by Math.atan in the range of –PI / 2 to PI / 2, or in degrees -180 to 180.

Tips and precautions

In Flash 4, the methods of the Math object are emulated, and may therefore not always be as accurate as in Flash 5 and above.
