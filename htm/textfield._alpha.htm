TextField._alpha

myTextField ._alpha

 

where myTextField is an instance of a text field instanc.

 

Compatible with Flash Player 6 and later .

 

The TextField._alpha property represents the alpha value (transparency) of a text field. Note: this property will only work if you use embedded fonts with the text field, see TextField.embedFonts .

Description

The TextField._alpha property represents the alpha transparency of a text field. Legal values are any integer ranging from 0 (totally transparent) to 100 (fully opaque).

Examples and practical uses

TextField._alpha controls the setting and retrieving of the value of the alpha channel, also known as the transparency, of a text field. An alpha value can range from 0 (the field is totally transparent) to 100 (the field is totally opaque). Objects that have a transparency of 0 (totally transparent) are still active and are useful for defining hidden fields areas. The value of transparency can be set inside code:

 

myTextField ._alpha = 100;

 

You can use the alpha value to visually denote text field focus; a selected (in focus) text field will appear at 100% alpha, and an unselected (out of focus) text field will appear at 60% alpha if you use something based on the following script;

 

my_txt.onSetFocus = function() {

   my_txt._alpha = 100;

};

my_txt.onKillFocus = function() {

   my_txt._alpha = 60;

};

my_txt._alpha = 60;

 

This code sets the initial alpha value of a text field my_txt to 60% (last line) and sets alpha to 100 and 60% after that based on the occurrence of the onSetFocus (text field has just become selected) and onKillFocus (text field has just become unselected). The concept is developed further in the example textfield._alpha.fla and textfield._alpha.swf , where the oldfocus argument of the textField.onSetFocus event is used to create a more general version of the above example.

Tips and precautions

Note that a totally transparent field is still selectable even though you cannot see it. To make a text field invisible and unselectable, you should use TextField._visible . You can also use the TextField.selectable property to control if a text field can be selected.

 

You must embed font outlines for the _alpha property to take effect. See the Tips and precautions section for TextField .
