TextFormat.blockIndent

myTextFormat.blockIndent

 

Where myTextFormat is a TextFormat instance

 

Compatible with Flash Player 6 and later .

 

The TextFormat.blockIndent property sets or retrieves the block indentation of text in a field.

Description

The TextFormat.blockIndent property is used to set or retrieve the block indentation characteristics of all the text in a field. The value is specified as a number of points. If set to null , there is no block indentation.

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myTextFormat.blockIndent = 10;

 

 

Indents all text 10 points

 

A null value means no indentation. Block indentation applies to all line in the field.

 

 

 

myTextFormat.blockIndent = "10";

 

 

The indentation must be specified as a number

 

 

 

Examples and practical uses

The TextFormat.blockIndent property is used set or retrieve the amount of indentation to apply to all the text in a field. The value is set as a number of points. For example, to set the value of the TextFormat.blockIndent property to have a five-point indentation for all lines in the text field, assign it a value:

 

myTextFormat.blockIndent = 5;

 

To read the value, assign the property to a variable:

 

myblock = myTextFormat.blockIndent;

 

See the example textformat.blockindent.fla and textformat.blockindent.swf .
