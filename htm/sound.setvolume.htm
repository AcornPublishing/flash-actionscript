Sound.setVolume

mySound.setVolume(volume);

 

Where the volume level of mySound will be set to the value of volume.

 

The object is compatible with Flash 5 and above. There are no known issues with any version of Flash that supports this object.

 

The setVolume method is used to modify the volume of the sound. The method will accept an integer between 0 and 100, where 0 is silent and 100 is full volume. The integer you pass actually represents a percentage of the total volume. See also Sound.getVolume and Sound.setTransform.

Description

This method defines the loudness of a Sound object, using an integer representing a percentage of the full volume.

 

 

Code

 

 

Notes


mySound.setVolume(50);

 

Setting the volume to 50% of full Volume.

 


on(release) {

 mySound.setVolume(mySound.getVolume()+10);

}

 

This is an example of controlling the sound through the onRelease event of a button. When the button is released, after being pressed, the sound will increase by 10%.

 

 

mySound.setVolume(“100”);

 

This code will fail.

 

Only an integer will be accepted by setVolume.

 

Examples and practical uses

There are two ways to specify the sound volume: absolute or relative. The following code will absolutely define the sound volume:

 

mySound.setVolume(50);

 

While the following code will specify the volume using a relative method:

 

mySound.setVolume(mySound.getVolume()-20);

 

When using the relative method to define a sound, you can use an if statement to evaluate the current sound volume and be sure that you can alter the volume. For example the sound level could already be 0, so the code above would have no effect. See the example soundsetvolume.fla / soundsetvolume.swf for an example of using relative volume.

Tips and precautions

Using the setVolume on a sound will only change the sound volume, relative to the full volume that the sound will play at. The volume parameter is actually representing a percentage, not an absolute sound level. For example, you might have two sounds, sound1 and sound2. sound1 is 50dB and sound2 is 120dB. You execute a setVolume method on sound2 setting the volume to 50%. The sound2 object will still be louder than sound1 however, because 50% of 120dB is 60dB, which is higher than sound1 ’s initial loudness.
