TextField.replaceSel

myTextField .replaceSel(text)

 

Where myTextField is a TextField instance and text is the text to insert.

 

Compatible with Flash Player 6 and later .

 

The TextField.replaceSel() method is supposed to replace the current selection with new text, but is bugged, so you are unlikely to use it.

Description

The TextField.replaceSel() method is designed used to replace a text selection with another text string. It is bugged however, so you are unlikely to use it. The example files in the Examples and practical uses section shows the error and a workaround.

Examples and practical uses

The TextField.replaceSel() method is supposed to remove the current text selection and replace it with the new text provided as an argument. It actually adds the argument to the front of the selection, and is therefore bugged.

 

myTextField .replaceSel("Quit");

 

 

Adds "Quit " to the front of whatever was previously selected in myTextField .

 

The example textfield.replacesel01.fla and textfield.replacesel01.swf , documents the bug, and the additional file textfield.replacesel02.fla and textfield.replacesel02.swf documents a possible workaround.

Tips and precautions

The new text is not treated as HTML even if the field has the TextField.html property set.
