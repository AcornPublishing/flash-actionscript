MovieClip._yscale

myMovieClip._yscale

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 4 and later.

 

The MovieClip._yscale property sets or retrieves the vertical scale of a movie clip. This property is exactly the same as MovieClip._xscale except that it affects the vertical scaling instead of horizontal scaling. See the entry for MovieClip._xscale.
