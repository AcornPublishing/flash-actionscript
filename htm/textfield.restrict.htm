TextField.restrict

myTextField .restrict

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.restrict property limits the characters that can be entered.

Description

The TextField.restrict property is used to limit the characters a user can enter into a text field. If the TextField.restrict property is set to null , any characters can be entered. If the TextField.restrict property is set to an empty string, no characters can be entered. Otherwise, you can specify a string that contains the characters or ranges of characters that are allowable.

Examples and practical uses

The TextField.restrict property lets you limit the characters a user can type into a text field. For example, the command:

 

myTextField .restrict = "AEIOU";

 

restricts the user to uppercase vowels only. To specify a range, use the hyphen:

 

myTextField .restrict = "A-M ";

 

This will restrict the characters to uppercase A through M as well as the space (which must be explicitly entered in the string. To allow any letter, any number, and spaces, the command would be:

 

myTextField .restrict = "A-Z a-z 0-9";

 

You can explicitly prevent certain characters by using the caret sign. For example, to allow any character but numbers, use the command:

 

myTextField .restrict = "^0-9";

 

The caret can be placed anywhere in the string. To allow all lowercase letters except m through p, use the command:

 

myTextField .restrict = "a-z^m-p";

 

To enter a backslash in the string only \- , \^ and \\ are allowed. Backslashes must be escaped with a second backslash, like this:

 

myTextField .restrict = "\\-\\^";

 

Instead of specifying characters you can specify the ASCII codes by preceding the code with \u and converting the code to hexadecimal such as:

 

myTextField .restrict = "\u0020";

 

to restrict entry to ASCII code 32 (the space).

Tips and precautions

The TextField.restrict property is used to restrict or allow entry of certain characters in text fields. Combinations of characters (such as state abbreviations) cannot be restricted in this manner as the TextField.restrict property works with single characters only.
