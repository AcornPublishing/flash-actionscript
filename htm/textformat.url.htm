TextFormat.url

myTextFormat.url

 

Where myTextFormat is a TextFormat instance.

 

Compatible with Flash Player 6 and later .

 

The TextFormat.url property indicates the URL this text hyperlinks to.

Description

The TextFormat.url property is a string value indicating the URL the text hyperlinks to. If an empty string is used, there is no hyperlink. The default is null which means an undefined value

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myTextFormat.url = "testpage.htm";

 

 

The string specifies the URL of the hyperlink text.

 

 

A null value means nothing.

 

Examples and practical uses

The TextFormat.url property lets you specify the URL that a text hyperlink in a text field corresponds to:

 

myTextFormat.url = "targetfile";

Tips and precautions

If TextFormat.url property is set to null , no hyperlink exists and there is no specific value. A value of an empty string means there is no hyperlink for the text.

 

Be careful not to confuse TextFormat.url and TextField._url, they are very different. For one, TextField._url is read-only. TextFormat.url is used to apply a hyperlink to a text, which has the TextFormat object applied to it while the TextField._url property defines which the SWF file in which the TextField is contained resides.
