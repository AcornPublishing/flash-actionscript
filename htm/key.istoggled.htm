Key.isToggled

Key.isToggled(code);

 

Where code is the key code value for either the Num Lock (144) or Caps Lock (20) keys.

 

Compatible with Flash Player 5 and later. There are no known issues with any versions of Flash that support this method.

 

Key.isToggled returns true if the specified key code is toggled on.

Description

Key.isDOWN returns true if the key code value parameter is toggled on. This method is used only for the Caps Lock and Num Lock key values.

 

 

Code

 

 

Additional explanation

 

Notes

 

Key.isToggled(144);

 

If the Num Lock key is pressed, returns true.

 

 

Macintosh Caps Lock and Num Lock keys have identical key codes.

 

Examples and practical uses

The Key.isToggled() method is used to determine if the Num Lock or Caps Lock key is toggled on. This is usually used with a conditional statement to filter input:

 

if( Key.isToggled(20)){

   trace ("Caps Lock is on");

}

 

Alternatively, you could use the Key property for the same task:

 

if( Key.isToggled(Key.CAPSLOCK)){

   trace ("Caps Lock is on");

}

Tips and precautions

Only the Caps Lock and Num Lock keys can be checked for condition with this method.

 

The other key methods do not differentiate between lower case and caps, so if your application needs to test for case (for example if you are developing a drawing program In Flash that uses keyboard shortcuts where caps do different things from lowercase), you must use this method.  Note that this varies between PCs and Macs.

 

The Macintosh maps the Caps Lock and Num Lock keys to identical key codes. There is no workaround for this.
