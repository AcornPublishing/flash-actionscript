MovieClip.onUnload

myMovieClip.onUnload

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip.onUnloadevent handler is invoked when the movie clip is removed from the timeline.

Description

The MovieClip.onUnloadevent handler is invoked when the movie clip is removed from the timeline. The function associated with the MovieClip.onUnloadevent handler is launched when this happens. The event handler resides in the first frame of the movie.

Examples and practical uses

The MovieClip.onUnloadevent handler is invoked when the movie clip instance is removed from the timeline. To define the MovieClip.onUnloadevent handler, the function is written like this:

 

myMovieClip.onUnload = function {

 // code to execute when the movie is unloaded

}

 

See also MovieClip.onLoad.

Tips and precautions

The MovieClip.onUnloadevent handler is one event handler of the MovieClip object. Usually you will write functions for all the event handlers available to the MovieClip object as part of the instance definition.
