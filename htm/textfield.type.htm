myTextField .type

 

Where myTextField is a TextField instance.

 

Flash MX and later. Compatible with Flash Player 6 and later .

 

The TextField.type property sets or retrieves the type of text field.

Description

The TextField.type property sets or retrieves the type of text field. There are two types: dynamic is a dynamic text field and cannot be selected by a user, and input that is a non-dynamic input text field.

myTextField .type = "dynamic";

 

 

Sets the text field type to be dynamic.

 

Only "dynamic" and "input" are allowed values.

 

 

 

myPath = myTextField .type() ;

Examples and practical uses

The TextField.type property is the type of text field, either dynamic or input. It can be read and stored in a variable or set as one of the two string values:

 

myType = myTextField .type;

myTextField .type = "dynamic";

 

See the example textfield.type.fla and textfield.type.swf .

Tips and precautions

The TextField.type property lets you indicate whether a text field is dynamic (and hence not user selectable) or input, a standard input text field.

 

You would generally only use this property to change the type of a text field that was created dynamically via MovieClip.createTextField (the default type for a dynamically created text field is dynamic).
