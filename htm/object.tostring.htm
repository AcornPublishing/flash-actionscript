Object.toString

myObject.toString()

 

Where myObject is an Object instance.

 

This method is compatible with Flash 5 and later.

 

The Object.toString() method converts the object to a string.

Description

The Object.toString() method converts the object to a string and then returns the string.

 

 

Code

 

 

Notes

 

 

myString = myObject.toString();

 

Converts myObject to a string and stores the result in myString. In most cases, this will result in a description of the data type used by myObject.

 

 

myObject.toString("test");

 

 

 

No parameters are allowed.

Examples and practical uses

The Object.toString() method is used to convert an object to a string. The value of the string can be stored in a variable. For example, to convert myObject to a string and store the result in myString, issue the command:

 

myString = myObject.toString();

Tips and precautions

Note that the results from the Object.toString() method depend on the contents of the object.
