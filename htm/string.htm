String

This object is addressed in Chapter 12.

 

The String object is used to manipulate and manage strings. It is a wrapper for the string primitive data type, therefore allowing you to use the String object’s properties and methods directly on primitive string data types.

 

In Flash MX, the String object has been converted to a native object. That means the String object has been converted to C++ code and compiled with the Flash Player where as in previous versions it was interpreted ActionScript. This means you’ll see a huge increase in performance from the String object. Incidentally, the XML and Array objects were also converted to native objects. See also String (function) and “” (String delimiter).

Description

This object is used to manipulate and manage strings.

Examples and practical uses

See String.indexOf, string.slice and String.concat.

Tips and precautions

The difference between string literals and string objects are as follows: string objects are created by the new constructor and have a data type of Object object. String objects contain methods and properties that are used internally (by other methods of the String object) and by developers. String literals are not objects and have a data type of string, rather than Object object.
