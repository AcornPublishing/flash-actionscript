TextField._ymouse

myTextField ._ymouse

 

Where myTextField is a TextField instance

 

Compatible with Flash Player 6 and later .

 

The TextField._ymouse property retrieves the vertical coordinate of the mouse relative to the text field.

Description

The TextField._ymouse property displays the vertical (y) coordinate of the mouse cursor relative to the text field.

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myMouse = myTextField ._ymouse;

 

 

 

Stores the y coordinate of the mouse to myMouse relative to myTextField .

 

 

Y coordinates are relative to the text field.

 

 

 

myTextField ._ymouse(100);

 

 

No method of this name exists

 

 

 

 

Examples and practical uses

The TextField._ymouse property is used retrieve the vertical coordinate of the mouse. For example:

 

myYMouse = myTextField ._ymouse;

 

will assign the current value of the mouse's vertical coordinate to myYMouse . The coordinates are always relative to the text field instance.

 

See the example textfield._ymouse.fla and textfield._ymouse.swf .

Tips and precautions

The TextField._xmouse and TextField._ymouse properties are used to obtain the position of the mouse cursor relative to the text field.
