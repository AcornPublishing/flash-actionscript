escape (as a function)

a = escape(b);

 

Where a will be returned as the URL encoded version of the string b. If b is not a string, escape will attempt to use the value of b expressed as a string.

 

This global function is compatible with Flash 5 and later. See also Chapter 12. To convert URL encoded text back to standard text, you should use unescape.

Description

This global function is used to convert a string to URL encoded text. In URL encoded text, all non-alphanumeric characters (including space) are deleted and a ‘%’ followed by the deleted character’s character code (in hexadecimal) is inserted at the same position.

The original “© ” character is converted to “%C2 ” (0xC2=169), and “ 2002 ” is converted to “%202002” (“%” followed by “20” (hexadecimal for 32) followed by “2002”.

 

Examples and practical uses

URL encoded text is sometimes required when communicating with some server side scripts that expect URL encoded text.
