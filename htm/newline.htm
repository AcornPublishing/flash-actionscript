newline

newline

 

This action is compatible with Flash 4 and later.

 

The newline constant is a predefined value for a carriage return.

Description

The newline constant inserts a carriage return and is used to move down a line. It is usually used in string variables to insert a line break.

 

 

Code

 

 

Additional explanation

 

 

 

Notes

 

Str1=”Hello”;

Str2=”Goodbye”;

trace(Str1 + newline + Str2);

 

Displays the contents of Str1 followed by a carriage return, followed by the contents of Str2 on the next line so the output looks like this:

 

Hello

Goodbye

 

 

New lines can be inserted anywhere in strings to force a line break.

 

Str1=”Hello”;

Str2=”Goodbye”;

trace(Str1 + newline + newline + Str2);

 

Displays the contents of Str1 followed by a blank line, followed by the contents of Str2 on the next line so the output looks like this:

 

Hello

 

Goodbye

 

 

Examples and practical uses

The newline constant is used to add carriage returns to a text string. For example, this code displays the contents of two strings separated by a carriage return:

  

  myResult = Str1 + newline + Str2;

 

The newline constant in this example allows you to force a line break and continue the output on the next line.

 

You can also use the newline constant at the end of strings to force a new line for the next string:

 

myMessage.text = “Your address is:” + newline;

myMessage.text = myData.street + newline;

myMessage.text = myData.city + newline;

myMessage.text = myData.state + newline;

myMessage.text = myData.zip + newline;

 

In this case, the new lines allow line breaks to be added as the text is built up, showing the four pieces of information on separate lines, one under each other.

Tips and precautions

The newline constant lets you add a blank line to any output you are generating. Although normally used in strings to break output, the newline constant can be used to force spacing in forms. It lets you create one or more blank lines in a form for data returned from a function or method, but which you do not have available at the moment you are creating the form. When the function or method does return data, it can be placed on the blank lines you created without overwriting anything else you created.

 

There is nothing returned from the newline constant.
