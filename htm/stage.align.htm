Stage.align

Stage.align = “TR”

 

Where the Flash movie will align to the top-right of the stage.

 

This method is compatible with Flash 6 and above. There are no known issues with any version of Flash that supports this object. See also Stage.scaleMode.

 

The align property is accessed and altered to effect the position of the Flash movie within the stage. There are some legal string values that must be used; all other values will center the movie to the stage, both horizontally and vertically. Note that the string value for this property is not case sensitive. The following is a list of legal properties:

Incorrect usage.

 

As align is a property of the object you need to explicitly define it’s value just like any other variable or property.

 

Examples and practical uses

Using the align property of the Stage object you can dynamically define the stage alignment. You can even allow the user to choose which type of stage alignment they want to use. See example files stagealign.fla and stagealign.swf.

Tips and precautions

To use the align property you must set it to one of the legal and accepted align modes, which are string values (refer to the table of string values above). If you were to set the align property to te for example, the t would be recognized and the movie would be aligned to the top. However the e wouldn’t be recognized so the movie would be centered horizontally. The same thing goes for the reverse situation; take for example the string el. The e wouldn’t be recognized so the movie would be centered vertically. However, the l would be recognized and the movie would be left aligned horizontally.
