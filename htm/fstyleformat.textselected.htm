FStyleFormat.textSelected

a.textSelected = b;

 

Where a is the name of your custom style format, and b is a RGB color assigned to the text.

 

Compatible with Flash MX. The RGB color value is assigned to this property must use the 0xRRGGBB format. Related include FStyleFormat.selection, FStyleFormat.selectionUnfocused, and FStyleformat.textDisabled.

Description

This property changes the RGB value of selected text within a component listening to the style format.

 

 

Code

 

 

Shortcut

 

 

Notes

 

myStyle = new FStyleFormat();

myStyle.textSelected = 0xFF0000;

myStyle.applyChanges();

myStyle.addListener (myListbox);

 

 

myStyle = new FStyleFormat

({textSelected:0xFF0000});

myStyle.applyChanges();

myStyle.addListener(myListbox);

 

 

This code creates a style format myStyle, changes the textSelected value to red and applies changes to the format. The myListbox component is listening to the style.

 

Examples and practical uses

This property can be used within a custom style format to adjust the color of text when it is selected within a component.

Programming

In the example fstyleformat.textselected.fla and fstyleformat.textselected.swf , three new colors are assigned to a custom style format called myStyle. The three properties, textSelection, selection, and selectionFocused are given new values so the text remains legible to the end user. The following values are used:

 

myStyle = new FStyleFormat({textSelected:0x99ccff, selection:0x000033, selectionUnfocused:0x006699});

myStyle.applyChanges();

myStyle.addListener(list1, combo1);

 

Tips and precautions

When working with text colors it is important to remember to make the background color with enough contrast so the text is legible. Test your movie, and adjust either the textSelected or selection properties if there is insufficient contrast between the two color values.
