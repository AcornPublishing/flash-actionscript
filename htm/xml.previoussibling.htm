XML.previousSibling

myXMLNode.previousSibling

 

Where myXMLNode is an instance of an XML object.

 

Compatible with Flash 5 Player and above. The XML.previousSibling property references the previous child of the object’s child list. Refer to related entries XML.firstChild, XML.lastChild, XML.childNodes, XML.nextSibling, andXML.parentNode.

Description

The XML.previousSibling property examines the child list of an XML object instance’s parent node and returns the name of the previous sibling in the list. If there is no previous sibling to the node indicated, a null is returned. This is a read-only property.

myXML_xml = new XML("<drinks><juice>iced tea</juice><slurpee>grape</slurpee></drinks>");

thenode = myXML_xml.firstChild;

trace(thenode.previousSibling);

 

 

 

null

 

The first child node does not have any previous siblings to trace. Therefore, null is returned.

 

 

Examples and practical uses

The XML.previousSibling property is a read-only value that returns the previous sibling in the parent node’s child list. If the previous sibling does not exist, a null is returned. This property is used in the same way as related properties such as nexSibling, firstChild, and so on. Using the property, you can navigate to a particular node, and then return the value or name. You can also use this property to navigate backwards through a structure.

 

myXML_xml = new XML("<drinks><juice>iced tea</juice><dairy>milk</dairy><water/><slurpee>grape</slurpee></drinks>");

thenode = myXML_xml.firstChild.lastChild;

trace(thenode);

      //<slurpee>grape</slurpee>

trace(thenode.previousSibling);

      //<water/>

trace(thenode.previousSibling.previousSibling);

      //<dairy>milk</dairy>

 

This code begins with setting a variable thenode to the last child in the data structure. Then, it navigates back from this final child to previous siblings. See the example xml.previoussibling.fla and xml.previoussibling.swf .

Tips and precautions

The XML.previousSibling property cannot be set, only read. Therefore, it cannot be used to manipulate children.
