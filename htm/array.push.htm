Array.push

myArray.push(value0, value1,… valuen);

 

Where MyArray is an array, and value0 to valuen are values that will be added to the end of myArray.

 

This method is compatible with Flash 5 or later. It is used to add new elements to the end of an array.

Description

The push() method adds new elements to the end of an array. You can perform more complex array concatenation using the Array.concat() method.

Examples and practical uses

Array.unshift allows you to add array elements to the start of an array, and is the opposite of Array.push. See also Array.unshift. When adding to arrays, you can use these two methods together to create various kinds of buffer including:

 

·         A First In First Out (FIFO) buffer. New elements are added and removed from the start of the buffer.

·         A First In, Last Out (LIFO) buffer. New elements are added to the start of the array, and removed from the end.

Tips and precautions

You can also delete elements to the end of an array with Array.pop.
