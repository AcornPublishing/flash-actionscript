System.capabilities.manufacturer

a = System.capabilities.manufacturer

 

a will be returned as a string value that returns an identifier for the current Flash player type/

 

This property is compatible with Flash 6 or later. This property is read only.

Description

This property is used to find out the current Flash player type. There is a different Flash player type for each family of the Flash player:

 

·       The Windows Flash player will return  "Macromedia Windows".

·       The Mac Flash player will return "Macromedia Macintosh".

·       Other pocket/mobile/wireless devices will return a specific string identifying the Flash player that runs on this product family.

 

Related properties include:

 

·       System.capabilities.os

·       System.capabilities.language

·       The Flash player version variable $Version.

Examples and practical uses

This property is only useful for non-standard systems, such as PDAs and pocket/mobile devices. Such devices will typically support a reduced set of the normal flash player's features, and may include custom or proprietary capabilities. By identifying the Flash player type, you may recognize the specific player version, and from this may be able to assume additional abilities not included in the System.capabilities object.

 

This property does not relate to the Flash player version, which is one of a number of additional pieces of information available. See also tips and precautions.

 

See the example system.capabilities.fla and system.capabilities.swf .

Tips and precautions

The following code will return the full available information on the users software environment:

 

// about the Flash player...

message = "I am running on a "+System.capabilities.manufacturer+" player, version "+$Version;

trace(message);

// about the user's OS environment...

message = "\nThe OS version is "+System.capabilities.os+" using language "+System.capabilities.language;

trace(message);
