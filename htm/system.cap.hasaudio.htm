System.capabilities.hasAudio

a = System.capabilities.hasAudio;

 

a will be returned as a Boolean value.

 

This property is compatible with Flash 6 or later. This property is read only.

Description

This property is used to find out whether the user has a Flash player installed that can handle audio. It is a Boolean, and will be true if an audio capability exists. For the Flash player designed for a traditional laptop or desktop, this is true, but may vary for versions designed for other devices.

Notes

 

if (System.capabilities.hasAudio){

  // do this

}

 

If the user's machine has audio options, you can check for this and act accordingly via simple if... logic.

 

Examples and practical uses

Loading music files conditionally via this property allows your SWF to become leaner as well as know it doesn't need to play the files themselves if it finds this property is false.

 

See the example system.capabilities.fla and system.capabilities.swf .

Tips and precautions

You should note that this value refers to the platform and not the specific user. For example, a PC will always have a Flash player that supports sound, but a particular user may not have a sound card installed. The system capabilities object will only look at the Flash player, not the detailed hardware per user. This property is geared towards versions of the Flash player on non-standard devices, and these will generally be mobile/pocket/wireless devices.
