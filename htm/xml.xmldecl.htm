XML.xmlDecl

myXML.xmlDecl

 

Where myXML is an instance of the XML object.

 

Compatible with Flash Player 5 and later. The XML.xmlDecl property sets and retrieves the XML declaration.

Description

The XML.xmlDecl property is used to retrieve information about an XML document’s XML declaration. If no declaration was found, the XML.xmlDecl property has a value of undefined.

Examples and practical uses

The XML.xmlDecl property lets you set and retrieve information about an XML document’s XML declaration. If there is no such declaration, the value of the XML.xmlDecl property is undefined.

 

In the following example, we will trace the XML declaration from an XML source (myXML_xml). Then, we will assign a new XML declaration:

 

myXML_xml = new XML("<?xml version=\"1.0\" encoding=\"UTF-8\" standalone=\"yes\" ?><root><child><element /></child><child><element /></child></root>");

 

trace(myXML_xml.xmlDecl);

 

newXML = new XML();

newXML.xmlDecl = "<?xml version=\"1.0\" ?>";

newXML.parseXML("<hello><world /></hello>");

 

trace(newXML.xmlDecl);

 

So, as you can see from the above example, you can set a value for the XML.docTypeDecl property using a standard assignment. The above code can be found in the example xml.xmldecl.fla and xml.xmldecl.swf .

Tips and precautions

The XML.xmlDecl property does not validate any values in the XML declaration, and it should be used with top level XML class nodes.
