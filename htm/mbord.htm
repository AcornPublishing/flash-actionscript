mbord

mbord(char)

 

Where char is a character to convert to an ASCII value

 

Compatible with Flash Player 4 and later. Deprecated by String.fromCharAt in Flash 5 and later.

 

The mbord() string function converts a character to an ASCII value

Description

The mbord() function is a string function for converting a character to an ASCII numeric value.

 

 

code

 

 

Notes

 

 

myVar=mbord(“A”)

 

 

Converts the “A” character to its ASCII value, assigning it to myVar.

 

 

myVar=mbord()

 

 

myVar=mbord (“ABCDE”)

 

 

An argument must be supplied.

 

The argument must be a single character, not a string.

Examples and practical uses

The mbord() function is used when you need to convert a character to an ASCII value. For example:

 

myVar = mbord(“@”)

 

… will return the value of 64.

Tips and precautions

The mbord() string function was included in Flash 4, but replaced by the String.fromCharAt function (part of the String object) in Flash 5. While mbord() is still supported, using the String object method is the preferable approach because of greater flexibility.
