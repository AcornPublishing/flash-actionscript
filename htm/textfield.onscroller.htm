TextField.onScroller

myTextField .onScroller

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.onScroller event handler is invoked when a text field's scroll properties change.

Description

The TextField.onScroller event handler is invoked whenever the text field's scroll properties are changed. The function associated with the TextField.onScroller event handler is launched when this happens.

Examples and practical uses

The TextField.onScroller event handler is invoked whenever the text field instance has one of its scroll properties changed Prior to any other functions being executed; the TextField.onScroller event handler is run. To define the TextField.onScroller event handler, the function is written like this:

 

myTextField .onScroller = function() {

  // code to execute when the scroll property changes

}

 

Tips and precautions

The TextField.onScroller event handler is one event handler in the TextField object. Usually you will write functions for all the event handlers available to the TextField object as part of the instance definition.
