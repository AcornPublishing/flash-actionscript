+ addition

c = a + b

 

Where c is the result of adding a and b. a and b can each be any literal or data type holding numbers or strings.

 

This operator is compatible with Flash 5 or later. You should note that the + operator in Flash 4 will add numbers only. To add two strings or a number and string in Flash 4, you need to use the add operator.

 

When adding two values, the + operator will yield the following:

 

1.       Flash 4 and above: If a and b are numbers or can be converted to numbers (or strings that can be converted directly into numbers), then c will be the numerical value a + b.

2.       Flash 5 or above: If a or b (or both) are strings that cannot be converted into numbers then the result will be a new string that is formed by concatenating a and b together.

3.       Flash 4 only: If a or b (or both) are strings, then the result will be NaN (Not a Number).

When adding Boolean logic, Flash will first convert true to 1 and false to 0.

 

The if statement will work out as:

 

if((1+0+1)>=2){

 

Which gives a true, so you will see the “hi”.

 

This can sometimes be useful when counting ‘how many things are true’. If you wanted to do something when two out of three things were true (also called a majority gate), you could simply add them as per the second listing.

 

 

a = 1 + undefined;

b = 1 + NaN;

c = “cat” + true

 

a = 1

b = NaN

c = “cattrue”

 

When adding, Flash will ignore any values that are undefined, as long as at least one expression is defined, so a = 1.

 

Anything added to NaN becomes NaN, so b = NaN. Note that a string and NaN are not quite the same thing, because Flash can differentiate between the two.

 

It’s also worth noting is that Flash doesn’t try to convert true to the number 1 when concatenating.

 

Examples and practical uses

The + operator is used as per normal addition when using numbers (or strings which can be directly converted to numbers). The + operator is used as per normal string concatenation for almost all other instances.

 

As noted above, you can add Booleans together to form majority gates rather than definite true/false logic. When combined with the if structure, majority gates are a quick and easy way of implementing new forms of logic such as fuzzy logic and neural nets. Both these types of logic work on a majority or ‘probability of being either true or false’, rather than a definite true or false.

 

The + operator is used extensively in the example FLAs. See:

 

·       (bitwiseor).fla and (bitwiseor).swf for an example of + used for string concatenation.

·       (global).fla and (global).swf for an example of arithmetic heavy scripts using +.

 

Because Flash is frame based, any formulas you create to define motion are from a special branch of math called iterative algorithms. This means that by constantly adding (or subtracting) every frame, you can simulate complex motion that would normally require more advanced math. An example of this is shown in (add).fla and (add).swf. Here we simulate the effects of gravity by simply adding a small value to a movie clip’s velocity every frame.

Tips and precautions

If you need to write Flash sites that are compatible with Flash 4 (not an unusual occurrence because it is the version that gives the maximum user compatibility whilst maintaining most of the core features of Flash), then be aware that the Flash 4 version of + does not handle strings. You should instead use add.

 

If you open a Flash 4 file into Flash 5 or above, you will see that the code is changed such that all statements of the form:

 

c = a + b;

 

… will be converted to:

 

c = Number(a) + Number(b);

 

This is the only way that Flash can make sure that the new features of + in Flash 5 or MX do not affect the way the original Flash 4 code operated, or the results that will be seen.
