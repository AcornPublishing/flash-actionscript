TextField._rotation

myTextField ._rotation

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField._rotation property is the number of degrees the text field is rotated.

Description

The TextField._rotation property is the number of degrees the text field is rotated. The property can be set and queried.

 

 

Code

 

 

Additional Explanation

 

 

myTextField ._rotation = 10;

 

Rotates the text field 10 degrees.

 

Examples and practical uses

The TextField._rotation property returns the number of degrees the text field is rotated. It can be assigned to a variable:

 

myRotn = myTextField ._rotation;

 

The TextField._rotation property can be set by assigning a value:

 

myTextField ._rotation = 90;

 

See the example textfield._rotation.fla and textfield._rotation.swf .

Tips and precautions

The rotation property will normally be used in text transitions and animated effects associated with novel user interface designs. It goes without saying that rotating an input field dynamically is not something the user will thank you for!

 

You must embed font outlines before Flash can treat a text field as a graphic. Unless Flash can treat text as a graphic, you cannot perform property-based animation (including _rotation , but not _x and _y ). See also the Tips and precautions section for TextField .
