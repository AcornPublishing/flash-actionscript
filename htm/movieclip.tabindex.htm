MovieClip.tabIndex

myMovieClip.tabIndex

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip.tabIndexproperty controls automatic tab ordering of objects in a movie.

Description

The MovieClip.tabIndexproperty lets you control how all objects in a movie are handled by tab ordering. If an object in a movie has the MovieClip.tabIndexproperty, automatic tab ordering is disabled and the values specified in the MovieClip.tabIndexproperty are used for ordering instead. If the MovieClip.tabIndexproperty is set to a positive integer then all integers are sorted into increasing order when tab ordering is performed. Only objects with the MovieClip.tabIndexproperty are included in the sort.

Notes

 

 

myMenu.tabIndex = 6;

 

Sets the tabIndex property of myMenu to 6.

 

By default all objects have the MovieClip.tabIndexproperty value undefined.

 

 

myMenu.tabIndex();

 

 

There is no method with this name, only a property.

 

 

 

Examples and practical uses

The MovieClip.tabIndexproperty controls the ordering of objects in a movie when tab ordering is invoked. If any object in a movie has the MovieClip.tabIndex property, then automatic tab ordering is disabled. For sorting, all objects that have a MovieClip.tabIndexproperty setting are sorted by value, from smallest to largest. The MovieClip.tabIndexproperty is a positive integer. Any object that does not have the MovieClip.tabIndex property is not included in the sort. By default, the MovieClip.tabIndexproperty is undefined for all objects.

 

You should use only unique numbers for the MovieClip.tabIndexproperty. All values are sorted in ascending order. To assign a value to an object’s MovieClip.tabIndexproperty, use the standard assignment:

 

myMovieClip.tabIndex = 4;

Tips and precautions

The MovieClip.tabIndexproperty takes precedence over automatic tab ordering if any object has this property. Index numbers should be unique.
