FStyleFormat.textDisabled

a.textDisabled = b;

 

Where a is the name of your custom style format, and b is the RGB value assigned to text in a disabled component.

 

Compatible with Flash MX. When a component is disabled, the text will change to the color value you assign for this property. The RGB color value is assigned using the 0xRRGGBB format. Related entries include FStyleFormat.backgroundDisabled andFStyleFormat.selectionDisabled.

Description

This property changes the color value of the text in a disabled component instance.

These will change the color of text when the component instance myListBox is in a disabled state.

 

Examples and practical uses

You can use this property to change the color of text when a component is in a disabled state.

Programming

As we see in our example fstyleformat.textDisabled.fla and fstyleformat.textDisabled.swf , this property controls the components text color when it is disabled. The instances button4, radio4, and check4 are all in a disabled state. They reflect a changed color value because all instances are listening to our style formattextColor.

Tips and precautions

This color value only affects the CheckBox, RadioButton, and PushButton components. list box and combo box components do not reflect this color value property when they are in a disabled state.
