MovieClip._visible

myMovieClip._visible

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 4 and later.

 

The MovieClip._visibleproperty determines whether the movie clip is visible or not.

Description

The MovieClip._visible property is a Boolean that indicates whether the movie clip instance is visible or not. If not visible, the clip is disabled.

 

 

Code

 

 

Additional explanation

 

 

Notes

 

 

myMovieClip._visible = true;

 

The myMovieClip object is visible and can be clicked.

 

 

Non-visible objects cannot be clicked.

 

 

myMovieClip._visible = “true”;

 

Only Booleans can be used for this property (this is a string).

 

 

 

Examples and practical uses

The MovieClip._visibleproperty controls whether a movie clip (such as a button movie clip) is visible or not. A non-visible movie clip cannot be clicked and is invisible to the user. The MovieClip._visibleproperty is assigned as a Boolean:

 

myMovieClip._visible = false;

Tips and precautions

Setting a movie clip invisible by setting  _visible = false is preferable to setting alpha to zero, as it is more processor friendly.
