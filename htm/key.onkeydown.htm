Key.onKeyDown

Key.onKeyDown;

 

Compatible with Flash Player 6 and later.The movie clip key events don’t work so you have to use this method if you want to detect key presses via an event driven route.

 

The Key.onKeyDown method is used with a listener object registered using Key.addListener. This method is triggered when a key is pressed.

Description

Key.onKeyDown is a method invoked when a key is pressed. It will be triggered only on a registered object (see Key.addListener).

Examples and practical uses

The Key.onKeyDown method (and its matching function Key.onKeyUp) is used with registered objects that are listening for a key press or key release. Key.onKeyDown is triggered when a key is pressed.

 

To define Key.onKeyDown as part of a key object instantiation:

 

myKeyobject = new Object();

 

myKeyobject.onKeyDown = () {

   // code for key presses

   }

 

After the onKeyDown method has been defined, the Key object is registered so that the onKeyDown method is invoked properly:

 

Key.addListener (myKeyobject);

 

In the above example, // code for key presses would normally use other methods of the Key object to work out exactly which key is being pressed (see Key.getASCII and Key.getCode).

Tips and precautions

The Key.onKeyDown method is persistent for as long as the key is held down, and it can be masked by multiple keys being down, so the Key.onKeyUp is more useful.
