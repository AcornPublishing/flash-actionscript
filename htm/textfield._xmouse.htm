TextField._xmouse

myTextField ._xmouse

 

Where myTextField is a TextField instance

 

Compatible with Flash Player 6 and later .

 

The TextField._xmouse property retrieves the horizontal coordinate of the mouse relative to the text field.

Description

The TextField._xmouse property equals the horizontal (x ) coordinate of the mouse cursor relative to the text field.

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myMouse = myTextField ._xmouse;

 

 

 

Stores the x coordinate of the mouse to myMouse relative to myTextField .

 

 

X coordinates are relative to the text field

Examples and practical uses

The TextField._xmouse property is used retrieve the horizontal coordinate of the mouse. For example:

 

myXMouse = myTextField ._xmouse;

 

will assign the current value of the mouse's horizontal coordinate to myXMouse . The coordinates are always relative to the text field instance.

 

See the example textfield._xmouse.fla and textfield._xmouse.swf .

Tips and precautions

There is little or no reason to ever use this property. Generally, you would want to know the value of this property every frame, and the best way to do this is via a MovieClip.onEnterFrame event. You should therefore consider using movie clips when tracking the mouse position, given that the movie clip also has an _xmouse property as well as an onEnterFrame event.

 

The only time you could ever want to apply it to text fields is if you were creating a text-based mouse follower, where letters follow the path created by the mouse. This is the sort of retro done-to-death-in-Flash-4-trick that will only make your fellow designers laugh out openly in the street about your lack of ideas, so don't do it!
