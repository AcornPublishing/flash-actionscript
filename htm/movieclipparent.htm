MovieClip._parent

myMovieClip._parent

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip._parentproperty is a reference to the movie clip that contains the current movie clip instance.

Description

The MovieClip._parentproperty is a reference to the name of the movie clip instance’s parent – the movie that contains the clip. This is often used in generic scripts to store the clip name in a variable.

 

 

Code

 

 

Notes

 

myName = myMovieClip._parent;

Assigns the path of the movie clip instance’s parent to MyName.

 

If the movie clip is not a child, nothing is returned.

 

 

myName = myMovieClip._parent(_parent);

 

 

No parameters can be specified.

Examples and practical uses

The MovieClip._parentproperty returns the name of the parent movie clip that contains the movie clip instance. This is used to reference the parent in other scripting commands. You can store the path to the parent in a variable, or use it directly in a command:

 

myName = myMovieClip._parent;

myXpos = _parent._xmouse;

Tips and precautions

The MovieClip._parentproperty simply returns the path to the parent of the movie clip (if there is one). By storing the name in a variable, the script can be used for any movies.
