XML.appendChild

 

myXML_xml.appendChild(a)

 

Where myXML_xml is an XML instance and a is a child node to be added to the instance.

 

Compatible with Flash 5 Player and above. This method is used to add a new child node to a specified XML object. It is used in conjunction with the createElement and createText node methods, and is used to create new XML data in a Flash movie. Related entries include XML.createElement and XML.createTextNode.

Description

The XML.appendChild() method lets you append a new child node to an XML object’s child list. If the node is already attached to a parent node, it is removed from the parent and attached to the XML instance.

Examples and practical uses

This method is used with XML.createElement or XML.createTextNode to create new nodes using Flash. You can add either kind of node type (element or text node) using appendChild and either method. This can be particularly useful if you need to send new XML to a server based on user input, to add to an XML structure.

 

To create a new node, you will first need to create the child node (either a text node, or an element), and then use appendChild to append it to your XML tree. You would create a new element node in the following way:

 

myXML_xml = new XML();

myElement = myXML_xml.createElement("mycat");

myXML_xml.appendChild(myElement);

 

This code will create a new element called mycat, output in the following way:

 

<mycat/>

 

Then, if you wanted to append a text node to this element, you would add the following two lines onto the previous code block:

 

myTextNode = myXML_xml.createTextNode("whiskers");

myXML_xml.firstChild.appendChild(myTextNode);

 

This code appends the text node to the firstChild element (mycat). It would now create the XML:

 

<mycat>whiskers</mycat>

 

See the example xml.appendchild.fla and xml.appendchild.swf for this in use.

Tips and precautions

The XML.appendChild() method allows you to both move nodes from one parent to another, and to create a new node and add it to a parent. You need to create a new node using createTextNode or createElement before you can append the node onto the existing XML structure.
