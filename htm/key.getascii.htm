Key.getASCII

Key.getASCII();

 

Compatible with Flash Player 5 and later. There are no known issues with any versions of Flash that support this method.

 

Key.getASCII returns the ASCII code of the last key pressed or released. No parameters are allowed.

Description

Key.getASCII returns the ASCII value of the last key press or key release.

Examples and practical uses

The Key.getASCII() method returns the ASCII value of the last key press or key release. This is sometimes assigned to a variable:

 

LastPress = Key.getASCII();

 

More usually, it is used along with a key based event to return the value of the last key pressed. When used in this way, the getASCII method will return the last key that changed, and do it in real time. The following code will output the last key released, and is useful for detecting single key presses:

 

keyListener = new Object();

keyListener.onKeyUp = function() {

   lastKeypress = Key.getASCII();

   message = "you released key with ASCII code "+lastKeypress;

   trace(message);

};

Key.addListener(keyListener);

 

The following code shows the disadvantage of using getASCII:

 

keyListener = new Object();

keyListener.onKeyDown = function() {

   lastKeypress = Key.getASCII();

   message = "you pressed key with ASCII code "+lastKeypress;

   trace(message);

};

keyListener.onKeyUp = function() {

   lastKeypress = Key.getASCII();

   message = "you released key with ASCII code "+lastKeypress;

   trace(message);

};

Key.addListener(keyListener);

 

If you try pressing two keys together you will notice that only one value is returned, so making this useless for detecting multiple keys that are constantly down. Use key.isDOWN instead.

Tips and precautions

No parameters are allowed with this method.

 

The Key.getASCII method will return zero if no key has been pressed. You should of course note that this method returns the value of the last key pressed or released, and this may have happened some time ago, unless you combine it with a key based event.
