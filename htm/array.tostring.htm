Array.toString

myString = myArray.toString();

 

Where myArray is an array. myString will be returned as all the elements of myArray turned to strings and concatenated, separated by commas.

 

For example:

 

numbers = [2, 4.5, 6];

numString = numbers.toString();

 

… would give numString = “2,4.5,6”

 

myString = myArray.toString() is exactly the same as myString = myArray.join() (toString is the same as join using the default argument). See Array.join() for more details.

 

In general, you should use the join method in preference to toString because:

 

·       Array.join() largely deprecates Array.toString().

·       Array.join() gives you more options on how the final string will appear.
