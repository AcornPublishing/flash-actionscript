System.capabilities.hasMP3

a = System.capabilities.hasMP3;

 

a will be returned as a Boolean value.

 

This property is compatible with Flash 6 or later. This property is read only.

Description

This property is used to find out whether the user has a Flash player installed that can handle MP3 audio. It is a Boolean, and will be true if the capability exists. For the Flash player designed for a traditional laptop or desktop, this is true, but may vary for versions designed for other devices.

 

 

Code

 

 

Notes

 

if (System.capabilities.hasMP3){

  // do this

}

 

If the user's machine has audio options, you can check for this and act accordingly via simple if… logic.

 

Examples and practical uses

This property will be returned as true for the PC and Mac but may vary for versions of the Flash player designed for other devices. IF you expect your Flash presentation to be accessed by a range of devices, you are advised to check this property before downloading MP3 assets.

 

See the example system.capabilities.fla and system.capabilities.swf .

Tips and precautions

You should note that this value refers to the platform and not the specific user. For example, a PC will always have a Flash player that supports MP3, but a particular user may have an integrated PDA and mobile phone that has a Flash player that only supports raw WAV files.

 

In particular, this property will not look at other sound codecs a PC or Mac may have installed in the operating system. This property is geared towards versions of the Flash player on non-standard devices, and these will generally be mobile/pocket/wireless devices.
