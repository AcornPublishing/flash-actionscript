XMLSocket.close

myXMLSocket.close()

 

Where myXMLSocket is an instance XMLSocket object.

 

Compatible with Flash Player 5 and later. The XMLSocket.close() method closes a connection made between the Flash movie and the socket server. Refer to the related entries on XMLSocket.connect() and XMLSocket.onClose.

Description

The XMLSocket.close() method closes a socket connection specified by the calling instance and terminating the persistent connection.

When a button instance is pressed, the connection is terminated. However, if the connection never existed in the first place, the command will fail.

Examples and practical uses

The XMLSocket.close() method is used to close a socket connection that is established. For example, the code in the example above will terminate a connection made by mysocket when the button instance myButton is released.

 

This method is used when you want to purposefully close the connection. The connection will automatically be closed when a user closes the Flash movie window. This method only needs to be used if you wish to offer the user the option to close their connection while leaving the browser window open. For more information on the close method, refer to the example xmlsocket.close.fla and xmlsocket.close.swf .
