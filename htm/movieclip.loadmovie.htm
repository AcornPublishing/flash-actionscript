MovieClip.loadMovie

myMovieClip.loadMovie (URL [, method])

 

Where myMovieClip is a movie clip instance, URL is the path to the movie SWF file, and method is the means of transferring variables (if any) to the method (GET or POST).

 

Compatible with Flash 5 and later.

 

The MovieClip.loadMovie() method plays a movie without closing the Flash player.

Description

The MovieClip.loadMovie() method is used to load a new movie into an existing Flash player and play that movie, without closing and reloading the player. The MovieClip.loadMovie() method can load SWF and JPEG files while a movie is still playing, allowing multiple movies to play one after another. (Usually, the Flash player loads a single movie and closes after that movie has played.)

Examples and practical uses

The MovieClip.loadMovie() method loads a movie into an existing Flash player and allows more than one movie to be played through that player, without reloading. A SWF movie file, or a JPEG image file can be loaded using the MovieClip.loadMovie() method at any time while a movie is being played. The MovieClip.loadMovie() method requires the URL of the SWF or JPEG file to be provided as an argument, either in absolute or relative path format. As usual the URL must be in the same subdomain as the existing movie file. For example:

 

myMovieClip.loadMovie(“nextMovie.swf”);

 

… loads the specified movie while an existing movie is played. Any movie read into a player using the MovieClip.loadMovie() method inherits the position, rotation and scale properties of the existing movie.

 

The optional parameter in the MovieClip.loadMovie() method lets you specify either a GET or POST method for transferring variables to the movie. The GET method uses a URL descriptor while the POST method uses an HTTP header. If there are a lot of variables to be sent, POST is the better method. If there are no variables to be transferred, this option can be ignored. To use the POST method to transfer variables you would use the command:

 

myMovieClip.loadMovie(“nextMovie.swf”, “POST”);

Tips and precautions

The MovieClip.loadMovie() method loads a new movie into the Flash player, allowing multiple movies to be played with one player instance. To unload a movie loaded with the MovieClip.loadMovie() method, use the MovieClip.unloadMovie() method. If you simply want to update variables in an existing movie, use the MovieClip.loadVariables() method.
