MovieClip._droptarget

myMovieClip._droptarget;

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 4 and later.

 

The MovieClip._ droptarget property returns the absolute path of the movie clip instance.

Description

The MovieClip._ droptarget property returns the absolute path (using slash notation) of the movie clip instance on which the clip was dropped. Although most Flash designers have stopped using slash notation, slash notation is still used on those methods that are likely to break Flash 4 SWFs still on the web if the Flash player were to be updated to return dot notation.

 

This is not a problem, because you can simply use eval to convert to dot notation.

Notes

 

myPath = myMovieClip._droptarget;

 

If myClip is on _root, then myPathwill be returned as:

 

/myMovieClip

 

 

myPath = eval(myMovieClip._droptarget);

 

 

This time eval is used to convert the path into dot notation, and myPathwill be returned as:

 

_level0.myMovieClip

 

Examples and practical uses

MovieClip._ droptarget returns an absolute path of the movie clip instance on which the clip was dropped. Slash notation (starting with a /) is used to return the path. For example:

 

myPath = myMovieClip._droptarget;

 

stores the absolute path of myMovieClip in a variable, although you would normally use it as a condition in an if statement or use it to form a path directly.

 

eval(myMovieClip._dropTarget).dropped = true;

 

… would set a variable inside the movie clip that you dropped mymovieClipto true.

 

See also the example file movieclipdroptarget.fla / movieclipdroptarget.swf.

 

Tips and precautions

You can make the dragged movie clip smoother by using updateAfterEvent().

 

The example file mcdroptargetsmooth.fla / mcdroptargetsmooth.swf shows how to do this, using the much misunderstood updateAfterEvent() action.
