Number.toString

myNumber.toString(num)

 

Where myNumber is an instance of the Number class or a variable, and num is the numeric base to use for conversion.

 

This property is compatible with Flash 5 and later.

 

The Number.toString() method converts a number to a string.

Description

The Number.toString() method converts a number to a string. By default, a base-10 radix is used for converting the number, but you can specify another radix in the range from 2 to 36. The use of a radix allows you to handle binary, octal, hexadecimal, and other number base systems.

Additional explanation

 

 

Notes

 

 

myNumber = 15

myString = myNumber.toString();

 

 

Converts myNumber ’s value to a string, which will be “15”.

 

If no radix is specified, base 10 is assumed.

 

 

myNumber = 12

myString = myNumber.toString(16);

 

 

Converts myNumber ’s value to a hexadecimal (base 16) string, resulting in “C”.

 

 

 

myNumber = 183

myString = myNumber.toString(2);

 

 

Converts myNumber ’s value to a binary (base 2) string, resulting in “10110111”.

 

Examples and practical uses

The Number.toString() method is used to convert a number to a string. For example, the command:

 

myString = myNumber.toString();

 

… will convert the result stored in myNumber to a string using base 10. To convert using a different radix, specify the radix in the parentheses:

 

myString = Number.toString(16);

 

This will convert the number stored in myNumber to a hexadecimal string. For example, the code:

 

myNumber = 24380

myString = Number.toString(16);

 

… will result in a value of “5F3C” being stored to myString.

 

Having said all this, there’s no real reason to use the Number.toString() method (see Boolean.toString() for the explanation) unless you need to convert between radix (base) values. This is often necessary when dealing with colors, which are referenced using hexadecimal. See the numbertostring.fla / numbertostring.swf example to see the Number.toString() method in use for this purpose.

Tips and precautions

The Number.toString() method allows for conversion to strings of any number, using any radix. For example, you can devise a calculator that shows binary, octal and hex string representations of a decimal number using this code (assuming the user enters the number into the variable myNumber ):

 

myBinary = Number.toString(2);

myOctal = Number.toString(8);

myHex = Number.toString(16);
