TextField.selectable

myTextField .selectable

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.selectable property indicates whether the text field is selectable or not.

Description

The TextField.selectable property is a Boolean that indicates whether the text field is selectable or not. If true the field can be selected; if false , the field cannot be selected. The property can be set and queried.

Notes

 

myTextField .selectable = true;

 

Lets the text field be selected by the user.

 

Only Boolean values are allowed.

 

Examples and practical uses

The TextField.selectable property indicates whether a text field is selectable or not. It can be assigned to a variable:

 

myRotn = myTextField .selectable;

 

The TextField.selectable property can be set by assigning a value:

 

myTextField .selectable = true;

 

See the example textfield.selectable.fla and textfield.selectable.swf .

Tips and precautions

The TextField.selectable property is a flag that you can use to disable text selection in a text field. It is common practice to set this property to true when displaying HTML text in a text field as it helps HTML users to feel a little more comfortable in Flash. It is also quite useful, it can be really annoying when you have to re-type a whole bunch of useful information simply because the developer wont allow you to select the text.

 

Generally, you would set this property manually via the Selectable button on the property inspector:

 

 

The only time you would want to do otherwise is:

 

·       When you create your text fields dynamically via MovieClip.createTextField .

·       When you have designed a page that takes a few seconds to load up, and you do not want the user to be able to select any text fields until all the page (and especially the code that handles the operation of the text fields) has loaded up and is operational.
