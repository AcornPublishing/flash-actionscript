TextField._height

myTextField ._height

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField._height property sets or retrieves the height of the text field in pixels.

Description

The TextField._height property is used to both set and display the height of a text field. The height is reported in pixels.

Examples and practical uses

The TextField._height property is used to set and retrieve the height of a text field in pixels. To use the TextField._height property to retrieve the height of a text field, assign the result of the property to a variable:

 

myHeight = myTextField ._height;

 

To set a height, assign a value to the property:

 

myTextField ._height = 100;

Tips and precautions

The TextField._height property is handy in setting the height of a text field, and checking the height during playback of a movie. The corresponding TextField._width property does the same task for the width of the text field.

 

Unlike the _height and _width properties for a movie clip or button, changing a text field height or width does not alter the height/width of the text font; it just increases/decreases the available text field space.

 

You are unlikely to want to change the size of a text field using this property, given that there are other methods available to resize a text field to cater for the size of text being entered (see TextField.autoSize ). The only real time you would want to specify a size for a text field is when you are creating them dynamically via MovieClip.createTextField . This method allows you to specify a width as part of its arguments, so you don't need to use _height directly even here!
