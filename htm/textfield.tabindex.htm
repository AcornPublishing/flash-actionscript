TextField.tabIndex

myTextField .tabIndex

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.tabIndex property controls automatic tab ordering of objects in a movie.

Description

The TextField.tabIndex property lets you control how all objects are handled by tab ordering. If a text field has the TextField.tabEnabled property set to false , it will not be included in TAB ordering.

 

If the TextField.tabEnabled property is set to true , then the ordering is defined by the TextField.tabIndex property. Buttons and movie clips can also be included in the tab index (they also have the tabEnabled and tabIndex properties).

Examples and practical uses

The TextField.tabIndex property controls the ordering of objects when tab ordering is invoked. If any object has the TextField.tabIndex property, then automatic tab ordering is disabled. For sorting, all objects that have a TextField.tabIndex property setting are sorted by value, from smallest to largest. The TextField.tabIndex property is a positive integer. Any object that does not have the TextField.tabIndex property is not included in the sort. By default, the TextField.tabIndex property is undefined for all objects.

 

You should use only unique numbers for the TextField.tabIndex property. All values are sorted in ascending order. To assign a value to an object's TextField.tabIndex property, use the standard assignment:

 

myTextField .tabIndex = 4;

 

See the example textfield.tabindex.fla and textfield.tabindex.swf .

Tips and precautions

The TextField.tabIndex property takes precedence over automatic TAB ordering if any object has this property. The automatic (or default) TAB ordering is the order that you place your graphics on the stage. Index numbers should be unique, and you are also advised to increment your tab ordering in 10s (i.e. 10, 20, 30, etc). This allows you to add additional elements within your tab ordering without having to renumber everything else!
