MovieClip.onSetFocus

myMovieClip.onSetFocus(oldobject)

 

Where myMovieClip is a movie clip instance and oldobject is the object that lost focus.

 

Compatible with Flash 6 and later.

 

The MovieClip.onSetFocusevent handler is invoked when a movie gets keyboard input focus. The object that loses the focus is specified inside the event handler function.

Description

The MovieClip.onSetFocusevent handler is invoked when a movie is given input focus. The function associated with the MovieClip.onSetFocusevent handler is launched when this happens and that function specifies a new object to receive the focus.

Examples and practical uses

The MovieClip.onSetFocus event handler is invoked when a movie clip gains keyboard input focus from another object that had focus. The function defined as part of the event handler definition is launched on receipt of data and contains the new object to receive focus. To define the MovieClip.onSetFocus event handler, the function is written like this:

 

myMovieClip.onSetFocus = function (oldobject) {

 // code for the focus change

}

 

The MovieClip.onSetFocus event handler function specifies the object that is to lose the focus as a parameter and can then invoke code in the body of the function. If no object had focus, a null can be used instead.

Tips and precautions

The MovieClip.onSetFocusevent handler is one event handler in the MovieClip object.

In the case of MovieClip.onKillFocus and MovieClip.onSetFocusevent handlers, the focus is being shifted from one object to another object.
