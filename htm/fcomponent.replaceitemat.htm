FComponent.replaceItemAt

(This method is applicable to FComboBox.replaceItemAt and FListBox.replaceItemAt. )

 

myComponent.replaceItemAt(index, label);

myComponent.replaceItemAt(index, label, data);

 

Where myComponent is a combo box or list box instance. The label at index index will be removed, and replaced by a new label label. You can optionally also add a new data item associated with the new label.

 

Where myComponent is a combo box or list box instance. The label at index index will be removed, and replaced by a new label label. You can optionally also ass a new data item associated with the new label.

 

This method is compatible with Flash 6.

Description

This method will delete the label at the specified index and replace it. The combo box or list box will be redrawn to reflect this change.

Notes

 

myListBox.replaceItemAt(4, “ blue ” );

myListBox.replaceItemAt(5, “ red ”, 45);

 

 

The first line removes the label at index 4 of the list box myListBox. This will remove the 5th label from the top and replace it with the label “ blue ”.

 

The second line will remove the label at index 5 (6th label down) and replace it with the label “ red ” and associated data value 45.

 

 

component.replaceItemAt(component.getSelectedIndex(), “ selected ” );

 

 

Replaces the last selected label with “ selected ”. See also FComponent.getSelectedIndex.

 

You would typically use this code within the component change handler.

 

Examples and practical uses

Replacing labels in a combo box or list box is one of the features you need to build dynamic and context sensitive controls.

 

See also the example fcomponentreplaceitemat.fla / fcomponentreplaceitemat.swf.
