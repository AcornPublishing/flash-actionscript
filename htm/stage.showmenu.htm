Stage.showMenu

Stage.showMenu = true;

 

Where the showMenu property of the Stage object is being set to true.

 

This property is compatible with Flash 6 and above. There are no known issues with any version of Flash that supports this object.

 

The showMenu property is used to hide the Zoom In, Zoom Out, 100% and Show All options from the right-click (Windows) or Ctrl -click (Mac) context menu. This setting controls both the Flash Player plug-in and the Flash Player standalone (or projector). You can use this property to define if you would like users to the see these items in the context menu. You may want to curb access to items on the menu such as Zoom In and Zoom out.

Description

This method determines if various options on the Flash player context menu are visible.

Stage.showMenu = true;

 

 

Standard usage of this property.

 


Stage.showMenu = “true”;

 

Incorrect usage.

 

The showMenu will only accept a Boolean value of true or false. Any incorrect string value will result in the property being equal to false ; therefore the context menu will hide the items.

 

Examples and practical uses

Setting the showMenu property of the Stage object to true will give you (the developer) greater control over what the user can do with your SWF file, by removing some options. However, this will also limit functionality. A great feature of SWF files is the ability to zoom in and out with no loss of quality when using vector graphics (you should be aware that bitmaps will pixelate). See the example stageshowmenu.fla / stageshowmenu.swf for methods on controlling the showMenu property.

Tips and precautions

The default value for the showMenu property is true. If this is not defined via ActionScript the menu options will be shown on the Flash player context menu. This setting can also be set in the HTML embed and object tags and reflects the settings chosen in the Publish Movie dialog box. However, if you set the property using ActionScript, this will override the HTML setting.
