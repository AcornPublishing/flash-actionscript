MovieClip.onKillFocus

myMovieClip.onKillFocus(newobject)

 

Where myMovieClip is a movie clip instance and newobject is the object to receive focus.

 

Compatible with Flash 6 and later.

 

The MovieClip.onKillFocusevent handler is invoked when a movie loses keyboard input focus. A new object to receive the focus is specified inside the event handler function.

 

This event is the same as the corresponding Button event Button.onKillFocus. See the entry for Button.onKillFocus.
