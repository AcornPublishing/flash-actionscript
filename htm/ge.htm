ge

a ge b

 

Where aand b are expressions, which can be numbers, strings or variables.

 

Compatible with Flash 4. This operator was deprecated in Flash 5 and does not work in either Flash 5 or Flash MX. Related entries include gt and the >= operator.

Description

This comparison operatoris used to compare two expressions. If the first expression is greater than or equal to the second expression, a Boolean value of true is returned. Otherwise, a false value is returned.

This operator is deprecated and does not return accurate results. Instead, write var1 >= var2.

 

Tips and precautions

Since this operator is not reliable in Flash MX, it should not be used in movies for the Flash Player 6. Instead, use >=, refer to the entry on this operator for more information on using it in your code. If you are importing Flash 4 (or earlier) files into Flash MX, make sure to change all ge operators to >=.
