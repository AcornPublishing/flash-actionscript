MovieClip.onMouseDown

myMovieClip.onMouseDown

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip.onMouseDownevent handler is invoked when a mouse button is pressed.

Description

The MovieClip.onMouseDownevent handler is invoked any time a mouse button is pressed. The function associated with the MovieClip.onMouseDownevent handler is launched when this happens.

Examples and practical uses

The MovieClip.onMouseDownevent handler is invoked when a mouse button is pressed. Whatever function has been defined as part of the event handler definition is launched on receipt of data. To define the MovieClip.onMouseDownevent handler event handler, the function is written like this:

 

myMovieClip.onMouseDown = function {

 // code for the mouse button press

}

Tips and precautions

The MovieClip.onMouseDownevent handler is one event handler in the MovieClip object.

Usually you will write functions for all the event handlers available to the MovieClip object as part of the instance definition. This event is used in draw program events (such as moviecliplinestyle.fla / moviecliplinestyle.swf). This event is also associated with updateAfterEvent.
