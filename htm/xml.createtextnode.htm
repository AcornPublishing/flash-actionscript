XML.createTextNode

myXML.createTextNode("a")

 

This is where myXML is an XML instance, and a is a text string of the newly created text node.

 

This method is compatible with Flash 5 Player and above. It is used to create text node data, but cannot be used for attributes or element nodes. See related entries on XML.appendChild, XML.createElement, and XML.attributes.

Description

The XML.createTextNode() method creates a new XML text node. The method returns a reference to the newly created text node. You must use the appendChild method in order to add the text node to the XML object instance.

myXML_xml = new XML("<laundry><shirts/></laundry>");

myText = myXML_xml.createTextNode("3");

myXML_xml.firstChild.firstChild.appendChild(myText);

trace(myXML_xml);

 

 

<laundry><shirts>3</shirts></laundry>

 

A new text node of 3 was created for the <shirts> element in the myXML_xml object.

 

 

myXML_xml = new XML("<laundry><shirts/></laundry>");

myText = myXML_xml.createTextNode("3");

myXML_xml.appendChild(myText);

trace(myXML_xml);

 

 

<laundry><shirts /></laundry>3

 

The child is not appended to the correct element, so it is not a well-formed structure!

 

Examples and practical uses

When we create a text node, we ideally want it to be within a new or existing element. You will need to make sure that you append the new child node to the correct XML element.

 

myXML_xml = new XML("<cat/>");

myText = myXML_xml.createTextNode("whiskers");

myXML_xml.firstChild.appendChild(myText);

trace(myXML_xml);

 

This code snippet will obviously return <cat>whiskers</cat>. When the text node has been created, the XML.createTextNode() method returns a reference to the object representing the new text node. In the previous code, the reference is stored in a variable myText:

 

myText = myXML_xml.createTextNode("whiskers");

 

Then, we append this variable to a specified node, in this case the firstChild which is <cat/> with the following line of code:

 

myXML_xml.firstChild.appendChild(myText);

 

See the example xml.createtextnode.fla and xml.createtextnode.swf for more information on this method.

Tips and precautions

You can also make new XML data using the XML.createElement() method, but it cannot be used to create new text nodes. You should also note that text node cannot have parent, children, or sibling nodes. This method can only be invoked on an instance of the top level XML class.
