Array.splice

MyArray.splice(start, numberOfDeletions);

 

Where myArray is an array. Starting from index start, a number of elements equal to numberOfDeletions are deleted.

 

You can also insert new elements starting from the start point with the following form:

 

myArray.splice(start, numberOfDeletions, newValue0, newValue1, … newValueN);

 

Where newValue1… newValueN are the new elements that will be added.

 

This method is compatible with Flash 5 or later. Array.splice() allows you to add/delete/replace portions of an array.

This time we delete zero elements starting from smiths[3], and add two new elements. This is the same as adding two elements starting from index 3, to give you the array shown below, with Fred and Brenda added between Bob and Bill.

integers.splice(2,2, 90, 91);

Two elements starting from integers[2] are deleted, and two new elements with values 90 and 91 take their place. The effect as if the values of integers[2] and integers[3] were simply overwritten.

 

You can also add more or less elements than you delete, allowing you to remove/add new elements at the same time as the deletion.

 

 

integers = [0, 1, 2, 3, 4, 5, 6];

integers2 = [90, 91];

integers.splice(2, 2, integers2);

 

Looking at the last example, you may be fooled into expecting that this code will insert an array integers2 into integers so that you finish with the same end result. In fact, the splice will create a nested array by inserting integers2 into integer[2]. Because integers2 counts as a single element, integer[3] ends up being deleted, and the other elements move down to fill the space.

 

 

Examples and practical uses

The Array.splice allows you to perform fairly complex cut/paste/replace operations within a single line. It is useful for complex editing of an array.

Tips and precautions

If you want to delete elements without shifting the remaining elements down (you want the array length to stay the same), you should replace elements with the same number of undefined element values:

 

integers = [0, 1, 2, 3, 4, 5, 6];

integers.splice(2,2, undefined, undefined);

Note that the length property of the array is preserved, because elements 4 - 6 are not allowed to move down to fill up the gap that deletion would cause.

 

The Array.splice method changes the original array directly. If you want to preserve the original array, copy the array and splice to the copy. The easiest way to copy an array is to simply use:

 

myArrayCopy = myArray;
