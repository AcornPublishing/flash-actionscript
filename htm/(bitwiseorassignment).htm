|= bitwise OR assignment

b |= a

 

Where a is a number (or expression that will resolve to one). b will return the result of:

 

1.       Converting the initial value of both a and b to unsigned 32 bit integers

2.       Performing the expression b = b|a

 

In short, b |= a is a shorthand way of writing the following:

 

b = b | a;

 

This syntax is the normal bitwise OR operator. Refer to the entry for | (bitwise OR) for further information.
