mbsubstring

mbsubstring(string, num, count)

 

Where string is a string, num is the number of the first character to extract, and count is the number of bytes to extract

 

Compatible with Flash Player 4 and later. Deprecated by String.substr in Flash 5 and later.

 

The mbsubstring() string function extracts a substring from a longer string, starting at a particular position and extending for a specified number of characters. The first character in the string is taken to be position 1.

Description

The mbsubstring() function is a string function extracting a substring from a longer string. You have to provide the string (either directly or through the use of a variable), as well as the starting position of the substring within the string, and the length of the substring to extract.

 

 

Code

 

 

Additional explanation

 

 

myVar=mbsubstring(“This is a test”, 6, 2)

 

 

Extracts a substring from “This is a test” starting at the sixth character and extending two characters.

 

The string value “is” is returned.

Examples and practical uses

The mbsubstring() function is handy for extracting parts of a larger string for further use. For example, this command parses a full name into two components:

 

myVar = “Tim Parker”;

mySurname = mbsubstring(myVar, 5, 6);

 

The substring capabilities are better when you use a search routine to locate a break point in the string, such as a space in the above example, and then use that value inside the mbsubstring parameters.

Tips and precautions

The mbsubstring() string function was included in Flash 4, but replaced by the String.substr function (part of the String object) in Flash 5. While mbsubstring() is still supported, using the String object method is the preferable approach because of greater flexibility. (See String.substr for examples of usage.)
