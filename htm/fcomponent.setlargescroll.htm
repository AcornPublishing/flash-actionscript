FComponent.setLargeScroll

(This method is applicable to FScrollBar.setLargeScroll. )

 

myScrollBar.setLargeScroll(integer);

 

Where integer is an integer value that specifies how many units the scrollbar will move by when the user clicks on the scrollbar’s scroll track.

 

This method is compatible with Flash 6.

Description

The ‘large scroll’ value is the amount that the scrollbar will move every time you click on the scroll track. A related value is the ‘small scroll’ and this is the amount the scroll bar will move every time the top and bottom arrows are clicked.

 

 

The large scroll value is usually set to the height of the text field it is controlling minus one line. This means that if you do a large scroll downwards, the line that was previously at the bottom line of the text field will end up at the top line. This setting occurs automatically as soon as you drop the scrollbar onto a text field. You can also set the page size dynamically via the FComponent.setScrollProperties method. The FComponent.setLargeScroll allows you to modify the large scroll value.

Notes

 

myScrollBar.setLargeScroll(20);

 

Sets the large scroll value of the scrollbar instance myScrollBar to 20.

 

Examples and practical uses

The default value set when you drop a scrollbar onto a text field is usually sufficient (and is actually the same sort of value that most operating systems use for their scrollbars, so most users will expect this value).

 

The only time you will ever need to change the properties of the scrollbar is when you are using the scrollbar as a spinner or slider (as you might want to use for a volume control for example). The following script will set up a scrollbar (instance name spinner ) as a slider, and this will control the value displayed in a text field spinner_txt.

 

function change(component) {

    spinner_txt.text = component.getScrollPosition();

}

spinner_txt.text = 0;

spinner.setEnabled(true);

spinner.setScrollProperties(40, 0, 100);

spinner.setLargeScroll(10);

spinner.setChangeHandler("change");

 

The scrollbar is set up to give values from 0 to 100. Clicking on the scroll track will advance the scroll bar value by 10, and clicking on the up or down arrows will advance it by 1.

 

See also the example files fcomponentsetlargescroll.fla and fcomponentsetlargescroll.swf.
