Number.POSITIVE_INFINITY

Number.POSITIVE_INFINITY

 

Where Number is the Number class.

 

This property is compatible with Flash 5 and later.

 

The Number.POSITIVE_INFINITY property represents infinity. See the Infinity entry for more information.

Description

The Number.POSITIVE_INFINITY property is a value representing infinity. You cannot perform mathematical operations on Number.POSITIVE_INFINITY into numbers representable by Flash (those below Number.MAX_VALUE ). Flash does not perform all manipulations with infinity properly. For example, infinity/infinity should result in 1, and infinity-infinity should result in 0, but Flash does not do this correctly.

Notes

 

myNum = Number.POSITIVE_INFINITY;

 

 

Assigns the value of infinity to myNum. You cannot perform mathematical operations on this value to get into the range specified by Number.MAX_VALUE.

 

The value POSITIVE_INFINITY can be used for comparison. It is the largest number imaginable.

 

 

 

myNum = Number. POSITIVE_INFINITY / 10000;

 

 

Any mathematical operation on POSITIVE _INFINITY still results in the value of POSITIVE _INFINITY.

 

 

 

 

Examples and practical uses

The Number.POSITIVE_INFINITY property represents a value of infinity. You cannot perform meaningful mathematical operations on Number.POSITIVE_INFINITY because the result will always be infinite. You can, however, perform comparisons on the value, such as:

 

myNum = 10/0;

if ( myNum == Number.POSITIVE_INFINITY){

 // code to handle infinite results

}

 

Any value divided by zero is infinite, so the above comparison will be true and the code inside the curly braces will be executed.

Tips and precautions

As noted earlier, Flash does not perform all manipulations with infinity properly. Some functions that should return a value of infinity do not, such as the statement:

 

Math.tan(PI/2);

 

This should result in infinity but doesn’t (although the result is large enough for all practical purposes).

 

The Number.POSITIVE_INFINITY property is the same as Flash’s Infinity property. See also isFinite() and isNan().
