TextField.bottomScroll

myTextField .bottomScroll

 

Where myTextField is a text field instance.

 

Compatible with Flash MX and later .

 

The TextField.bottomScroll property represents the bottom-most line of a text field. This property is commonly used when creating scrollbars and other elements that interact with text fields. The value of this property is a 1-based integer, meaning the first line of the text field is 1.

Examples and practical uses

The TextField.bottomScroll property sets the bottom line of text that can be viewed in a text field using a 1-base numbering system. The value of the TextField.bottomScroll property is an integer. For example, the command:

 

myTextField .bottomScroll = 10;

 

means the bottom line that can be displayed is line 10. You can use the TextField.bottomScroll property to determine visible lines in the text field: all lines between TextField.scroll and TextField.bottomScroll are visible at all times.

Tips and precautions

This method is useful in creating scrollbars and the like. Because ready built scrollbar components are available anyway, you are unlikely to need this method.
