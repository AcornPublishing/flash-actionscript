TextField.length

myTextField .length

 

Were myTextField is a TextField instance.

 

Compatible with Flash Player later .

 

The TextField.length property indicates the number of characters in a text field.

Description

The TextField.length property provides a count of the number of characters in a text field, including any special characters such as Enter and Tab .

 

 

Code

 

 

Additional Explanation

 

 

Notes

 

myStrLength = myTextField .length;

 

 

 

Counts the number of characters in the myTextField text field and stores the result in myStrLength .

 

 

An integer is returned.

 

 

 

myStrLength = length( myTextField );

 

 

The TextField.length property is similar to the length property but is not a method when called as a TextField instance.

 

 

 

Examples and practical uses

The TextField.length property indicates the length of the text in a text field, including special characters. The TextField.length property can be assigned to a variable :

 

myStrLength = myTextField .length;

 

See the example textfield.length.fla and textfield.length.swf .

Tips and precautions

The TextField.length property allows you to check the length of a string entered into a text field. This can be used for validation or for parsing. You would typically do this within an onChanged event handler; when the text entr is the required length, you can assume user input is complete. The TextField.length property and the length property of a text instance do the same task, but the TextField.length property is faster.
