LoadVars.toString

instancename.toString()

 

where instancename is an instance of the LoadVars object

 

Compatible with Flash Player 6 and later.

 

The LoadVars.toString() method returns a string of all variables in the instance.

Description

The LoadVars.toString() method returns a string of all the variables and their values in an instance of the LoadVars object, using the application/x-www-urlform-encoded MIME format. This string can be assigned to a variable.

 

 

Code

 

 

Additional explanation

 

Notes

 

myVars=myMovieVars.toString();

 

 

 

 

 

Creates a MIME format string from the variable names and values in the myMovieVars instance and stores them in the variable myVars (which will be a string)

 

 

 

Examples and practical uses

The LoadVars.toString() method is used to create a string that contains all the variables and values in a LoadVars instance. The string is formatted in application/x-www-urlform-encoded MIME format. To use this method it has to be associated with an instance of the LoadVars object, and then can be called for that instance like this:

 

myMovieVars = new LoadVars();

myMovieVars.Surname = "Parker";

myMovieVars.Firstname = "Tim";

myMovieVars.BirthYear = 1958;

myMovieVars.City = "Ottawa";

myString = myMovieVars.toString();

 

This will write the variables from the myMovieVars instance into the string myString, which looks like this:

 

Surname=Parker&Firstname=Tim&BirthYear=1958&City=Ottawa

 

The string could be stored into a file and read using LoadVars.load to store this information into a LoadVars instance, or you could create the file and store in an instance at the same time by using the LoadVars.sendAndLoad() method. You could also display the results of the string as output by using a trace action:

 

trace(myMovieVars.toString);

Tips and precautions

Probably the only reason to convert data to a string is if the server expects it, and you are about to send the data back after you have altered it.
