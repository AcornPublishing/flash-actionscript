TextField.textColor

myTextField .textColor

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.textColor property sets or reads the color of the text of the text field.

Description

The TextField.textColor property is the color of any text content in a text field. Colors are defined using hexadecimal values.

 

 

Code

 

 

Additional Explanation

 

 

myPath = myTextField .textColor ;

 

 

myTextField .textColor = 0x000000

 

 

Stores the color property of myTextField in myPath.

 

Sets the text color to black.

 

myPath = myTextField .textColor() ;

 

 

There is no method with this name.

Examples and practical uses

The TextField.textColor property is the color assigned to any text in a text field. It can be read and stored in a variable or set as a hex value:

 

myPath = myTextField .textColor;

myTextField .textColor = 0xFF0000;

 

See the example textfield.textcolor.fla and textfield.textcolor.swf .

Tips and precautions

Where you want to change the color of many text fields as part of a general style change, consider using a TextFormat object. You can set the color of a TextFormat object with TextFormat.color .
