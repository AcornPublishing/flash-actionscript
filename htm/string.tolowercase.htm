String.toLowerCase

newString = myString.toLowerCase();

 

Where newString will be the same string as myString, only any uppercase characters appearing in myString will become lowercase.

 

The object is compatible with Flash 5 and above. There are no known issues with any version of Flash that supports this object.

 

The toLowerCase method will take a string and create a new string that equals the original string, only all characters will be lowercase. So all uppercase characters in the original string will be forced to lowercase. See also String.toUpperCase and ===.

Description

This method will duplicate the specified string, only all characters will be lowercase.

myString = toLowerCase("My String");

 

 

Returns undefined.

 

toLowerCase is a method, so it must be invoked like a method.

 

Examples and practical uses

Programming

The toLowerCase method is excellent when working with XML. When you’re parsing the XML nodes in Flash it is a good idea to use XMLNode[i].toLowerCase() when checking the name of the node against what you think it will be, especially if your receiving XML from unknown sources. See the example stringtolowercase.fla / stringtolowercase.swf.
