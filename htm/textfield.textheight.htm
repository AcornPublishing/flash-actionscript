TextField.textHeight

myTextField .textHeight

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField.textHeight property sets the height of the text in the text field.

Description

The TextField.textHeight property is the height of any text content in a text field.

 

 

Code

 

 

Additional Explanation

 

 

myPath = myTextField .textHeight ;

 

 

myTextField .textHeight = 10

 

 

Stores the height property of myTextField in myPath .

 

Sets the text height to 10.

myPath = myTextField .textHeight () ;

There is no method with this name

Examples and practical uses

The TextField.textHeight property is the height of any text in a text field. It can be read and stored in a variable or set as a number:

 

myPath = myTextField .textHeight;

myTextField .textHeight = 9;

 

See the example textfield.textheight.fla and textfield.textheight.swf .

Tips and precautions

To change the height of the text field (something you may need to do if you make the text within it larger, use TextField._height .
