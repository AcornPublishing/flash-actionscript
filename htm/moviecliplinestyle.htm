MovieClip.lineStyle

myMovieClip.lineStyle ( [thickness [, color [ ,alpha]]])

 

Where myMovieClip is a movie clip instance, thickness is the width of the line in points, color is the hexadecimal code of the color, and alpha sets the transparency.

 

Compatible with Flash 6 and later.

 

The MovieClip.lineStyle()method sets the line style that will be used by drawing methods.

Description

The MovieClip.lineStyle()method is used to set the default line style for any future drawing methods. The line style set with the MovieClip.lineStyle()method remains in effect until reset with another lineStyle()method or a MovieClip.clear() method is seen.

Note

Point sizes for lines range from 0 (hairline) to 255 points.

 

Examples and practical uses

The MovieClip.lineStyle()method sets the line style for future drawing commands, such as lineto(). The MovieClip.lineStyle()method can be used with no arguments:

 

myMovieClip.lineStyle();

 

… which sets the line style to not be drawn (any drawing command has no visible appearance). Setting the thickness to undefined also has the same effect.

 

The MovieClip.lineStyle()method can be used with up to three parameters. The first sets the thickness of the line in points (valid values are 0 to 255; if a value less than 0 is given, 0 is assumed and if a value greater than 255 is given, 255 is used). If there is no line thickness (or the value of the thickness is undefined, then the line is not drawn; a value of 0 is a hairline. For example:

 

myMovieClip.lineStyle(10);

 

… sets the line style to a 10 point width, black in color. To set the color of the line, use the hexadecimal value for the color. When no color is specified, black is used by default.

 

The third parameter is the alpha (transparency) setting. This ranges from 0 to 100. If a value is less than 0, 0 is assumed, and if the value is greater than 100, 100 is assumed. This value sets how transparent the line is. A value of 0 is totally transparent, while a value of 100 is totally solid. For example:

 

myMovieClip.lineStyle(25, 0x0000FF, 75);

 

… sets the line style to a blue 25 point line, with 75% opacity.
