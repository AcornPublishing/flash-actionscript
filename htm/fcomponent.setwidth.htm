FComponent.setWidth

(This method is applicable to FListBox.setWidth. )

 

FListBox.setWidth(width)

 

Where FListBox is a list box component and width is the width in pixels that you want to set it to.

 

This method is compatible with Flash 6.

Description

You can resize your list box width during content creation using the standard Flash tools (Free Transform tool or Property inspector/Transform panel), and the setWidth method allows you to do the same dynamically. You should also consider using the FComponent.setSize method, which lets you set both width and height together.

 

The default width of a list box is 100 pixels.

 

 

Code

 

 

Notes

 

list.setWidth(200);

 

Sets the width of the list box instance list to 200 pixels.

 

Examples and practical uses

Changing the sizes of a component is something that you will not generally need to do. You are more likely to change the sizes to the maximum values required by your data during content creation, and this can be done manually via the standard Flash interface.

Tips and precautions

Components should not be scaled using the movie clip properties _xscale, _yscale, _width and _height. This will stretch and squash the component, and result in odd results.
