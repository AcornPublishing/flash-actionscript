The TextField.scroll property shows the vertical position of text in a text field.

Description

The TextField.scroll property shows the position of the current text in a text field vertically. The property can be set and queried.

Examples and practical uses

The TextField.scroll property shows the position of the current text in the entire text field, used to control scrolling in the text field with long passages of text. It can be assigned to a variable:

 

The primary use of the TextField.scroll property is to use it in the creation of scrollbars. You can then use the scrollbars to scroll through the text in the text field.

 

See the example textfield.scroll.fla and textfield.scroll.swf .

Tips and precautions

The TextField.scroll property lets you control long passages of text, moving to specific locations in the text within the available text field display area.

 

The scrolling is done on a per line basis, and not on a per pixel basis. You would need the latter if for example, you wanted to add scrolling titles before a video clip. To perform such effects, you should consider embedding the text inside a movie clip and moving the clip to give the scroll effect, remembering to embed the font outlines of the text (you cannot animate text fields via property based animation unless you do; the Tips and Precautions section for TextField refers).

 

The scroll property is associated with the creation of scrollbars for text fields. Because you can use ready built scrollbar components, the need to ever use this property in practical situations is low; Macromedia have already done all the hard work for you!
