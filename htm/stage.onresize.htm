Stage.onResize

myListenerObject = new Object();

myListenerObject.onResize = function () { //code }

Stage.addListener(myListenerObject);

 

var success = Stage.addListener(myListenerObject);

 

Where myListenerObject is assigned as a listener object of the Stage object. The onResize method of myListenerObject will be executed when the onResize event of the Stage object is invoked.

 

This event is compatible with Flash 6 and above. There are no known issues with any version of Flash that supports this object.

 

Being the only event of the Stage object, the onResize event will execute the onResize method of any listening objects. Your onResize even handler code should reside within the listener object’s onResize method. To register the object as a listener of the Stage object, use the Stage.addListener method. See also Stage.addListener and Stage.removeListener.

Description

This event is invoked when the size of the stage is modified.

Incorrect usage.

 

You cannot directly define a named or anonymous function to the onResize event. You should instead create a listener object of the Stage object, which is used as callback function to handle the onResize event.

 

Examples and practical uses

The onResize event is most commonly used to reposition graphic elements in your movie when the stage is resized. This is great for creating navigation systems that automatically compensate for an increase or decrease in available size. See the example stageonresize.fla / stageonresize.swf.

Tips and precautions

To take advantage of the onResize event you need to create a listener object. You cannot directly define a function to the onResize event. It simply won’t work. The following snippet of code is the basis of creating and applying a listener object:

 

myListener = new Object();

Stage.addListener(myListener);

 

myListener.onResize = function() {

trace(Stage.width, Stage.height);

}
