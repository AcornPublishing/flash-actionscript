TextField._width

myTextField ._width

 

Where myTextField is a TextField instance.

 

Compatible with Flash Player 6 and later .

 

The TextField._width property sets or retrieves the width of the text field in pixels.

Description

The TextField._width property is used to both set and display the width of a text field. The width is reported in pixels.

Examples and practical uses

The TextField._width property is used to set and retrieve the width of a text field in pixels. To use the TextField._width property to retrieve the width of a text field assign the result of the property to a variable:

 

myHeight = myTextField ._width;

 

To set a width, assign a value to the property:

 

myTextField ._width = 100;

 

The TextField._width and TextField._height properties are often used together to set a text field's dimensions.

 

See the example textfield._width.fla and textfield._width.swf .

Tips and precautions

The TextField._width property is handy in setting the width of a text field. The corresponding TextField._height property does the same task for the height of a text field.
