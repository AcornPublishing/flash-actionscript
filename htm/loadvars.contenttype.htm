LoadVars.contentType

instancename.contentType

 

where instancename is an instance of the LoadVars object.

 

Compatible with Flash Player 6 and later.

 

The LoadVars.contentType property is used to specify a MIME content type used by an instance’s LoadVars.send() and LoadVars.sendAndLoad() methods.

Description

LoadVars.contentType is a property of a LoadVars instance that is used to set the MIME type for the Loadvars.send() and Loadvars.sendAndLoad() methods. If the MIME type is set incorrectly, the data may not be properly understood. The default MIME type is application/s-www-urlform.

 

 

Code

 

 

Additional explanation

 

Notes

 

myMovieVars.contentType = "application/s-www-urlform";

 

 

 

Sets the MIME content type for the myMovieVars instance

 

The default MIME type of application/s-www-urlform is widely used but can be overridden.

Examples and practical uses

The LoadVars.contentType property is used to set the MIME type of data being sent to and from a movie using the.send() and.sendAndLoad() methods of the instance. To use this property it has to be associated with an instance of the LoadVars object, and then can be called and assigned for that instance like this:

 

myMovieVars = new LoadVars();

myMovieVars.contentType = "application/s-www-urlform";

Tips and precautions

The default MIME format setting of application/s-www-urlform is used for most CGI scripts. This uses a layout where each variable name is followed by an equal sign, the value assigned to the variable, an ampersand, and then the next variable-value pair until all variables are assigned, like this:

 

Petname=pepper&PetType=cat&PetAge=5&PetColor=tabby

 

Words within an application’s MIME format string must be separated by a plus sign, like this:

 

Petname=Mr+Big&PetType=cat&PetAge=5&PetColor=tabby

 

Avoid using any other MIME format than application/s-www-urlform unless you are sure you need to; this format is used by almost all CGI scripts and is the most widely used.

 

The contentType property can only be referenced from an instance of the LoadVars object.
