scroll

textfieldname.scroll = val;

 

Where textfieldname is the text field and val is the value to scroll.

 

This property is compatible with Flash 4 and later. Note that in Flash 4 and 5 the textfieldname refers to the variable name of the text field – this is depreciated in Flash MX. In Flash MX, textfieldname refers to the instance name of the text field.

 

The scroll property defines where in the text field content display starts for the user.

Description

The scroll property is used to control how much of a text field’s content is shown to the user in a text field. The property defines the starting point in the content that is shown on-screen.

 

 

Code

 

 

Notes

 

myText.scroll += 5;

 

 

Moves down 5 lines in the content of myText.

 

Examples and practical uses

The scroll property is a value that can be set or retrieved that indicates where in a text field the display of the content is to be started. This is used in long content that exceeds the display area of the text field. The value of the scroll property indicates the starting point for the display. The scroll property is often used to move through long documents, moving to various points in the content as starting points in the field. The user can still scroll the document however they wish; the scroll property indicates a starting point only. For example the code:

 

on (release){

 myTextField.scroll += 15;

}

 

… is used to move 15 lines further down in the document every time a button is released in the user interface.

Tips and precautions

The scroll property can be set or retrieved and can have different values based on button actions. Usually, the scroll property is used with scrollbars in text fields. Flash MX and later already provide a functional (and customizable) scrollbar, so the scroll property is not used much with Flash MX or later.
