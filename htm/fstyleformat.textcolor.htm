FStyleFormat.textColor

a.textColor = b;

 

Where a is the name of your custom style format, and b is a RGB value being assigned to the text within the component instance(s).

 

Compatible with Flash MX. This will be the primary color of all text within the component instances assigned to this style format. The RGB color value is assigned to the property using the 0xRRGGBB format. Related entries include those for FStyleFormat.textSelected, and FStyleFormat.textDisabled.

Description

This property assigns a new color value to text within components listening to the custom style format.

Notes

 

myStyle = new FStyleFormat();

myStyle.textColor = 0xFF0000;

myStyle.applyChanges();

myStyle.addListener (myComponent);

 

 

myStyle = new FStyleFormat({textColor:0xFF0000});

myStyle.applyChanges();

myStyle.addListener (myComponent);

 

 

This code changes the text color in the instance myComponent to red.

 

Examples and practical uses

In our example fstyleformat.textcolor.fla and fstyleformat.textcolor.swf , text color is changed within several components. The textColor value is red on the first frame of the movie. A button is pressed to take you to a second page. On this page, the textColor value is changed to reflect a new overall theme. This illustrates how you can update a custom style, and not have to necessarily construct a new custom style format. You can simply update one property within the style format when necessary, and use the FStyleFormat.applyChanges method.

Tips and precautions

Using this property will change the primary text color of your components listening to the style. When you are working with color values in your custom style format, make sure you test that your text is still legible against the background and face color values assigned to the style. This color is controlled using the FStyleFormat.background property.

 

You should also remember that different colors will be active if the text is selected or disabled. These values can be assigned using the FStyleFormat.textDisabled and FStyleFormat.textSelected properties.
