FComponent.getRowCount

(This method is applicable to FComboBox.getRowCount and FListBox.getRowCount. )

 

a = myComponent.getRowCount();

 

Where the variable a equates to the number of labels that are visible at any one time for the instance myComponent.

 

This method is compatible with Flash 6.

Description

This method is applicable for those components that display a list of options, and this includes the combo box and list box. For the combo box and list box shown below, the getRowCount method will return 8 and 6 respectively.

Examples and practical uses

The getPaneWidth method can be used in conjunction with the FComponent.getLength method to see what proportion of the labels can be seen at any one time. In particular, if the getpaneWidth method returns a lower value than the getLength method, the combo box or list box in question will not be able to display all its labels, and will have an active scrollbar (as shown below left). If the getLength is the greater value, there will be no need for a scrollbar (or it will show up as inactive) (see below right).

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

If you need to increase the row count, you can do so via the FComponent.setRowCount method. You would know you needed to do this so that all available options were visible if the getLength method returned a value greater than the getRowCount value. The code below will automatically increase the size of the drop-down menu in our combo box instance perryCombo if, after adding a new label, it is found that there is not enough space to show all the labels currently defined.

 

perryCombo.addItem("new label");

if (perryCombo.getLength()>perryCombo.getRowCount()) {

    perryCombo.setRowCount(perryCombo.getRowCount()+1);

}
