MovieClip.play

myMovieClip.play()

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 5 and later.

 

The MovieClip.play()method begins playback of the movie.

Description

The MovieClip.play()method begins playback of the movie at the current playhead location.

 

 

Code

 

 

Additional explanation

 

 

Notes

 

myMovieClip.play();

 

 

Starts playing myMovieClip.

 

Playback continues until stopped by some event or action.

 

 

myMovieClip.play(5);

 

 

No parameter can be specified for this method.

 

If parameters are provided, the command fails.

 

Examples and practical uses

The MovieClip.play()method starts the playhead moving in the current movie and continues playback until stopped. The MovieClip.play()method is called simply by reference to the clip name:

 

myMovieClip.play();

Tips and precautions

The MovieClip.play()method starts playback of a movie after it has been stopped or loaded.
