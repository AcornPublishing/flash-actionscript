MovieClip.onReleaseOutside

myMovieClip.onReleaseOutside

 

Where myMovieClip is a movie clip instance.

 

Compatible with Flash 6 and later.

 

The MovieClip.onReleaseOutsideevent handler is invoked when a mouse button has been pressed over a button movie clip then is released outside that clip.

 

This event is the same as the corresponding Button event Button.onReleaseOutside. See the entry for Button.onReleaseOutside.
