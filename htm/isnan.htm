isNaN

isNaN(a)

 

Where a is an expression which may include a variable or a Boolean value.

 

Compatible with Flash 5. You would use this function to check if a value returned is a number or a string, which would indicate a mathematical error. You would not use this function to check for the validity of the value. Related entries include isFinite.

Description

This function will return true if the value returned is not a number, and false if the value is a number.

trace (isNaN(33r));

An error is thrown if you attempt to trace a value beginning with a number and ending with a string. However, if you trace isNaN(r33) a value of false is returned.

 

Examples and practical uses

This function is used to test whether an expression is numeric. You could use this function to ensure that the user has correctly filled in a form value. If you have a movie where a user enters data that needs to be numeric, then you'd have to call isNaN function to ensure that it is a valid value.

Tips and precautions
