Welcome to the friends of ED Macromedia Flash MX Designer's ActionScript Reference CD.

On this CD you will find all the code to accompany the book, three bonus chapters as well as an extended interactive version of the ActionScript Dictionary.

The code for each chapter is available as a self extracting archive for PC and Mac, and also as individual FLAs within the code folder on the CD.

The bonus chapters form a guide to using XML with Flash MX and are provided as a print quality PDF file.

The dictionary is best viewed at a minimum screen resolution of 1024x768 pixels. It requires that you have the Flash Player 6 plug installed, as it contains many example Flash movie files. FLA files are also provided, for which you will need Macromedia Flash MX. If your web browser will not open the example .fla file links directly into Macromedia Flash MX please see guide notes.

Problems opening fla files directly from web browsers.

If your web browser will not open the example FLA file links directly into Macromedia Flash MX, you may need to adjust the File Helper settings in the browser preferences. Clicking on the 'Add...' button will allow you to add the .fla suffix to the list of file types that the browser can handle.

The description should be flash document and the extension .fla.

Select Flash MX as the application and .fla as the File Type. The File creator should be MFL2.

In the 'How to handle' option, select 'Post-Process with Application' and selct Flash MX as the application.
